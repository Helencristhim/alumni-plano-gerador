Capítulo 3: El Código

Reportar un Problema

no funciona · no hay · está roto

Descubre la Gramática

Escucha y Observa

"El aire acondicionado no funciona."

"No hay toallas limpias."

"La luz está rota."

"Quería reportar un problema con la habitación."

FrasePara decir que...
... no funcionaAlgo no trabaja (el aire, la tele)
No hay...Falta algo (toallas, jabón)
... está roto / rotaAlgo se rompió (la luz, la ventana)
Quería reportar un problema con...Abrir la queja con cortesía

"Roto/rota" cambia según el género: la luz está rota, el aire está roto.

Error Común

Correcto vs Incorrecto

✗ INCORRECTO

"El aire no trabaja."

✓ CORRECTO

"El aire no funciona."

✗ INCORRECTO

"¡Esto está fatal! ¡Arréglenlo!"

✓ CORRECTO

"Quería reportar un problema con el aire, por favor."
