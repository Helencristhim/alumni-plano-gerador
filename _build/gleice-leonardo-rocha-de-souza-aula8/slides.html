Chapter 3 -- The -s Code

The Present Simple

With he, she, it -- the verb adds -s

The -s Code

Add the -s

BaseHe / She / ItBaseHe / She / It
developdevelopsneedneeds
sellsellsmakemakes
supplysupplieslaunchlaunches

Key idea: with he, she, it and singular nouns (the company, a patient), the present simple verb adds -s. Verbs ending in consonant + y change to -ies: supply to supplies. For negatives and questions, use does plus the base verb: Sanofi does not develop, Does Sanofi develop?

Common Mistake

Watch Out

"Sanofi develop new medicines. ... Does the company sells products?"
"Sanofi develops new medicines. ... Does the company sell products?"

In the affirmative, add -s with he, she, it. After does, always use the base verb.
