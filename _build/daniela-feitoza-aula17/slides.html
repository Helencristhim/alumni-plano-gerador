Habit Rule

Used to, Would, but Now

FormUseExample
used to + base verbPast habit OR state, not true now"I used to work on call."
would + base verbRepeated past action (not for states)"The system would crash every night."
used to be / used to havePast STATE (would is wrong here)"We used to have a backlog."
but now + presentContrast with today"...but now it is automated."
Common Mistake

Wrong Form vs Right Form

✗ WRONG

"I would have an old laptop." (state, not a repeated action)

✓ RIGHT

"I used to have an old laptop." (state = used to)

✗ WRONG

"I use to work on call."

✓ RIGHT

"I used to work on call." (always "used to" with -d)

Use "used to" for past states; "would" only for repeated past actions. Always write "used to" with the -d.

0:000:00
1. When do you use "used to", according to Sarah?
For a habit or a situation that was true in the past but is not true now.
2. What is "would" good for, and what is it NOT good for?
It is good for repeated past actions and routines, but not for past states. For a state, you must use "used to".

Add a present sentence with "but now", like "I used to work on call, but now we have a rotation."
