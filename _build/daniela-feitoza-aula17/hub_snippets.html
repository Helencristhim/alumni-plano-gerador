Warm-up — Callback da Aula 16

Review

Na Aula 16 você aprendeu a ordenar uma história com o past perfect ("By the time he arrived, I had escalated it") + sequenciadores. Agora você compara o passado com o presente: como algo costumava ser. Para hábitos e estados do passado que mudaram, use used to + verbo base ("I used to work on call"). Para ações repetidas do passado, use would + verbo base ("the system would crash"). E feche com but now para o contraste. (Say how it used to be, then how it is today.)

Stage 1.1: Vocabulary Cards

Vocabulary

Ouça cada palavra, leia o exemplo e a tradução. Toque em Listen para ouvir.

Legacy -- an old system or technology that is still in use (PT: sistema legado / antigo em uso)
"The legacy system used to crash every night."
Manual -- done by hand, without automation (PT: manual / feito à mão)
"We used to do the deployment manually."
Rotation -- a system where people take turns doing a duty (PT: rodízio / escala)
"Now we have a rotation, so no one is always on call."
Outdated -- too old to be useful or correct anymore (PT: ultrapassado / desatualizado)
"The outdated scripts are gone now."
Streamline -- to make a process simpler and more efficient (PT: simplificar / otimizar)
"We streamlined the process and removed extra steps."
Backlog -- a large amount of unfinished work that has built up (PT: fila de pendências / acúmulo)
"We used to have a huge backlog of tickets."
Handover -- passing work or responsibility from one shift to the next (PT: passagem de turno / transferência)
"The handover between shifts used to be a mess."
Tedious -- boring and tiring because it is slow and repetitive (PT: tedioso / maçante)
"The old process was slow and tedious."
Automate -- to make a machine do a task without a person (PT: automatizar)
"Now the deployment is automated."
Overhaul -- a complete change or renovation of a system (PT: revisão completa / reformulação)
"We did a full overhaul of the old code."

Stage 1.3: Grammar in Context

GRAMMAR

Leia o texto e responda às perguntas.

Two years ago, the team worked very differently. They used to firefight every night — a past habit, so Daniela uses used to + base verb. The legacy system would crash, and one person would fix it alone: these are repeated past actions, so she uses would + base verb. The process used to be slow and tedious — this is a past state, so she must use used to be, never "would be". To show the change, she adds but now: "We used to deploy manually, but now it is automated." Remember: "used to" covers habits and states; "would" covers only repeated actions, not states.

1. Why does Daniela say "they used to firefight every night"?
A It was a past habit that is not true now.
B It is a plan for the future.
C It is still true today.
2. Why does she use "used to be" and not "would be" for the process?
A Because it was a repeated action.
B Because it is a past state, and "would" is not used for states.
C Because it is in the future.
3. Which sentence is correct?
A "I would have an old laptop."
B "I used to have an old laptop."
C "I use to work on call."

Stage 1.4: Grammar Tip — used to / would & but now

GRAMMAR

Hábitos e estados do passado vs ações repetidas (bilingue EN + PT).

FormUseExample
used to + verbo baseHábito OU estado do passado que mudou"I used to work on call."
would + verbo baseAção repetida do passado (não para estados)"The system would crash every night."
but now + presenteContraste com o presente"...but now it is automated."
Regra (PT)Use used to + verbo base para hábitos E estados do passado. Use would + verbo base APENAS para ações repetidas (rotinas) — NUNCA para estados (não se diz "I would have"; o certo é "I used to have"). Sempre escreva used to com o -d. Feche com but now para o contraste.
