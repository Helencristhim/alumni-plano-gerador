Reading Contracts II -- Conditional Clauses with if / unless / provided

Read the conditions of a contract and say when something happens, on what condition, and what would stop it. New vocabulary: provision, governing law, jurisdiction, terminate, comply, waiver, amend, enforce. Grammar: conditional clauses with if (the condition), unless (= if not), and provided (= only if this is true).

Vocabulary

Provision -- a specific rule or condition in a contract
"There is a provision that ends the contract." (Disposição / cláusula específica)
Governing law -- the law that controls the contract
"The governing law is German law." (Lei aplicável)
Jurisdiction -- the country or court that decides a dispute
"We prefer Brazilian jurisdiction." (Jurisdição / foro)
Terminate -- to end a contract
"The contract terminates if a party does not pay." (Rescindir / encerrar)
Comply -- to follow a rule of the contract
"Each party must comply with the deadline." (Cumprir)
Waiver -- when a party gives up a right on purpose
"We accept the waiver only in writing." (Renúncia / dispensa)
Amend -- to change part of a contract officially
"We can amend the contract in writing." (Alterar / emendar)
Enforce -- to make a rule of the contract happen by law
"We enforce the contract from June." (Fazer valer / executar)

Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

Simone reads the conditions of the new draft. There is a provision that says the contract terminates if a party does not pay within thirty days. She thinks this is too strict, so she will ask for sixty days, provided the partner accepts a small interest. The draft also sets German governing law, but Simone prefers Brazilian jurisdiction, unless the partner has a strong reason. Finally, she will not accept the waiver unless it is signed by both parties. If the partner agrees, the parties will comply with the new terms and enforce the contract from June.

1. When does the contract terminate?
A If a party does not pay within thirty days.
B If the partner changes the governing law.
C If both parties sign the waiver.
2. On what condition will Simone ask for sixty days?
A Provided the contract follows German law.
B Provided the partner accepts a small interest.
C Provided the waiver is removed.
3. When will Simone accept the waiver?
A Unless the partner pays on time.
B Only if it is signed by both parties.
C Only if the governing law is Brazilian.

Stage 1.4: Grammar Tip -- Conditional Clauses

GRAMMAR

Para dizer EM QUE CONDIÇÃO algo acontece num contrato, use uma conditional clause. If = se (a condição). Unless = a não ser que / se não (= if not). Provided = desde que (= só se isto for verdade). O verbo fica no presente nas duas partes: "the contract ends IF a party FAILS to pay". Não use will depois de if/unless/provided. / To say ON WHAT CONDITION something happens in a contract, use a conditional clause. If = the condition. Unless = if not. Provided = only if this is true. Use the present in both parts: "the contract ends if a party fails to pay". Do not use will after if, unless or provided.

Word / PalavraMeaning / SentidoExample / Exemplo
if / sethe condition"The contract terminates if a party does not pay."
unless / a não ser que= if not"We wait, unless the partner agrees."
provided / desde queonly if this is true"We accept it provided it is in writing."
