Stage 1.4: Grammar Tip -- is & has

GRAMMAR

Usado para descrever um objeto: qualidades e recursos. / Used to describe an object: qualities and features.

Form / FormaMeaning / SentidoExample / Exemplo
It / The aircraft is + adjectiveUma qualidade / A quality (fast, safe, modern)It is fast.
It / The aircraft has + a/an + featureUma parte ou recurso / A part or featureIt has a powerful engine.
very / so + adjectiveReforça a qualidade / Makes the quality strongerIt is very safe.

Key point / Ponto-chave: use is antes de um adjetivo (qualidade) e has antes de um recurso (uma coisa que ele inclui). Com it e the aircraft, a palavra é has, nunca have: it has, não "it have".
