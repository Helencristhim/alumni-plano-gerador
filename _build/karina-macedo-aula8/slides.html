Chapter 3 -- The Code

Grammar Discovery

is -- a quality -- has -- a feature

Grammar Discovery

Listen and Notice

How do we describe an aircraft?

"The aircraft is fast."

"It is very safe."

"It has a powerful engine."

"It has a long range."

Grammar Rule

is & has

StructureWhen we use it
It / The aircraft is + adjective (it is fast)A quality: fast, safe, modern
It / The aircraft has + a/an + feature (it has an engine)A part or feature it includes
very / so + adjectiveMake the quality stronger

Key idea: use is before an adjective (a quality) and has before a feature (a thing it includes). For it and the aircraft, the word is has, never have (it has, not it have).

Common Mistake

Watch Out

"It have a powerful engine. The cabin has comfortable."
"It has a powerful engine. The cabin is comfortable."

Use has (not have) with it, and use is before an adjective like comfortable.
