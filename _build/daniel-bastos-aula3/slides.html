Capítulo 3: El Código

La Pasiva con se

se cultiva · se exporta · se tuesta

Descubre la Gramática

Escucha y Observa

"El café se cultiva en la finca."

"El grano se tuesta para darle sabor."

"Los granos se exportan por barco."

"El precio se decide en el mercado."

EstructuraUsoEjemplo
se + verboProcesos generales (no decimos quién)El café se cultiva.
se + verbo singularCon un objeto singularSe exporta el grano.
se + verbo pluralCon un objeto pluralSe exportan los granos.

El verbo concuerda con la cosa: una cosa = singular, varias cosas = plural.

Error Común

Correcto vs Incorrecto

✗ INCORRECTO

"El café cultiva en Brasil."

✓ CORRECTO

"El café se cultiva en Brasil."

✗ INCORRECTO

"Se exporta los granos."

✓ CORRECTO

"Se exportan los granos."
