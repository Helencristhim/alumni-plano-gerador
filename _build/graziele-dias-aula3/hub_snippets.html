Etapa 1.3: Grammar in Context

Grammar

Leia o texto e responda as perguntas. (Read the text and answer.)

Welcome to my hospital. It is a big building. On the first floor, there is a reception next to the door. There are two elevators near the reception. On the fourth floor, there are many patients and many rooms for them. On this floor, there isn't a kitchen, but there is a small office.

1. Why do we say "there is a reception" but "there are two elevators"? (Por que "there is" e depois "there are"?)
A there is = one thing; there are = many things. (there is = uma coisa; there are = muitas.)
B They mean the same thing. (Significam a mesma coisa.)
C "there is" is for the future. (there is e para o futuro.)
2. How do we say there is NO kitchen? (Como dizer que NAO ha cozinha?)
A There no is a kitchen. (There no is a kitchen.)
B There isn't a kitchen / there is not a kitchen. (There isn't a kitchen.)
C There aren't a kitchen. (There aren't a kitchen.)
3. Where are the patients? (Onde estao os pacientes?)
A On the first floor. (No primeiro andar.)
B On the fourth floor, in many rooms. (No quarto andar, em muitos quartos.)
C In the elevators. (Nos elevadores.)

Etapa 1.4: Grammar Tip -- There is / There are

Grammar

Como dizer que algo existe em um lugar. / How to say something exists in a place.

How many / QuantosAffirmative (+)Negative (-) / Question
One / Um (singular)There is a room.There isn't a room. / Is there a room?
Two+ / Dois+ (plural)There are two rooms.There aren't two rooms. / Are there two rooms?
Key idea / Ideia-chaveUma coisa = there is + a/an. Muitas coisas = there are + plural com -s. / One = there is + a/an; many = there are + plural -s.
