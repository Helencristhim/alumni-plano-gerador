Chapter 3

The Code

used to / would / get used to

Discovery

What Do These Have in Common?

"We used to rely on manual spreadsheets." (past habit -- no longer true)

"Every Monday, someone would compile the data by hand." (repeated past action)

"I struggled to get used to the new interface." (adjusting to something new)

"Now I am used to working with the automated system." (already adjusted)

Grammar Rule

Past Habits and Adaptation

StructureUseExample
used to + verbPast habit or state (no longer true)We used to rely on spreadsheets.
would + verbRepeated past ACTION only (not states)Someone would compile the data every Monday.
get used to + -ingProcess of adjusting to something newI struggled to get used to the new interface.
be used to + -ingAlready adjusted (current state)I am used to working with the system now.
Common Mistake

Watch Out

"We would be a small team." (state -- cannot use would)
"We used to be a small team." (state -- correct with used to)

would = repeated ACTIONS only ("would compile", "would email"). used to = actions AND states ("used to be", "used to have", "used to rely").

Spot the Error

Find the Mistake

0 / 4 errors found

"We would be a small team three years ago."
"We USED TO be a small team." (would = actions only, not states. Use USED TO for be/have.)
"I am getting used to work with the new system."
"I am getting used to WORKING with the new system." (get used to + -ING, not base verb)
"She use to handle all the documentation manually."
"She USED TO handle all the documentation manually." (used to, not use to in affirmative)
"I am used to compile the data every Monday."
"I am used to COMPILING the data." or "I USED TO compile the data." (be used to + -ing vs used to + base verb)
