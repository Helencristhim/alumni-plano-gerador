Chapter 3 -- The Code

Grammar Discovery

Used to, would, and get used to

Grammar Discovery

Listen and Notice

Past habit, repeated past action, or adapting to something new?

"We used to track every order on paper."

"Every Friday we would reconcile the numbers by hand."

"We are getting used to working in the new system."

"The team used to rely on a legacy system."

Grammar Rule

Used to / Would / Get Used to

StructureUse
used to + base -- "We used to track orders on paper."A past habit or state, no longer true
would + base -- "Every Friday we would reconcile by hand."A repeated past action (not for states)
get / be used to + -ing -- "We are getting used to working in the new tool."Becoming or being accustomed to something

Key idea: used to + base talks about the past. be / get used to + -ing talks about adapting. Do not say "I am used to work"; say "I am used to working".

Common Mistake

Watch Out

"I am used to work late every night."
"I am used to working late every night."

After be / get used to, use the -ing form. Compare it with used to + base for a past habit (I used to work late).
