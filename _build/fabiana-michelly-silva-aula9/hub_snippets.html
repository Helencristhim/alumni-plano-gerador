The Way We Used to Work -- Processes That Evolved

Talking about past habits and how a process changed: used to + base for a past habit or state that is no longer true (We used to track orders on paper), would + base for a repeated past action (Every Friday we would reconcile by hand), and get/be used to + -ing to talk about adapting (We are getting used to the new workflow). New vocabulary: legacy system, phase out, streamline, obsolete, transition, rollout, workflow, automate, adopt, cumbersome, ramp up, pivot, benchmark, iteration, throughput.

Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Legacy system -- an old system that is still in use
"We are finally replacing the legacy system." (Sistema legado)
Phase out -- to gradually stop using something
"We will phase out the old tool by June." (Descontinuar gradualmente)
Streamline -- to make a process simpler and faster
"We streamlined the approval steps." (Simplificar)
Obsolete -- no longer useful; outdated
"The paper form is now obsolete." (Obsoleto)
Transition -- the process of changing from one state to another
"The transition took two quarters." (Transição)
Rollout -- the gradual introduction of a new system
"The rollout happened in stages." (Implantação)
Workflow -- the sequence of steps to complete work
"The new workflow saves hours." (Fluxo de trabalho)
Automate -- to make a task run by machine, not by hand
"We automated the weekly report." (Automatizar)
Adopt -- to start using something new
"The team adopted the new platform." (Adotar)
Cumbersome -- slow and complicated to use
"The old process was cumbersome." (Trabalhoso)
Ramp up -- to increase activity or output gradually
"It took a month to ramp up." (Acelerar)
Pivot -- to change strategy or direction
"We had to pivot when the design failed." (Mudar de direção)
Benchmark -- a standard point of reference for comparison
"We set a benchmark for throughput." (Referência)
Iteration -- one version in a repeated cycle of improvement
"Each iteration was better than the last." (Iteração)
Throughput -- the amount of work processed in a given time
"Throughput nearly doubled this year." (Vazão)

Stage 1.4: Grammar Tip -- Used to / Would / Get used to

GRAMMAR

Usado para falar de hábitos do passado e de adaptação a mudanças. / Used to talk about past habits and adapting to change.

Form / FormaMeaning / SentidoExample / Exemplo
used to + baseHábito ou estado passado, hoje falso / Past habit or state, no longer trueWe used to track orders on paper.
would + baseAção repetida no passado (não para estados) / Repeated past action (not for states)Every Friday we would reconcile by hand.
get / be used to + -ingAcostumar-se / estar acostumado / Become or be accustomedWe are getting used to working in the new system.

Key point / Ponto-chave: used to + base (We used to work...) descreve um hábito passado. be/get used to + -ing (We are used to working...) descreve adaptação. Não confunda: "I am used to work" está errado; o correto é "I am used to working".
