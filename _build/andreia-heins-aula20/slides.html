Language Rule

The Chair's Flow

StageStructureExample
OpenQuestion / invitationShall we start with the budget?
DiscussFirst / second conditionalIf we ordered two, we would save time.
FeedbackPast modal (could/should have)We could have planned it earlier.
ClosePolite request + next stepsCould you send the next steps by Friday?

Key idea: A chair leads the meeting through four stages. Open with a question or invitation; discuss options with the first or second conditional; give feedback softly with a past modal; and close with a polite request and clear next steps. This lesson reviews ALL the language from Block 2 in one real meeting.

Watch Out

Watch Out

✗ "We begin now. Talk." (too direct for a chair)

✓ "Shall we begin? Maria, would you like to take the floor?" (invite politely)

✗ "Meeting is over. Send the report."

✓ "Let us adjourn. Could you send the next steps by Friday?" (close with a polite request + next steps)
