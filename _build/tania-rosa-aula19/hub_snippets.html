Grammar

Stage 1.4: Grammar Tip -- Should & Had Better (Advice)

Grammar

Para dar conselhos, use should + verbo (sem "to"): "You should ask for a refund." Para negar, use should not: "You should not wait." Para um conselho FORTE (quase um aviso), use had better + verbo: "You had better call now." Nunca use "to" depois de should/had better, e nunca acrescente -s (she should, NUNCA she shoulds). / To give advice, use should + verb (no "to"): "You should ask for a refund." For negative, use should not. For STRONG advice (almost a warning), use had better + verb. Never use "to" after should/had better, and never add -s.

Estrutura / StructureEnglishPortuguês
should + verboYou should ask for help.Você deveria pedir ajuda.
should not + verboYou should not wait.Você não deveria esperar.
had better + verboYou had better call now.É melhor você ligar agora.
Should + sujeito + verbo?Should I make a complaint?Devo fazer uma reclamação?

Stage 4: Situational Quiz

Quiz

Choose the best response for each situation.

Your hotel room has a broken air conditioner. What do you say at the front desk?
A "I am sorry to complain, but my air conditioning is broken."
B "My air conditioning is very good."
C "I will fix the air conditioning myself."
A friend's flight is delayed. You give normal advice.
A "You should to ask for a meal."
B "You should ask for a free meal."
C "You shoulds ask for a meal."
The problem is urgent. You give STRONG advice.
A "You had better to change rooms."
B "You had better change rooms now."
C "You better changing rooms."
