Chapter 3

The Some & Any Code

some bread, any coffee, a meal

Grammar Discovery

Listen and Notice

"I eat some bread."

"I have some snacks."

"I do not have any coffee."

"Do you have any water?"

SentenceWord
Positive (+)I have some bread / some snacks
Negative (−)I do not have any coffee
Question (?)Do you have any water?
Count or nota meal, two snacks (count) · water, bread (no count)
Common Mistake

Right vs Wrong

✗ WRONG

"I have a bread."

✓ RIGHT

"I have some bread."

✗ WRONG

"I do not have some water."

✓ RIGHT

"I do not have any water."

✗ WRONG

"Do you have some snacks?"

✓ RIGHT

"Do you have any snacks?"

Use some in positive sentences. Use any in negatives and questions. Food we cannot count (bread, water, coffee) never takes a or a plural.
