Leading the Handover & Documenting the Process

Explaining how a team works so the job runs without one person, using the passive voice in the present (the report is reviewed) and past (the build was tested), and naming the agent with by when it matters. New vocabulary: handover, document, procedure, approve, assign, monitor, maintain, standardize.

Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Handover -- passing your work, tasks, and knowledge to another person
"The handover is designed so the team continues without me." (Passagem / transição de trabalho)
Document -- to write something down clearly so others can follow it
"Every change is documented in the wiki before it goes live." (Documentar / registrar)
Procedure -- a fixed set of steps for doing a task the same way every time
"The procedure is simple and it is followed by everyone." (Procedimento)
Approve -- to officially accept something so it can go forward
"The build is only approved when every check passes." (Aprovar)
Assign -- to give a task or ticket to a specific person
"Tickets are assigned at the Monday stand-up." (Atribuir / designar)
Monitor -- to watch a system or task over time to catch problems
"After release, the system is monitored for forty-eight hours." (Monitorar / acompanhar)
Maintain -- to keep something updated and in good working condition
"The runbook is maintained by everyone, not just me." (Manter / conservar atualizado)
Standardize -- to make something the same across teams, one agreed way
"The process was standardized across all three squads." (Padronizar)

Stage 1.4: Grammar Tip -- The Passive Voice

GRAMMAR

Use a voz passiva quando a AÇÃO importa mais que a pessoa, ou quando a pessoa é óbvia ou desconhecida. A forma é sempre be + past participle. Se quiser dizer quem fez, use by + pessoa. / Use the passive voice when the action matters more than the person. The form is always be + past participle. Add by + person only if you want to name who did it.

Form / FormaStructure / EstruturaExample / Exemplo
Presentam / is / are + past participleThe report is reviewed every Friday.
Present (plural)are + past participleTickets are assigned at the stand-up.
Pastwas / were + past participleThe build was tested before launch.
With the agent (by)be + past participle + by + personThe runbook was written by the team.

Key point / Ponto-chave: a passiva precisa SEMPRE do verbo be (is, are, was, were) + o past participle. Não esqueça o be: "The report reviewed" está errado; "The report is reviewed" está certo. Use by + pessoa só quando você quer nomear quem faz.
