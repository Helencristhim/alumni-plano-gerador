Grammar Discovery

Conditionals: if you heat water, it boils -- if we run the trial, we will know

Grammar Discovery

What do the orange words have in common?

"If we ignored the parameter, we would reach the wrong conclusion."

Grammar Rule

Conditionals for Cause

TypeStructureExample
Zero (general truth)If + present, presentIf you heat water, it boils.
First (real future)If + present, will + verbIf we run the trial, we will know the cause.
Second (hypothetical)If + past, would + verbIf we ignored it, we would be wrong.
Cause-effect linksleads to / due to / thereforeActivity leads to a better endpoint.

Key idea: the if half never takes will or would. Zero = always true; first = real future (present + will); second = a hypothesis (past + would). Remember: an association is real, but only a controlled study lets you claim causation.

Common Mistake

Watch Out

"If we will run the trial, we will know the cause."
"If we run the trial, we will know the cause."

After if, use the present (first) or the past (second), never will or would.

Grammar Practice

Build the Conditional

"If you heat water, it _____boils." (zero -- general truth)
"If we run a controlled trial, we _____will know the real cause." (first -- real future)
"If we ignored that parameter, we _____would reach the wrong conclusion." (second -- hypothetical)
"More activity _____leads to a better endpoint." (cause-effect link)

Use a first conditional: the trial will give you the cause.
