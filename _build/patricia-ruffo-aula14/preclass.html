Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Association -- a link where two things tend to change together
"There was a strong association between dose and recovery." (Associação)
Causation -- when one thing directly produces or causes another
"An association is not the same as causation." (Causalidade)
Determinant -- a factor that can change and shape the result
"Age is another determinant we must consider." (Determinante / fator)
Consequence -- the result or effect that follows from a cause
"A higher dose was linked to a better consequence for recovery." (Consequência)
Threshold -- the level at which an effect starts to appear
"Above a certain threshold, the effect became visible." (Limiar)
Meaningful -- large or clear enough to be important and trusted
"The result was meaningful after we controlled for age." (Significativo / relevante)
Attribute -- to say that a result is caused by a particular thing
"Part of the effect can be attributed to age." (Atribuir)
Confound -- when a hidden variable confuses the link between cause and effect
"Age may confound the link between dose and recovery." (Confundir / distorcer)

Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

At the results meeting, Patricia explains the link in the data. There was a strong correlation between a higher dose and a better outcome: when the dose went up, the recovery score also went up. But she is careful, because a correlation is not causation. The patients on the higher dose were also younger, so age is a confounding variable. Part of the effect can be attributed to age, not only to the dose. Her plan is clear. If the team runs a second trial, they will control for age from the start. And if the correlation stays strong after they control for that variable, the result will be much more significant -- above the threshold where the effect really matters. Only then can she say, with confidence, that the dose has a real effect.

1. What did the data show between dose and outcome?
A A strong correlation: a higher dose was linked to a better outcome.
B Proof that the dose caused the recovery.
C No link at all.
2. Why is the correlation not proof of causation?
A Because the sample was too small.
B Because age is a confounding variable: the higher-dose patients were younger.
C Because the study was too short.
3. What will the team do in the second trial?
A Use a higher dose for everyone.
C Stop measuring recovery.
B Control for age from the start.

Stage 1.4: Grammar Tip -- The first conditional

GRAMMAR

Para falar de causa e resultado provável, usamos a primeira condicional: if + presente (a causa) + will + verbo (o resultado). / To talk about a cause and its likely result, we use the first conditional: if + present (the cause) + will + verb (the result).

Function / FunçãoStructure / EstruturaExample / Exemplo
Cause / Causaif + present"If the dose is higher, ..."
Effect / Resultadowill + verb"..., the outcome will improve."
Negative / Negativowill + not + verb"..., the result will not be reliable."
Question / PerguntaWill + ... + if + present"Will it be significant if we control for age?"

Key point / Ponto-chave: A primeira condicional é if + presente (a causa) + will + verbo (o resultado provável). Depois de if, use o presente, nunca will (não se diz if we will control, e sim if we control). Lembre também: uma correlation mostra que duas coisas andam juntas, mas não prova causation -- uma variável escondida pode confundir o resultado.
