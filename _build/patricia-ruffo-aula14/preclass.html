Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Association -- a relationship in which two things tend to change together
"There is a strong association between the two parameters." (Correlação)
Causation -- when one thing directly produces another
"Association does not prove causation." (Causalidade / relação de causa)
Parameter -- a factor in a study that can change or be measured
"We changed one parameter at a time." (Variável)
Endpoint -- the result or effect that a study measures
"The treatment improved the main endpoint." (Resultado / desfecho)
Threshold -- a level or point at which something becomes true or important
"The result crossed the significance threshold." (Limiar)
Conclusive -- large or real enough to be unlikely to happen by chance
"The difference was statistically conclusive." (Significativo)
Attribute -- to say that something is the cause of a result
"We cannot attribute the effect to one cause yet." (Atribuir)
Confound -- a hidden parameter that affects the result and confuses the cause
"Time outdoors was a confounding parameter." (Variável de confusão / confundir)

Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

In her latest study, Patricia found a strong association between vitamin D levels and recovery scores. At first it looked simple: if you only read the chart, you would think that vitamin D causes faster recovery. But association is not causation. When the team looked closer, they found a hidden parameter: patients with higher vitamin D also spent more time outdoors and were more active. This is a confounding parameter. So the team was careful: they could not attribute the better endpoint to vitamin D alone. The difference was statistically conclusive -- it crossed the threshold -- but only a controlled study, where you change one parameter at a time, can prove a real cause.

1. What did Patricia find between vitamin D and recovery?
A Proof that vitamin D causes recovery.
B A strong association, not proven causation.
C No relationship at all.
2. Why could the team NOT attribute the endpoint to vitamin D alone?
A A confounding parameter (time outdoors) could explain it.
B The data was too small to read.
C The patients refused to answer.
3. What proves a real cause, according to the text?
A A strong association on a chart.
C A high vitamin D level.
B A controlled study that changes one parameter at a time.

Stage 1.4: Grammar Tip -- Conditionals for cause & effect

GRAMMAR

Para falar de causa e efeito, usamos condicionais: o que sempre acontece, o que vai acontecer e o que aconteceria. / To talk about cause and effect, we use conditionals: what always happens, what will happen, and what would happen.

Type / TipoStructure / EstruturaExample / Exemplo
Zero (general truth)If + present, present"If you heat water, it boils."
First (real future)If + present, will + verb"If we run the trial, we will know the cause."
Second (hypothetical)If + past, would + verb"If we ignored the parameter, we would be wrong."
Cause-effect linksleads to / results in / due to"Activity leads to a better endpoint."

Key point / Ponto-chave: O zero conditional descreve verdades gerais (if + present, present). O first descreve um futuro real (if + present, will). O second descreve uma hipótese (if + past, would). Lembre-se: uma association mostra que duas coisas se movem juntas, mas não prova causation -- pode haver uma confounding parameter escondida.
