Stage 1.3: Grammar in Context

GRAMMAR

Read the short text, then answer the questions. / Leia o texto e responda às perguntas.

Percival will deliver his report to the board. First he greets the audience and states the objective. Then he walks them through the three sections. He does not read every number; he points out only the key figures on the visual and emphasises the two high risks. A short handout helps the board read along. He keeps each sentence concise. For the recommendation he uses the first conditional: if the board approves the control, the risk will drop; if they agree today, the team will start next month.

Stage 1.4: Grammar Tip -- The First Conditional

GRAMMAR

O primeiro condicional é if + present simple (a condição) + will + verbo (o resultado). Usamos para um resultado real e provável no futuro -- ótimo para recomendações numa apresentação: "If we approve this control, the risk will drop". A ordem pode inverter (sem vírgula): "The risk will drop if we approve this control". / The first conditional is if + present simple + will + verb, for a real, likely future result.

Condition / Condição (if + present)Result / Resultado (will + verb)
If we approve this control,the risk will drop.
If the board agrees,we will start next month.
If we check the data,the figures will be accurate.
If I keep it concise,the audience will follow.

Key point / Ponto-chave: depois de if, use o presente simples ("if we approve", não "if we will approve"), e depois de will use o verbo base sem -s ("will drop", não "will drops"). / After "if" use the present simple, and after "will" use the base verb with no -s.
