Chapter 3 -- First Conditional

The First Conditional

If + present, will + verb -- say what will happen

Grammar Notice

Notice the If

What is the same in all of these?

"If we approve this control, the risk will drop."

"If the board agrees, we will start next month."

"If we check the data again, the figures will be accurate."

"The audience will follow easily if I keep it concise."

Grammar Rule

The First Conditional

Condition (if + present)Result (will + verb)
If we approve this control,the risk will drop.
If the board agrees,we will start next month.
If we check the data,the figures will be accurate.
If I keep it concise,the audience will follow.

Key idea: the first conditional is if + present simple (the condition) + will + verb (the result). Use it for a real, likely result in the future -- perfect for recommendations: "If we approve this control, the risk will drop." You can also start with the result: "The risk will drop if we approve this control" (no comma when "if" comes second). After will, the verb has no -s: will drop, not "will drops".

Common Mistake

Watch Out

"If we will approve this, the risk will drops."
"If we approve this, the risk will drop."

After if, use the present simple (approve, not "will approve"), and after will, use the base verb with no -s (will drop, not "will drops").
