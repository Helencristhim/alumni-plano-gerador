Etapa 1.1: Cartoes de Vocabulario

Vocabulary

Estas sao as palavras-chave da aula. Ouca cada uma e leia o exemplo. Toque Listen para ouvir. (Listen = Ouvir)

Progress -- progresso, andamento
"Here is the progress on the order." (Aqui esta o andamento do pedido.)
Finish -- terminar, concluir
"I have finished the report." (Eu terminei o relatorio.)
Complete -- completo, pronto (100%)
"That part is complete now." (Essa parte esta completa agora.)
Already -- ja (antes do esperado)

Yet -- ainda (em frase negativa)
"The deposit has not arrived yet." (O deposito ainda nao chegou.)
On track -- no caminho certo, no prazo
"We are still on track for the deadline." (Ainda estamos no prazo.)
Behind schedule -- atrasado em relacao ao plano
"We are a little behind schedule." (Estamos um pouco atrasados.)
Recently -- recentemente, ha pouco
"The client has confirmed the price recently." (O cliente confirmou o preco recentemente.)
Pending -- pendente, aguardando
"The deposit is still pending." (O deposito ainda esta pendente.)

Grammar

Leia o texto e responda as perguntas. (Read the text and answer.)

But the deposit has not arrived yet, so she will follow up with the bank. The team was a little behind schedule, but now they are back on track for the deadline.

Etapa 1.4: Grammar Tip -- Present Perfect (have/has + participle)

Grammar

Para dizer que uma tarefa esta feita e importa agora, use o present perfect: have/has + participio (sent, signed, finished). Use have com I/you/we/they e has com he/she/it. Use already para feito cedo e not... yet para ainda nao feito. / To say a task is done and matters now, use have/has + past participle. Use already for done early, not...yet for not done now.

Uso / UseForma / FormExample / Exemplo
Feito (I/you/we/they) / Donehave + participioI have sent the report. (Eu enviei.)
Feito (he/she/it) / Donehas + participioShe has signed it. (Ela assinou.)
Feito cedo / Done earlyhave/has + alreadyWe have already sent it. (Ja enviamos.)
Ainda nao / Not done nowhave/has not + ... + yetIt has not arrived yet. (Ainda nao chegou.)
Pergunta / QuestionHave/Has + sujeito + ...?Have you finished it? (Voce terminou?)

Atencao: use 'have' com I/you/we/they e 'has' com he/she/it -- 'I have', nunca 'I has'. Depois de have/has, use o participio (sent, finished, arrived), nunca o verbo base. / Use 'have' with I/you/we/they and 'has' with he/she/it. After have/has, use the past participle.
