Chapter 3: The Code

Have / Has + Done

already for done early · not...yet for not done

Discover the Grammar

What Comes Before the Verb?

"I have sent the report."

"The client has signed the contract."

"The deposit has not arrived yet."

UseFormExample
Done (I/you/we/they)have + past participleI have sent the report.
Done (he/she/it)has + past participleShe has signed it.
Done earlyhave/has + already + ...We have already sent it.
Not done nowhave/has not + ... + yetIt has not arrived yet.
QuestionHave/Has + subject + ...?Have you finished it?

Present perfect = have/has + past participle (sent, arrived, signed, finished). Use it for tasks that are done and matter now. "already" = done early; "not...yet" = not done now.

Common Mistake

Wrong vs Right

✗ WRONG

"I has sent the report."

✓ RIGHT

"I have sent the report."

✗ WRONG

"She has finish the report."

✓ RIGHT

"She has finished the report."

Use "have" with I/you/we/they and "has" with he/she/it. After have/has, use the past participle (sent, finished, arrived), never the base verb.
