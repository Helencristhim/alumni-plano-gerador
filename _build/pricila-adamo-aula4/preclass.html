Grammar

Stage 1.4: Grammar Tip -- Modals of Deduction

Grammar

Para fazer deducoes (adivinhar com base em pistas), use must be (quase certeza de que e verdade), might/could be (e possivel) e can't be (certeza de que NAO e). Para o passado, use must have been / might have been / can't have been + particicio. / To make deductions (guessing from evidence), use must be (almost sure it's true), might/could be (it's possible) and can't be (sure it's NOT true). For the past, use must have been / might have been / can't have been + past participle.

Certeza / CertaintyEnglishPortugues
Almost sure (90%)With those crowds, it must be a famous landmark.Com essa multidao, deve ser um marco famoso.
Possible (50%)It's so empty, it might be off the beaten track.Esta tao vazio que pode ser fora dos roteiros.
Sure it's NOT trueThere's no one here, it can't be a tourist hotspot.Nao tem ninguem aqui, nao pode ser ponto turistico.
Past deductionThis old photo -- the place must have been awe-inspiring.Essa foto antiga -- o lugar deve ter sido deslumbrante.
Common mistakeNOT "It must be beautiful yesterday." Say "It must have been beautiful."Deducao no passado = must have been + particicio.
