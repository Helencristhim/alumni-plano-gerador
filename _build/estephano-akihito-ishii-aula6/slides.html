Chapter 3 -- The Code

Grammar Discovery

some / any

Grammar Discovery

Listen and Notice

Look at the orange words. When do we use some? When any?

"I have some rice." (+)

"I don't have any eggs." (-)

"Do you have any water?" (?)

"There is some bread." (+)

Grammar Rule

Some and Any

FormWordExample
Positive (+)someI have some rice.
Negative (-)anyI don't have any eggs.
Question (?)anyDo you have any water?

Key idea: use some in (+) sentences, and any in (-) sentences and (?) questions.

Common Mistake

Watch Out

"I don't have some eggs."
"I don't have any eggs."

In a negative sentence, use any, not "some".
