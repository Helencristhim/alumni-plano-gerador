Stage 1.4: Grammar Tip -- some / any

GRAMMAR

Usado para falar de quantidade sem número. / Used to talk about quantity without a number.

Form / FormaWord / PalavraExample / Exemplo
Positive (+) / afirmativosomeI have some rice.
Negative (-) / negativoanyI don't have any eggs.
Question (?) / perguntaanyDo you have any water?
Key point / Ponto-chaveUse some no afirmativo e any no negativo e na pergunta. Funciona com palavras incontáveis (rice, water) e plurais (eggs, apples).
