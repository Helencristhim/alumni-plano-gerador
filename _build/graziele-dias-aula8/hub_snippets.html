Asking for Help at Work

Peca ajuda no trabalho com educacao usando "Can": pergunte "Can you help me?", peca para repetir "Can you repeat, please?", peca para falar devagar "Can you speak slowly?" e peca permissao "Can I ask a question?". Regra: Can + you/I + verbo base (sem "to").

Etapa 1.3: Grammar in Context

Grammar

Leia o texto e responda as perguntas. (Read the text and answer.)

In the meeting, Dr. Carter explains a new process. I have a question, so I say: "Can I ask a question?" He speaks fast, so I ask: "Can you speak slowly, please?" I do not hear the last part, so I say: "Can you repeat, please?" There is no problem — when I need something, I just ask: "Can you help me?" Now I understand, and I say thank you.

1. How do you ask for help? (Como pedir ajuda?)
A "Can you help me?" (Voce pode me ajudar?)
B "Can you to help me?" (errado)
C "You help me?" (errado)
2. How do you ask permission to ask? (Como pedir permissao para perguntar?)
A "You can question?" (errado)
B "Can I ask a question?" (Posso fazer uma pergunta?)
C "I ask a question now." (errado)
3. Dr. Carter speaks fast. What do you say? (Ele fala rapido. O que voce diz?)
A "You speak slow?" (errado)
C "Speak more fast." (errado)
B "Can you speak slowly, please?" (Voce pode falar devagar, por favor?)

Etapa 1.4: Grammar Tip -- Asking with "Can"

Grammar

Como pedir de forma educada com "Can". / How to ask politely with "Can".

To do this / Para istoSay / Diga
Pedir ajudaCan you help me?
Pedir para repetirCan you repeat, please?
Pedir para falar devagarCan you speak slowly?
Pedir permissaoCan I ask a question?
A regraCan + you/I + verbo base (sem "to")

Can you...? = Voce pode...? · Can I...? = Posso...? · please = por favor. Cuidado: "Can you help me?" (CERTO), NAO "Can you to help me?" (ERRADO).
