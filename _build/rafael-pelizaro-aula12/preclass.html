Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Target -- a result or number you plan and aim to reach
"We set a target of forty systems in the cloud." (Meta)
Metric -- a number you measure to track success
"Our most important metric is customer satisfaction." (Métrica)
On track -- going well and likely to reach the goal on time
"The project is still on track." (No caminho certo)
Exceed -- to do more than the target you set
"That result has exceeded our goal." (Superar / exceder)
Deliverable -- a finished result or piece of work you deliver
"Two key deliverables have gone live." (Entrega)
Setback -- a problem that slows your progress for a while
"We have had one setback this quarter." (Revés / contratempo)
Recap -- a short summary of the main points or results
"Let me give you a quick recap of the quarter." (Resumo)
Progress -- forward movement; getting closer to the goal
"The team has made real progress." (Progresso)
Objective -- the main goal you are working to achieve
"The migration has been our main objective." (Objetivo)
Achievement -- a good result you have worked hard to reach
"That is a real achievement for the team." (Conquista)

Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

Rafael is getting ready for the quarterly review. He opens with a quick recap of the team's main objective: the Banco Atlas migration. "So far, we have made real progress," he writes. "We have moved thirty-eight of the forty systems, and the project is still on track." Then he shares the numbers. The team set a target of forty systems, and two key deliverables have gone live this quarter. The most important metric, customer satisfaction, has reached eighty-two percent, so it has exceeded the goal. He is honest about the setback too: a security test found a problem, so the rollout has slipped by two weeks. "Even so, the team has made strong progress, and the client has noticed." He uses the present perfect for every result, because each one connects the past to now.

3. What setback does Rafael report?

Stage 1.4: Grammar Tip -- The Present Perfect

GRAMMAR

Usamos o present perfect para falar de resultados e mudanças até AGORA, sem dizer o momento exato. / We use the present perfect for results and changes up to NOW, without saying the exact time.

Form / FormaStructureExample / Exemplo
Affirmative / Afirmativohave / has + past participleWe have moved 38 systems.
Third person / 3ª pessoahas + past participleIt has reached 82%.
Negative / Negativohave / has notThe quarter has not been perfect.
Question / PerguntaHave / Has + subject + participleHave you finished the report yet?

Key point / Ponto-chave: use o present perfect para resultados até agora, com palavras como so far, already, yet e this quarter. Com um tempo passado fechado (last week, yesterday), use o past simple: "We moved 38 systems last week." Depois de have / has, use o particípio passado (gone, não went).

Stage 4: Situational Quiz

Quiz

Choose the best response for each review situation.

The board asks for a quick update on the migration. The best answer is:
A "We are on track. We have moved 38 of the 40 systems so far."
B "We move 38 systems every day."
C "We have moved all the systems last week."
You want to report a metric that beat its goal. You say:
A "Satisfaction is exceed the target."
B "Customer satisfaction has exceeded its target."
C "Customer satisfaction has exceeded its target yesterday."
A colleague asks if the report is ready. The most accurate answer is:
A "I have finished it last Monday."
B "Not yet. I have not added the setback section yet."
C "I am finish the report now."
