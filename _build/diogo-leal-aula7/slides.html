Grammar Rule

Two Leading Tools

ToolExample
Imperative -- Let's + base verbLet's start the meeting.
Imperative -- Please + base verbPlease share your update.
Modal -- Could / Shall (polite request)Could you go first? Shall we begin?
Modal -- Should (recommendation)We should cover three topics.

Key idea: After a modal verb (can, could, should, shall) use the base verb with NO "to": "We should cover" (never "should to cover"). Imperatives also use the base verb: "Please start."

Common Mistake

Watch Out

"We should to cover three topics."
"We should cover three topics."

After a modal verb (can, could, should, shall), use the base verb with no to. The same is true for imperatives: "Please start", not "Please to start".
