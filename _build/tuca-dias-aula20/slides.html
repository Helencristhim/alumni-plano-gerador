From Then to Now

Talking About Change

Grammar Rule

Reflecting: used to / now I can / I have learned to

StructureExampleNote
used to + base verbI used to feel nervous.a past habit that is no longer true
now I can + base verbNow I can ask for directions.an ability you have today
I have learned to + base verbI have learned to adapt.a new skill you gained over time
contrast then & nowI used to plan everything; now I am spontaneous.join past and present with "now"

Key idea: To reflect on your growth, contrast then and now. Use used to + base verb for an old habit ("I used to feel nervous"), now I can + base verb for what you can do today ("Now I can order food in English"), and I have learned to + base verb for a new skill ("I have learned to adapt"). All three take the base verb, never the -ing form.

Common Mistake

Watch Out

"I used to feeling nervous." / "I have learned to adapting."
"I used to feel nervous." / "I have learned to adapt."

After used to, I have learned to, and can, always use the base verb (feel, adapt, ask), never the -ing form.
