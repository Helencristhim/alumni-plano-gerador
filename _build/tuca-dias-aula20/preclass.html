Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

To reflect on how you have grown, you can contrast then and now. Use used to + a base verb for an old habit that is no longer true: "I used to feel nervous at the airport." Use now I can + a base verb for something you are able to do today: "Now I can ask for directions with confidence." And use I have learned to + a base verb for a new skill you gained over time: "I have learned to adapt when plans change." All three take the base verb, never the -ing form. With this confident mindset, you stay resilient and resourceful, you immerse yourself in new places, and you thrive. Looking back, the whole journey was truly worthwhile.

1. Which sentence describes a past habit that is no longer true?

2. What form of the verb follows "used to" and "I have learned to"?

3. How do you say you are able to do something today?

Stage 1.4: Grammar Tip -- used to / now I can / I have learned to

GRAMMAR

Três formas de refletir sobre como você mudou. / Three ways to reflect on how you have changed.

Structure / EstruturaExampleNote / Nota
used to + verbo baseI used to feel nervous.hábito passado que não é mais verdade / a past habit, no longer true
now I can + verbo baseNow I can ask for directions.capacidade que você tem hoje / an ability you have today
I have learned to + verbo baseI have learned to adapt.nova habilidade que você conquistou / a new skill you gained
verbo base sempre / always base verbfeel, adapt, ask (não feeling)nunca use -ing depois dessas estruturas / never -ing

Key point / Ponto-chave: para refletir sobre seu crescimento, contraste o antes e o agora. Use used to + verbo base para um hábito antigo ("I used to feel nervous"), now I can + verbo base para o que você consegue fazer hoje ("Now I can order food"), e I have learned to + verbo base para uma habilidade nova ("I have learned to adapt"). As três usam o verbo base, nunca o -ing. / All three take the base verb, never the -ing form.
