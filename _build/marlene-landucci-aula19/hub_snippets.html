Stage 1.1: Vocabulary Cards

Vocabulary

Ouça cada palavra e leia o exemplo. Toque em Listen para ouvir. / Listen to each word and read the example.

Emergency -- a serious problem that needs help now (emergência)
"This is an emergency." (Isto é uma emergência.)
Call -- to telephone someone for help (ligar, chamar)
"Please call for help." (Por favor, chame ajuda.)
Police -- the people who keep the city safe (polícia)
"Call the police!" (Chame a polícia!)
Ambulance -- a vehicle that takes sick people to the hospital (ambulância)
"Call an ambulance!" (Chame uma ambulância!)
Accident -- a sudden bad event that hurts someone (acidente)
"There is an accident on the street." (Há um acidente na rua.)
Danger -- a situation that is not safe (perigo)
"I am in danger." (Estou em perigo.)
Hospital -- a place where doctors care for sick people (hospital)
"Take me to the hospital." (Leve-me ao hospital.)

Stage 1.4: Grammar Tip -- Call...! / There is...

GRAMMAR

Para uma ordem urgente, comece com o verbo: Call the police! Para dizer o que há, use There is a / an + palavra. Use an antes de som de vogal (an accident, an ambulance, an emergency). / For an urgent order, start with the verb. To say what there is, use "There is a / an + word".

UseExample (EN)Tradução (PT)
OrderCall the police!Chame a polícia!
There is (an)There is an accident.Há um acidente.
I needI need an ambulance.Preciso de uma ambulância.
Question (?)What is your address?Qual é o seu endereço?

Key point / Ponto-chave: comece a ordem com o verbo (Call the police!) e use an antes de som de vogal: an accident, an ambulance, an emergency.
