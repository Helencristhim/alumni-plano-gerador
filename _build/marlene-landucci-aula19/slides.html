The Rule

Call...! / There Is...

UsePatternExample
Give an orderVerb + the/an...Call the police!
Say what there isThere is a / an + wordThere is an accident.
Say what you needI need + wordI need an ambulance.

Key idea: start an urgent order with the verb (Call the police!). To describe, use There is a / an + word. Use an before a vowel sound: an accident, an ambulance, an emergency.

Common Mistake

Watch Out

"You call the police you."
"Call the police!"

For an urgent order, start with the verb: "Call the police!", "Help me!". No "you" is needed.
