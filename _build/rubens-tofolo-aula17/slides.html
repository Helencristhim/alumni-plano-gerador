Chapter 3 -- The Code

Grammar Discovery

Polite requests: Could I...? / Where can I...?

Grammar Discovery

Listen and Notice

How do we ask for things and information politely?

"Could I have a window seat, please?" (polite request)

"Can I take this bag as a carry-on?" (asking permission)

"Where can I find gate twenty-six?" (asking for information)

"Could you tell me when boarding starts?" (asking someone)

Grammar Rule

Could I...? + base verb

FormExample
Request: Could I + base verb?Could I have a window seat, please?
Permission: Can I + base verb?Can I take this bag as a carry-on?
Information: Where can I + base verb?Where can I find gate twenty-six?
Asking someone: Could you + base verb?Could you tell me when boarding starts?

Key idea: after Could / Can / Where can / Could you we use the base verb (have, take, find -- never "to have"). Could is a little more polite than Can, and adding please makes any request softer.

Common Mistake

Watch Out

"Could I to have a window seat?" / "Where I can find the gate?"
"Could I have a window seat?" / "Where can I find the gate?"

After Could I / Can I, use the base verb. In Where can I...?, the verb can comes before I.
