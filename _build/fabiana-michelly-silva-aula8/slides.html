Chapter 3 -- The Code

Grammar Discovery

Mixed conditionals (past cause and present result; present truth and past result)

Grammar Discovery

Listen and Notice

Are the two halves in the same time, or different times?

"If we had used dual-sourcing, we would not be facing a stockout now."

"If our forecasting were better, we would not have missed the demand spike."

"If we had run a post-mortem sooner, we would know the root cause today."

"If the supplier were reliable, the line would not have stopped last week."

Grammar Rule

Mixed Conditionals

StructureUse
If + had + past participle ... would + base (now) -- "If we had double-sourced, we would not be stuck today."A past cause with a present result
If + past simple / were ... would have + past participle -- "If we were more careful, we would not have failed."A present truth with a past result

Key idea: a mixed conditional joins two different times. The if-clause NEVER uses would. Use the past perfect for a finished cause, or were for a present truth.

Common Mistake

Watch Out

"If we would have used dual-sourcing, we would not be stuck."
"If we had used dual-sourcing, we would not be stuck."

The if-clause never takes would. Use the past perfect (had + participle) for a finished cause, or were for a present truth.

Artefact

Post-Mortem Report

BLOOMBERG -- INCIDENT POST-MORTEM

Owner: Fabiana Silva -- Line 4 Stoppage -- Severity: High

1. What happened

A key part ran out during a demand spike. The stockout stopped Line 4 for four days.

2. Root cause

One supplier with no backup -- a single point of failure. We held no safety stock, only a thin stopgap.

3. Fallout

A ripple effect across three regions. In hindsight, the warning signs were clear for weeks.

4. Corrective action

Dual-sourcing on critical parts, a real safety buffer, and a full overhaul of forecasting to build resilience.

Your job as owner: explain what should have happened with a mixed conditional, and name the corrective action that prevents a recurrence.
