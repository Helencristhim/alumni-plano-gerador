Chapter 3

The Code

Mixed conditionals: connecting past and present

Discovery

What Do These Have in Common?

"If we had implemented the system last year, we would not be dealing with these errors today."

past action → present result

"If we were more proactive about audits, we would not have ended up in this situation."

present habit → past result

"If the vendor had run proper inspections, the defective units would not be in our warehouse right now."

past action → present result

"If our audit process were stronger, we would be in a much better position today."

present condition → present result (with past implication)

Grammar Rule

Mixed Conditionals

TypeIF clause (condition)Main clause (result)
Past → PresentIf + had + past participlewould/could + base verb (now)
ExampleIf we had implemented the system, we would not be dealing with errors today.
Present → PastIf + were/was + ...would have + past participle
ExampleIf we were more proactive, we would not have ended up in this situation.
Common Mistake

Watch Out

"If we had implemented the system, we would not have dealt with errors." (3rd conditional -- both about the past)
"If we had implemented the system, we would not be dealing with errors today." (MIXED -- past cause, present effect)

Mixed = different time in each clause. The word "today/now/right now" is your clue that the result is about the present.

Artifact

Lessons Learned Document

LESSONS LEARNED -- APAC Terminal Rollout

Post-Mortem Report -- Q3 Review

Key takeaways:

  1. If we had established clearer quality benchmarks, the defective units would not have passed inspection.
  2. Our corrective action was effective but late. If we were better at early detection, we would catch issues sooner.
  3. The root cause was not just the vendor. If our internal audit process had been stronger, we would be in a better position today.

Corrective actions taken:

  • Recalled all defective units from warehouse
  • Revised inspection protocol with vendor
  • Implemented emergency quality check for remaining rollout

Preventive measures:

  • Quarterly vendor audits with measurable benchmarks
  • New inspection checklist for all incoming shipments
  • Automated alert system for quality deviations

Recommendation: If we follow through on these measures, situations like this would not happen again.

Drafted by: Fabiana Michelly Silva, Supply Chain -- Bloomberg APAC

Spot the Error

Find the Mistake

0 / 4 errors found

"If we had implemented the system, we would not have dealt with errors." (both past = NOT mixed)
"If we had implemented the system, we would not BE DEALING with errors TODAY." (past→present = MIXED)
"If we are more proactive, we would not have ended up here." (present tense in if-clause wrong)
"If we WERE more proactive, we would not have ended up here." (were = hypothetical present)
"If the vendor had ran proper inspections, the units would not be defective."
"If the vendor had RUN proper inspections..." (run = past participle, not ran)
"If our audit process would be stronger, we would have caught the issue."
"If our audit process WERE stronger, we would have caught the issue." (were, not would be in if-clause)
