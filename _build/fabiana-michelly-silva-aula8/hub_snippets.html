Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Root cause -- the fundamental reason behind a problem
"The root cause was a flawed quality control process." (Causa raiz)
Corrective action -- steps taken to fix a problem after it happens
"The corrective action plan was implemented immediately." (Ação corretiva)
Lessons learned -- knowledge gained from analyzing past experiences
"The lessons learned document was shared with all teams." (Lições aprendidas)
Benchmark -- a standard used to measure quality or performance
"We need clearer quality benchmarks for all vendors." (Referência)
Defective -- having a fault that prevents proper function
"The defective units are sitting in our warehouse." (Defeituoso)
Inspection -- a careful examination to check quality
"The defective units should not have passed inspection." (Inspeção)
Audit -- an official examination of records or processes
"We need quarterly vendor audits." (Auditoria)
Preventive -- actions taken to stop a problem before it happens
"The preventive measures include a new checklist." (Preventivo)
Protocol -- a set of official rules or procedures
"We need a revised inspection protocol." (Protocolo)
Rollout -- the process of launching a new system or product
"The APAC terminal rollout had several issues." (Implantação)
Takeaway -- the most important conclusion from a review
"The biggest takeaway is that we need earlier detection." (Conclusão principal)
Flawed -- having a weakness or imperfection
"The vendor's process was fundamentally flawed." (Falho)

Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

The APAC terminal rollout failed due to a flawed quality control process. If the vendor had run proper inspections, the defective units would not be sitting in our warehouse right now. The root cause analysis revealed that the protocol was outdated. If our internal audit process were stronger, we would be in a much better position today. The corrective action was effective, but the team agreed that preventive measures are more important. The key takeaway from the lessons learned document is clear: if we had established clearer benchmarks from the start, this situation would not exist today.

1. What type of conditional is "If the vendor had run inspections, the units would not be sitting here now"?
A Third conditional (both clauses about the past)
B Mixed conditional (past condition, present result)
C Second conditional (hypothetical present)
2. What type is "If our audit process were stronger, we would be in a better position"?
A Third conditional
B First conditional
C Second/mixed conditional (present condition, present result with past implication)
3. What is the difference between corrective and preventive actions?
A Corrective = fix AFTER the problem; Preventive = stop BEFORE it happens
B They mean the same thing.
C Corrective = future plans; Preventive = past actions

Stage 1.4: Grammar Tip -- Mixed Conditionals

GRAMMAR

Mixed conditionals MISTURAM tempos: passado na condição + presente no resultado (ou vice-versa). / Mixed conditionals MIX tenses: past in the condition + present in the result (or vice versa).

Type / TipoIF clauseResult clause
Past → PresentIf + had + past participlewould/could + base verb (agora)
ExemploIf we had implemented the system, we would not be dealing with errors today.
Present → PastIf + were/waswould have + past participle
Ponto-chaveA palavra "today/now/right now" é a pista de que o resultado é PRESENTE. Se o resultado é sobre o passado, use "would have".
