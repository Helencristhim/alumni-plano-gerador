Chapter 3: The Code

I Can Help

can + verb for ability · can't for no ability

Discover the Grammar

What Comes Before the Verb?

"I can help you."

"I cannot open the file."

"Can you help me?"

UseFormExample
Offer / ability (yes)can + base verbI can help you.
No ability (no)cannot / can't + base verbI cannot open the file.
Ask for helpCan + you + base verb?Can you help me?

After "can", use the base verb -- never "can helps" or "can to help". "Cannot" (one word) is the same as the short form "can't".

Common Mistake

Wrong vs Right

✗ WRONG

"I can to help you."

✓ RIGHT

"I can help you."

✗ WRONG

"Can you helping me?"

✓ RIGHT

"Can you help me?"

After "can", use the base verb with no "to" and no "-ing": "can help", never "can to help" or "can helping".
