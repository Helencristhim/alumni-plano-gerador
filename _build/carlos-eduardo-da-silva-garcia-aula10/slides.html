StructureUse it forExample
to be (am/is/are)who / what you areI am an analyst.
present simpleyour routineI check documents.
there is / there areplaces that existThere are many farms.
can / couldability and requestsI can present. Could you repeat?
simple pastlast weekI visited a farm. I found a problem.

Five structures = your whole Block 1. Use the base verb after can and did not. / Cinco estruturas = todo o Bloco 1.

Language Focus

Which Structure?

Common Mistake

Wrong vs Right

✗ WRONG

"Has many farms in my region."

✓ RIGHT

"There are many farms in my region."

✗ WRONG

"The farm did not sent it."

✓ RIGHT

"The farm did not send it."

To say a place exists, use there is / there are. After did not, use the base verb. / "Tem" = there is/are; depois de did not, verbo base.
