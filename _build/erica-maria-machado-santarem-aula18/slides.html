Chapter 3

The Advice Code

You should... · You should not...

Grammar Discovery

Listen and Notice

"You should rest at home."

"You should drink a lot of water."

"You should not go to work today."

"Should I see a doctor?"

FormHow to give advice
Advice (+)should + base verb — You should rest.
Advice (–)should not + base verb — You should not work today.
QuestionShould + I/you + base verb? — Should I see a doctor?
Say how you feelI have a + symptom — I have a headache.
Common Mistake

Right vs Wrong

✗ WRONG

"You should to rest."

✓ RIGHT

"You should rest."

✗ WRONG

"She shoulds rest."

✓ RIGHT

"She should rest."

✗ WRONG

"You do not should work."

✓ RIGHT

"You should not work."

After should, use the base verb (no "to", no "-s"). For the negative, use should not.
