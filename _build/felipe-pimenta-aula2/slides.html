Chapter 4: The Code

The Grammar of Hindsight

If we had known... we would have...

Grammar Discovery

Listen and Notice

"If we had run proper due diligence, we would have seen the risk."

"If the board had modeled the downside, we wouldn't have overpaid."

"We would have kept our runway if we hadn't closed that deal."

"In hindsight, the valuation may have been too optimistic."

The green verbs are in the past perfect. The blue ones are would have + participle. So — did any of it actually happen?

FormUseExample
Third Conditional
If + past perfect, would have + participle		An imaginary past. The condition did NOT happen, so the result did NOT happen either.If we had run diligence, we would have seen it.
Negative
hadn't / wouldn't have		The same imaginary past, the other way round.We wouldn't have overpaid if we hadn't rushed.
Order is freeEither half can come first. Comma only when the if-half opens.We would have kept our runway if we hadn't closed.
Hedging
in hindsight / arguably / may have been		Softens the blame so you stay senior instead of sounding defensive.In hindsight, the valuation may have been optimistic.

Say it out loud as a native does: If we'd run diligence, we'd have seen it. The contraction is not laziness — it is the sound of fluency.

Common Mistake

Right vs Wrong

"If we would have known the risk, we didn't buy the company."
"If we had known the risk, we wouldn't have bought the company."
"If I knew about the tax liability, I would not sign the contract."
"If I had known about the tax liability, I wouldn't have signed the contract."
"In the hindsight, we payed too much."
"In hindsight, we overpaid."

The if-half NEVER takes would. It takes the past perfect (had + participle). The other half takes would have + participle. And the expression is in hindsight — no article.

Chapter 5: In the Room

The Audit Committee Wants a Word

The meeting begins

Dialogue

Explain the Write-Off

D
Felipe, thank you for coming. The committee has read the memo. Before we discuss the numbers, I want to hear it from you: what went wrong?
F
The short answer is that we moved too fast. If we had run proper due diligence, we would have found the tax liability before we signed.
D
There were red flags, though. The auditor's note. Why did nobody stop?
F
In hindsight, we underestimated that note. The valuation looked cheap, and cheap is persuasive. If I had pushed back harder, the deal would have died — and I should have let it die.
D
That is a fair answer. So what protects us next time?
F
Three things: a modeled downside, a contingency of ten percent, and a rule that no deal closes without audited numbers. I have already written them into the process.
Comprehension

About Diana

His own account — she wants to hear what went wrong from him.

The red flags — especially the auditor's note.
3. How does she react to his explanation?
She calls it a fair answer, and asks what protects the company next time.
