Chapter 3 -- The Polite Order

Grammar Discovery

Would like + countable and uncountable nouns

Grammar Discovery

Listen and Notice

How do we order politely? When do we use "many" and when "much"?

"I'd like a table for two, please."

"Would you like some guacamole?"

"How many tacos would you like?"

"How much rice would you like?"

Grammar Rule

Would like + Quantities

ToolHow it works
Would like / I'd likeA polite way to ask for something -- softer than "I want"
Countable nouns (a taco, two tacos)Use a/an, some, many, a few, and "How many?"
Uncountable nouns (rice, water, sauce)Use some, much, a little, and "How much?"

Key idea: to order politely, start with "I would like" or "I'd like". To talk about amount, ask yourself: can I count it? Tacos, yes -- use "many" and "a few". Rice and water, no -- use "much" and "a little".

Common Mistake

Watch Out

"I want tacos. How much tacos can I get?"
"I'd like some tacos. How many can I get?"

Say I'd like to be polite, never "I want." And use How many for countable nouns like tacos, How much for uncountable nouns like rice.
