Grammar Rule

Asking Prices

WordUseExample
How much is it?price of one thingHow much is this apple?
How much are they?price of many thingsHow much are these apples?
this / thatone thing: this (near), that (far)this bag / that bag
these / thosemany things: these (near), those (far)these apples / those bananas

Key idea: use How much is it? for one thing and How much are they? for many. Use this and that for one thing (this = near, that = far), and these and those for many things (these = near, those = far).

Common Mistake

Watch Out

"How much is these apples?" / "these apple"
"How much are these apples?" / "this apple"

Use is for one thing and are for many. Use this/that + one thing and these/those + many things.
