Chapter 3

The Grammar

Past simple with regular verbs (-ed)

Grammar Discovery

Listen and Notice

"I finished the report on Friday."

"We planned the sprint last week."

"She tested the new feature."

"Did you fix the bug? I did not finish it."

StructureUseExample
verb + -edAffirmative (same for all)I finished the report.
did not / didn't + base verbNegative (no -ed)I did not finish it.
Did + subject + base verb?Question (no -ed)Did you fix the bug?
-ed sound/t/, /d/, or /id/fixed /t/, planned /d/, tested /id/

Past simple regular = verb + -ed. After did / didn't, use the base verb (no -ed).

Common Mistake

Right vs Wrong

✗ WRONG

"Did you finished the report?"

✓ RIGHT

"Did you finish the report?"

✗ WRONG

"I no finished the task."

✓ RIGHT

"I didn't finish the task."

✗ WRONG

"Yesterday I fix the bug."

✓ RIGHT

"Yesterday I fixed the bug."

After did/didn't, use the base verb (no -ed). Use didn't for negatives. Add -ed for the past.

Catching Up

Dialogue

Monday Morning

R
Hi Danilo! Happy Monday. How was last week for you?
D
Hi Rachel! It was busy. I finished the data report on Friday.
R
Nice! What else did you do?
D
On Tuesday, I fixed a big bug, and then I tested the new feature.
R
Great. Did you plan the next sprint too?
D
Yes, we planned it on Wednesday, and I organized all the files.
R
Good job. I reviewed your code on Thursday. It looked clean.
D
Thank you! Did you solve the problem with the dashboard?
R
Yes, I solved it on Friday, and I delivered my report to the manager.
D
Perfect. We worked hard last week!
