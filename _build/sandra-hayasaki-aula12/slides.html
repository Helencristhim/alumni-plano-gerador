Chapter 3 -- The Past Habit Code

Past Habits with Used To / Would

How we say what we did again and again, long ago

Listen and Notice

What Shows a Past Habit?

Each sentence is about a past habit. Notice the orange words.

"I used to drink three coffees a day."

"Every weekend, I would walk to the market."

"I never used to work out."

"Did you use to live downtown?"

The Rule

Used to + verb, Would + verb

FormUseExample
used to + base verbpast habits AND past states"I used to live downtown."
would + base verbrepeated past actions only"I would walk every weekend."
did not use to / never used tonegative past habit"I never used to work out."
Did you use to ...?question about a past habit"Did you use to live here?"

Tip: used to works for habits and states. Would works only for repeated actions, not states (say used to be, not would be). In questions and negatives, drop the -d: did you use to....

Common Mistake

States Use Used To, Not Would

"I would be very tired in the mornings."
"I used to be very tired in the mornings."

For past states (be, have, like), use used to, not would. In questions and negatives, drop the -d: Did you use to...?
