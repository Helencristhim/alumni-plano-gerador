Stage 1.3: Grammar in Context

GRAMMAR

Read the short text, then answer the questions. / Leia o texto e responda às perguntas.

After the meeting, Percival writes a follow-up email. He chooses a clear subject: Risk Review, Next Steps. He adds a short greeting, Dear team, and a quick recap. Then he writes: "Please find attached the supplier report. I would be grateful if you could send your comments by Thursday." He attaches the report, adds Daniel in CC, and closes with "Best regards, Percival." Before he sends it, he reads his draft one more time.

B To say the meeting is cancelled.
C To greet the team.
2. How does Percival make a polite request?

B "I would be grateful if you could send your comments by Thursday."
C "You must answer today."
3. Who does Percival add in CC?
A Daniel, so he can see the email too.
B No one.
C Only himself.

Stage 1.4: Grammar Tip -- Polite Email Phrases

GRAMMAR

Num email de trabalho você aponta para um anexo com Please find attached + o arquivo, e faz um pedido educado com I would be grateful if you could + verbo na forma base. Depois de "if you could" usamos o verbo base, sem "to" e sem "-ing". / In a work email you point to a file with Please find attached + the file, and make a polite request with I would be grateful if you could + the base verb. After "if you could" we use the base verb.

Função / FunctionPatternExample
Apontar um anexo / Point to a filePlease find attached + the filePlease find attached the report.
Pedir com educação / Ask politelyI would be grateful if you could + verbI would be grateful if you could reply.
Pedir com educação / Ask politelyI would be grateful if you could + verbI would be grateful if you could confirm.
Fechar o email / Close the emailBest regards, + your nameBest regards, Percival

Key point / Ponto-chave: escrevemos Please find attached (não "find in attach") e, depois de if you could, usamos o verbo base: send, reply, confirm (não "to send" nem "sending"). / We write Please find attached, and after if you could we use the base verb.
