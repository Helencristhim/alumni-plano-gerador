Lesson 19 -- Pre-class

You Should, You Could, You Might -- Modal Verbs for HR Advice

Giving advice and weighing options with modal verbs: should / should not (advice), could (option), and might (possibility), all with the base verb. New words: advisable, alternative, cautious, potential, likelihood, dilemma, weigh up, drawback, worthwhile, feasible, tentative, downside.

Stage 1.4: Grammar Tip -- Modal Verbs (Should / Could / Might)

GRAMMAR

Use should para conselho ou recomendação (you should...), could para opção ou possibilidade (we could...) e might para possibilidade menos certa (they might...). Estrutura: modal + verbo base (sem to, sem -s). Negativo: should not (shouldn't).

UsoEstruturaExemplo
Conselhoshould / should not + base verbYou should talk to HR.
Opçãocould + base verbWe could offer training.
Possibilidademight + base verbThey might agree.
Regra geralmodal + verbo base (sem to, sem -s)She should decide now.

Atenção: depois de should/could/might vem o verbo base, sem to e sem -s. O negativo mais comum de conselho é should not (shouldn't).
