Etapa 1.3: Grammar in Context

Grammar

Leia o texto e responda as perguntas. (Read the text and answer.)

Daniel guides Fernanda through the shipment file on a shared screen. He says: "First, open the trading folder. Then click on the Colombia file." Fernanda opens it. Daniel says: "Scroll down and check the dates. Please do not save it yet." Fernanda checks the dates. Then Daniel says: "Type your name and save the file." Fernanda saves it and says: "Let's review it together tomorrow. Do not send it to the supplier yet."

1. What is the first instruction? (Qual e a primeira instrucao?)
A Open the trading folder. (Abra a pasta de trading.)
B Save the file. (Salve o arquivo.)
C Send it to the supplier. (Envie ao fornecedor.)
2. "Do not save it yet" -- what does it mean? (O que significa?)
A Save it now. (Salve agora.)
B Wait, do not save now. (Espere, nao salve agora.)
C Delete the file. (Apague o arquivo.)
3. "Let's review it together" -- who reviews it? (Quem revisa?)
A Only Daniel. (So o Daniel.)
B Only the supplier. (So o fornecedor.)
C Daniel and Fernanda, together. (Daniel e Fernanda, juntos.)

Etapa 1.4: Grammar Tip -- The Imperative

Grammar

O imperativo da uma instrucao. Comeca com o verbo, sem sujeito. / The imperative gives an instruction. Start with the verb, no subject.

Tipo / TypeForma / FormExample / Exemplo
Faca / Do itverb + ...Open the file. (Abra o arquivo.)
Nao faca / Don'tDo not / Don't + verbDo not save it yet. (Nao salve ainda.)
Educado / PolitePlease + verbPlease click here. (Por favor, clique aqui.)
Juntos / TogetherLet's + verbLet's start now. (Vamos comecar agora.)

Atencao: nunca use 'You' antes do verbo numa instrucao, e use 'Do not' (nao so 'not'). / Do not use 'You' before the verb, and use 'Do not' (not just 'not').
