Chapter 3: The Code

Open. Click. Save.

start with the verb to give an instruction

Discover the Grammar

How Do These Sentences Start?

"Open the file."

"Do not save it yet."

"Let's review it together."

TypeFormExample
Do itverb + ...Open the file.
Do not do itDo not / Don't + verbDo not save it yet.
Be politePlease + verbPlease click here.
Do it togetherLet's + verbLet's start now.

The imperative gives an instruction. We start with the verb and use no subject (no I, you, he). Add "Do not" to stop someone, "Please" to be polite, "Let's" to invite.

Chapter 4: At Work

Dialogue

The Shipment File

D
Hi Fernanda, the shipment file is ready. Let's open it together.
F
Sure. I am sharing my screen. What is the first step?
D
First, open the trading folder. Then click on the file called Colombia.
F
Done. The file is open. Now what?
D
Scroll down to the new dates and check them. Please do not save it yet.
F
Okay, I am scrolling. The dates look good. Should I type something?
D
Yes. Type your name at the bottom, then save the file.
F
Saved. One thing: do not send it to the supplier yet. Let's review it tomorrow.
