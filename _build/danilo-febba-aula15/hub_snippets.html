Describing a Simple Process -- Step by Step at Work

Explique um processo simples de trabalho passo a passo. Palavras-chave: step, process, stage, prepare, save, start, repeat, complete. Gramática: palavras de sequência first / then / next / after that / finally para colocar os passos em ordem (First, I open the file. Then, I check the data. Finally, I save my work.).

Stage 1.4: Grammar Tip -- Sequencing Words (first / then / next / after that / finally)

GRAMMAR

Para descrever os passos de um processo em ordem, use as palavras de sequência. Comece o primeiro passo com First e o último com Finally. (To describe the steps of a process in order, use sequencing words. Start the first step with First and the last with Finally.)

Marcar o... (PT)Estrutura (EN)
primeiro passoFirst, + frase: First, I open the file.
passos do meioThen, / Next, / After that, + frase.
último passoFinally, + frase: Finally, I save my work.
o verbopresente simples: I open, I check, I save.

Atenção (erro comum): para o primeiro passo diga First (não "At first", que significa "no início, com mudança depois"). Para um passo do meio, diga After that (não só "after"). Para o último passo, diga Finally (não "In end"). Use vírgula depois da palavra de ordem.
