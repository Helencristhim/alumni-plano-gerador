Etapa 1.1: Tarjetas de Vocabulario

Vocabulario

Escucha cada palabra y lee el ejemplo. Toca Escuchar para oírla.

Agenda -- la lista de temas de una reunión
"La agenda tiene cinco puntos."
Propuesta -- una idea que presentas para decidir
"Tengo una propuesta para el equipo."
Punto -- cada tema de la agenda
"El primer punto es el presupuesto."
Decisión -- lo que el grupo elige hacer
"Tomamos una decisión juntos."
Aprobar -- decir que sí a una propuesta
"Vamos a aprobar el plan."
Posponer -- dejar algo para más tarde
"Podemos posponer el punto tres."
Sugerir -- proponer una idea con cuidado
"Sugiero empezar por lo urgente."
Acta -- el documento con lo que se decidió
"Valentina escribe el acta."

Etapa 1.3: Gramática en Contexto

Gramática

Lee la reunión y responde las preguntas.

Daniel dirige la reunión: "El primer punto es el envío. Debemos cambiar de proveedor. Sugiero decidir hoy. Sobre el presupuesto, podemos aprobarlo ahora o deberíamos posponerlo. Podríamos votar." Valentina escribe el acta.

1. "Debemos cambiar de proveedor" significa que es...
A Algo necesario o muy recomendable.
B Algo prohibido.
C Algo del pasado.
2. "Podríamos votar" es una forma de...
A Dar una orden fuerte.
B Sugerir con suavidad.
C Negar una idea.
3. "Sugiero decidir hoy" sirve para...
A Cerrar la reunión.
B Proponer una idea al grupo.
C Escribir el acta.

Etapa 1.4: Consejo de Gramática -- Verbos Modales

Gramática

Con deber y poder propones, recomiendas y decides en una reunión.

VerboUsoEjemplo
Debemosrecomendar / obligaciónDebemos decidir hoy.
Podemosposibilidad / permisoPodemos aprobar el plan.
Deberíamossugerir (suave)Deberíamos posponer el punto.
Podríamossugerir (suave)Podríamos votar ahora.
RecuerdaModal + verbo en infinitivo: debemos decidir, podemos aprobar.
