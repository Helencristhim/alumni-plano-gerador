Chapter 3: The Code

Used To, Be Used To, Get Used To

Three structures, three meanings

Grammar Discovery

Listen and Notice

"I used to work in an office, but now I work from home."

"I am used to working from home now."

"At first, it was hard to get used to the silence."

"She is getting used to the new tools."

Which one is a past habit? What comes after used to versus after be/get used to?

StructureMeaningForm & Example
used toa past habit, not true nowused to + base verb — I used to work in an office.
be used toalready accustomed (a state)be used to + noun / -ing — I am used to working alone.
get used tobecoming accustomed (a process)get used to + noun / -ing — I got used to the routine.
Key ruleAfter used to (past habit) use the base verb. After be/get used to use a noun or -ing — never "I am used to work".

Use these three to talk about how work changed, what feels normal now, and what you are still adapting to.

Common Mistake

Right vs Wrong

"I use to work in an office."
"I used to work in an office."
"I am used to work from home."
"I am used to working from home."
"I didn't used to like remote meetings."
"I didn't use to like remote meetings."

Past habit is used to (with -d), but after did not it goes back to use to. After be/get used to, use -ing.

Chapter 4: A New Team

Settling In with Bruno

A welcome chat on Ana's first week

Dialogue

The Welcome Chat

B
Welcome to the team, Ana! How are you settling in so far?
A
Thanks, Bruno. It's a big transition. I used to work in the office every day, so remote is still unfamiliar to me.
B
That's completely normal. I'm used to it now, but honestly, it took me months to get used to the quiet.
A
Good to hear. What helped you adjust the most?
B
A clear routine. Once I got used to blocking focus time, the change felt much easier.
A
That makes sense. I'm slowly getting used to writing things down instead of just talking.
B
Exactly. Small habits, one at a time. Are you used to our stand-up format yet?
A
Not completely, but I'm getting there. Every day it feels less strange.
B
You're adapting fast, Ana. In a month, this will be your new comfort zone.
