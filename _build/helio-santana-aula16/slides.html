Capítulo 3: El Código

Hablar de Duración

llevo · hace... que · desde hace

Descubre la Gramática

Escucha y Observa

"Llevo dos días en Ciudad de México."

"Hace una hora que espero el equipaje."

"Trabajo con Carolina desde hace un año."

"Llevo media hora esperando en el mostrador."

EstructuraUsoEjemplo
Llevar + tiempoCuánto tiempo en una situaciónLlevo dos días aquí.
Llevar + gerundioCuánto tiempo haciendo algoLlevo media hora esperando.
Hace + tiempo + queCuánto tiempo dura algoHace una hora que espero.
Desde hace + tiempoDesde cuándoTrabajo aquí desde hace un año.

Para una duración que sigue ahora, usa "llevo", no "estoy por": "Llevo dos días", no "Estoy por dos días".

Error Común

Correcto vs Incorrecto

✗ INCORRECTO

"Estoy aquí por dos días."

✓ CORRECTO

"Llevo dos días aquí."

✗ INCORRECTO

"Hace una hora espero el equipaje."

✓ CORRECTO

"Hace una hora que espero el equipaje."

Capítulo 7: Cierre

Misión Cumplida

Autoevaluación

Lo Que Aprendí

Puedo hablar de un viaje de negocios: vuelo, aduana, equipaje.
Uso "llevar + tiempo" para la duración (Llevo dos días aquí).
Uso "hace... que" y "desde hace" para el tiempo.
Uso "llevar + gerundio" (Llevo media hora esperando).
Conozco las palabras: vuelo, equipaje, embarque, alojamiento, estancia, mostrador, aduana, itinerario.
