Stage 1.1: Vocabulary Cards

Vocabulary

Ouça cada palavra e leia o exemplo. Toque em Listen para ouvir. / Listen to each word and read the example.

Menu -- the list of food and prices (cardápio)
"Can I see the menu, please?" (Posso ver o cardápio, por favor?)
Waiter -- the person who serves you (garçom)
"The waiter is very friendly." (O garçom é muito simpático.)
Dish -- one kind of prepared food (prato)
"The fish dish is very good." (O prato de peixe é muito bom.)
Drink -- something you drink (bebida)
"Would you like a drink?" (Você gostaria de uma bebida?)
Dessert -- a sweet food after the main dish (sobremesa)
"Would you like some dessert?" (Você gostaria de uma sobremesa?)
Bill -- the paper that shows what you pay (conta)
"Could I have the bill, please?" (Pode trazer a conta, por favor?)
Tip -- extra money for the waiter (gorjeta)
"We leave a small tip." (Deixamos uma pequena gorjeta.)

Stage 1.4: Grammar Tip -- a / some / any

GRAMMAR

Use a/an para uma coisa que dá para contar (a coffee). Use some em frases afirmativas com comida que não dá para contar (some water, some rice). Use any em perguntas e frases negativas (any dessert?, not any bread). / Use a/an for one countable thing; some in positive sentences for uncountable food; any in questions and negatives.

WordExample (EN)Tradução (PT)
a / an (count)I would like a coffee.Eu gostaria de um café.
some (+)Could I have some water?Pode me trazer um pouco de água?
any (?)Do you have any dessert?Vocês têm alguma sobremesa?
any (-)I do not want any dessert.Não quero nenhuma sobremesa.

Key point / Ponto-chave: a/an = uma coisa contável (a coffee). some = afirmativa, não contável (some water). any = pergunta/negativa (any dessert?). Nunca "a water" -- use "some water".
