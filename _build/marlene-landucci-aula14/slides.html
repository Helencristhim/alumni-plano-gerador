Chapter 3 -- Some or Any?

Language Discovery

a / an + some / any

Discovery

Listen and Notice

What do the orange words have in common? Can you count the food, or not?

"I would like a coffee." (one, count)

"Could I have some water?" (cannot count)

"Do you have any dessert?" (question)

"I do not want any dessert." (negative)

The Rule

a / an + some / any

WordWhen to useExample
a / anone thing you can countI would like a coffee.
somepositive (cannot count or plural)Could I have some water?
anyquestions and negativesDo you have any dessert?

Key idea: use a/an for one thing you can count (a coffee). Use some in positive sentences for food you cannot count (some water, some rice). Use any in questions and negatives (any dessert?, not any bread).

Common Mistake

Watch Out

"Could I have a water?"
"Could I have some water?"

We cannot count water, so use some (some water), not "a water". But we can count a coffee: a coffee.
