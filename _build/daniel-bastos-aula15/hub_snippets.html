Etapa 1.1: Tarjetas de Vocabulario

Vocabulario

Escucha cada palabra y lee el ejemplo. Toca Escuchar para oírla.

Negociar -- hablar para llegar a un acuerdo (PT: negociar)
"Vamos a negociar el contrato de café."
Regatear -- discutir el precio para pagar menos (PT: pechinchar)
"El cliente quiere regatear el precio."
Rebaja -- una bajada del precio (PT: desconto, abatimento)
"Te pido una rebaja del cinco por ciento."
Concesión -- algo que das para llegar a un acuerdo (PT: concessão)
"Necesitamos una concesión sobre el precio."
Volumen -- la cantidad total que se compra o vende (PT: volume)
"Si compra más volumen, le hago una rebaja."
Contrapropuesta -- otra oferta que respondes a la primera (PT: contraproposta)
"Tengo una contrapropuesta para usted."
Ceder -- dar un poco de tu posición (PT: ceder)
"Estoy dispuesto a ceder en el plazo de pago."
Margen -- el dinero que ganas tras pagar los costes (PT: margem)
"Esa rebaja es demasiada para mi margen."

Etapa 1.3: Gramática en Contexto

Gramática

Lee la negociación y responde las preguntas.

Daniel negocia: "Si compra más volumen, le hago una rebaja. A cambio de un contrato de dos años, mantengo el precio. Estoy dispuesto a ceder en el plazo de pago. ¿Y si cerramos el trato esta semana?"

1. "Si compra más volumen, le hago una rebaja" sirve para...
A Hacer una oferta condicional.
B Saludar al cliente.
C Cerrar el correo.
2. "A cambio de" sirve para...
A Pedir el acta.
B Ofrecer algo a cambio (un intercambio).
C Describir un gráfico.
3. "Estoy dispuesto a ceder" significa que...
A No doy nada.
B Acepto dar un poco de mi posición.
C Subo el precio.

Etapa 1.4: Consejo de Gramática -- Frases de Negociación

Gramática

Con frases hechas haces ofertas y ofreces algo a cambio en una negociación. (PT: com expressões fixas você faz ofertas e oferece algo em troca.)

FraseUso (PT)Ejemplo
Si... , firmamosoferta condicional (se... , assinamos)Si nos hace una rebaja, firmamos hoy.
A cambio de...ofrecer a cambio (em troca de)A cambio de dos años, mantengo el precio.
Estoy dispuesto a...mostrar lo que aceptas (estou disposto a)Estoy dispuesto a ceder en el pago.
¿Y si...?proponer una idea (e se...?)¿Y si cerramos el trato esta semana?
CuidadoSe dice "a cambio de", no solo "cambio de". (PT: diz-se "a cambio de".)

Etapa 4: Cuestionario Situacional

Cuestionario

Elige la mejor respuesta para cada situación.

El cliente pide una rebaja. Quieres ofrecer algo a cambio. ¿Qué dices?
A "No. El precio es el precio."
B "Si compra más volumen, le hago una rebaja."
C "Adiós."
La rebaja que pide es demasiado alta. ¿Cómo respondes?
A "Es demasiado para mi margen. Tengo una contrapropuesta."
B "Vale, le doy todo."
C "No me importa su problema."
Quieres proponer cerrar el trato pronto. ¿Qué dices?
A "No sé cuándo."
B "¿Y si cerramos el acuerdo esta semana?"
C "El gráfico sube."
