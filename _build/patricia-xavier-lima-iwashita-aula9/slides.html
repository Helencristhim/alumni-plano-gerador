Chapter 3: The Code

If It Fails, We Will...

First conditional · unless · as soon as

Grammar Discovery

Listen and Notice

"If we escalate now, the engineers will start immediately."

"As soon as I have news, I will call the customer."

"Unless we act fast, we will miss the deadline."

"If the fix does not work, we will roll back."

Which verb is in the present, and which uses will?

FormUseExample
First Conditional
If + present, ... will + verb		A real future result that depends on a conditionIf it fails, we will escalate.
as soon as / whenThe time clause stays in the present — never "will"As soon as I have news, I will call.
unlessunless = if not. Already negative — do not add another "not". Unless we act = If we do not act.

The if / as soon as / unless part keeps the verb in the present. Only the result uses will.

Common Mistake

Right vs Wrong

"If the system will go down, we will escalate."
"If the system goes down, we will escalate."
"Unless we don't act now, we will lose the client."
"Unless we act now, we will lose the client."

Keep the verb in the present after if, unless and as soon as. And remember: unless already means "if not", so never add another "not".

Chapter 4: The Critical Call

"The Dashboard Is Down"

A customer needs an escalation, fast

Dialogue

Handling the Pressure

D
Patrícia, this is critical. Our dashboard has been down for an hour, and the impact on my team is huge.
P
I understand, David. I am raising this as an escalation right now. If it is a server problem, our engineers will start immediately.
D
How fast can you move? We have a hard deadline tomorrow morning, and the downtime is costing us money.
P
As soon as I have an update from engineering, I will call you. Unless we hit a blocker, we will have a fix tonight.
D
Thank you. And the severity — is this your top priority?
P
Yes. I am raising it to critical severity, and I make you a clear commitment: a full update every thirty minutes.
