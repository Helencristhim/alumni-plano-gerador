Grammar Rule

Polite Information Questions

FormExample
Ask politely (place)Can you tell me where the tour starts?
Ask politely (time)Can you tell me what time it opens?
Ask politely (length)Can you tell me how long the tour is?
Ask if it existsIs there a shop nearby?
Short answersYes, there is. / No, there isn't.

Key idea: to ask politely, start with Can you tell me + where / what time / how long + normal word order. To ask if something exists, use Is there + a/an + noun.

Common Mistake

Watch Out

"Can you tell me where does the tour start? Is there nearby a shop?"
"Can you tell me where the tour starts? Is there a shop nearby?"

After Can you tell me, keep the normal order: subject + verb.
