Chapter 3

How to Ask

Polite requests and asking the times

Discover the Rule

Listen and Notice

"I would like to book a tour."

"When does the tour start?"

"What time does the museum open?"

"How much is admission?"

StructureUseExample
I would like to + verbA polite requestI would like to book a tour.
When does + it + verb?Ask the time of a tour or placeWhen does the tour start?
What time does + it + open?Ask opening or closing timeWhat time does the museum open?

Use "I would like to..." to ask politely. Use "does" with the time questions.

Common Mistake

Right vs Wrong

✗ WRONG

"I would like book a tour."

✓ RIGHT

"I would like to book a tour."

✗ WRONG

"When the tour starts?"

✓ RIGHT

"When does the tour start?"

After "would like" use to + verb. In time questions use does + the base verb.
