Chapter 3

The Invitation Code

Would you like to...? · Let's...

Grammar Discovery

Listen and Notice

"Would you like to go to the cinema?"

"Let's have dinner together."

"Why don't we go to a concert?"

"Yes, I'd love to. That sounds good!"

WayHow to invite or suggest
Invite (question)Would you like to + base verb? — Would you like to go to the cinema?
Suggest (we)Let's + base verb (no to) — Let's have dinner.
AcceptYes, I'd love to. That sounds good!
RefuseSorry, I'm busy. Maybe next time.
Common Mistake

Right vs Wrong

✗ WRONG

"Would you like go out?"

✓ RIGHT

"Would you like to go out?"

✗ WRONG

"Let's to have dinner."

✓ RIGHT

"Let's have dinner."

✗ WRONG

"Yes, I love to go."

✓ RIGHT

"Yes, I'd love to."

To invite, use Would you like to + base verb. To suggest, use Let's + base verb (no "to"). To accept, say "Yes, I'd love to."
