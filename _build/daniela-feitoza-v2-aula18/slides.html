"If the server goes down, we will escalate the problem." (real, likely)

"If the test passes, we will migrate on Monday." (real plan)

"If we had a bigger budget, we would migrate to the cloud." (imagined, unlikely now)

Chapter 3

The If-Then Rule

First conditional for real plans, second for imagined ones

If-Then Discovery

Listen and Notice

If the server goes down, we will escalate the problem.

If the test passes, we will migrate on Monday.

If we had a bigger budget, we would migrate to the cloud.

If I were you, I would tell the directors now.

The first two are real plans (will). The last two are imagined (would). What changes?

If-Then Rule

Real vs Imagined

TypeStructureExample
First (real)if + present simple, will + base verb"If the test passes, we will migrate."
Second (imagined)if + past simple, would + base verb"If we had more time, we would test more."
Advice formIf I were you, I would + base verb"If I were you, I would escalate it."
Key ideaFirst = real / likely (will). Second = imagined / unlikely now (would). The past form in the 2nd is not the past — it is an imagined present.

After will and would always use the base verb (will go, would go — never "will to go").

Common Mistake

Wrong Form vs Right Form

✗ WRONG

"If the server will go down, we will escalate."

✓ RIGHT

"If the server goes down, we will escalate." (if + present)

✗ WRONG

"If we would have more time, we would test more."

✓ RIGHT

"If we had more time, we would test more." (if + past)

The "if" part takes the present (1st) or past (2nd). Will and would stay only in the result.

2. What will happen if there is an outage during the migration?

Listening

The Coach’s Tip

A voice note about conditionals. Sound only.

0:000:00
1. When do we use the first conditional, and what is the form?
For real, likely situations: if + present simple, then will + base verb, like "if the server goes down, we will escalate the problem."
2. When do we use the second conditional, and what is the form?
For imagined or unlikely situations: if + past simple, then would + base verb, like "if we had a bigger budget, we would migrate everything to the cloud."
3. In the second conditional, is the past form really about the past?
No — it is an imagined present. The past form is used, but it talks about an imagined now, not the real past.
