Warm-up — Callback da Aula 17

Review

Na Aula 17 você aprendeu a falar de quanto tempo algo vem acontecendo com o present perfect continuous: "I have been working on this migration for two years". Aquilo olhou para trás. Agora olhe para a frente: se isto acontecer, o que vai acontecer? E se a situação fosse outra, o que você faria? Hoje você vai ligar uma condição a um resultado: first conditional (if + presente, will) para planos reais e second conditional (if + passado, would) para cenários imaginados. (A condition and a result — real plans, and imagined ideas.)

Stage 1.1: Vocabulary Cards

Vocabulary

Ouça cada palavra, leia o exemplo e a tradução. Toque em Listen para ouvir.

Outage -- a period when a system or service is not working (PT: queda / interrupção do sistema)
"If there is an outage, we will lose access to the data."
Downtime -- the time when a system is offline and cannot be used (PT: tempo de inatividade / fora do ar)
"If we had less downtime, we would migrate faster."
Migrate -- to move data or systems to a new platform (PT: migrar)
"If the test passes, we will migrate the data on Monday."
Back up -- to make a safety copy of data (PT: fazer backup / cópia de segurança)
"If we back up the data first, we will be safe."
Escalate -- to pass a problem to someone with more authority (PT: escalar / encaminhar para um nível superior)
"If the issue is serious, I will escalate it to the director."
Prioritize -- to decide which task is the most important to do first (PT: priorizar)
"If time is short, we will prioritize the data first."
Scenario -- a possible situation that you imagine or plan for (PT: cenário)
"In this scenario, if the system fails, what would we do?"
Worst-case -- the worst possible situation you prepare for (PT: pior cenário possível)
"In the worst-case scenario, we would roll back the system."

Stage 1.3: Grammar in Context

GRAMMAR

Leia o texto e responda às perguntas.

The migration plan is full of conditions. If the test passes, the team will migrate on Monday — this is a real, likely plan, so we use the first conditional: if + present simple, will + base verb. If there is an outage, they will escalate the problem and roll back the system. But there are also imagined ideas. If they had more downtime allowed, Marcus would do the whole thing in one weekend — this is the second conditional: if + past simple, would + base verb, for a situation that is not real now. And for advice he says, "If I were you, I would tell the directors today." Notice that "will" and "would" never go inside the "if" part.

1. Why is "If the test passes, we will migrate" a first conditional?
A It is a real, likely plan: if + present, will + base verb.
B It is an imagined, unlikely situation.
C It is something that happened in the past.
2. "If they had more downtime, Marcus would do it in one weekend" is...
A A first conditional (real plan).
B A second conditional (imagined, not real now).
C A past habit with "used to".
3. Which sentence is correct?
A "If the server will go down, we will escalate."
B "If the server goes down, we will escalate."
C "If we would have time, we would test more."

Stage 1.4: Grammar Tip — First & Second Conditionals

GRAMMAR

Como ligar uma condição a um resultado (bilingue EN + PT).

TipoStructureExample
First (real)if + present simple, will + verbo base"If the test passes, we will migrate."
Second (imaginado)if + past simple, would + verbo base"If we had more time, we would test more."
AdviceIf I were you, I would + verbo base"If I were you, I would escalate it."
Regra (PT)Use o first conditional (if + presente, will) para situações reais e prováveis. Use o second conditional (if + passado, would) para situações imaginadas / improváveis agora — o passado aí não é passado de verdade, é um "agora imaginado". NUNCA ponha will ou would dentro do if: o if leva presente (1st) ou passado (2nd).

Aula 18 — Conditionals I & II: If This, Then That

Conteúdos para reforçar os condicionais — first conditional (if + presente, will) para planos reais e second conditional (if + passado, would) para cenários imaginados. Marque como concluído ao terminar.

Series
Mr. Robot — any crisis or "what if the system goes down" scene

Characters plan around outages and worst-case scenarios in real time. Listen for first conditional plans ("if this happens, we will...") and imagined what-ifs ("if we had..., we would...") — the exact forms from Lesson 18.

Dica: anote uma frase com "will" (plano real) e uma com "would" (cenário imaginado).

Podcast
The Cloudcast / Google SRE — "incident response & disaster planning" episodes

Engineers discuss what they will do if an outage happens and what they would do in a perfect scenario. Hear the first conditional for real escalation plans and the second conditional for hypothetical "if we had more budget" ideas.

Dica: pause e repita cada frase com "if... will" ou "if... would" que você ouvir.
