Chapter 3 -- The Emphasis Code

Grammar Discovery

Emphasis for persuasion -- making your point land

Grammar Discovery

Listen and Notice

What are the orange words? How do they make the idea stronger?

The tool is not only faster, but also more cost-effective.

What we really need is a faster rollout, not a bigger budget.

She did not only recommend it, but also named the downside.

Neutral: "We need a faster rollout." (less emphasis)

Grammar Rule

Emphasis for Persuasion

StructureUseExample
not only X, but also Yjoin two points, give weight to both"not only faster, but also cheaper."
X and Y same formbalance both sides"not only recommend it, but also name the risk."
What we need is + nounpush the main idea to the front"What we need is a faster rollout."
Neutral sentenceless emphasis, plain order"We need a faster rollout."

Key idea: use not only X, but also Y to give weight to two points (keep X and Y in the same form), and What we need is... to push your main idea to the front. Both structures add emphasis, so your recommendation lands and you are more likely to win support.

Common Mistake

Watch Out

"It is not only faster, but also it saves money."
"It is not only faster, but also cheaper."

After not only, keep both parts in the same form (adjective + adjective, verb + verb). This balance makes your point sound clear and strong.

Grammar Practice

Make It Stronger

"The tool is not only faster, _____but also more cost-effective." (add emphasis)
_____What we need is a faster rollout, not a bigger budget." (push the idea to the front: We need a faster rollout.)
"She did not only recommend it, _____but also named the downside." (balance the verbs)
_____What this team needs is clear buy-in from leadership." (emphasize: The team needs buy-in.)
