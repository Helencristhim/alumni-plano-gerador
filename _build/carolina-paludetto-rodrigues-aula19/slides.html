Grammar Rule

Talking About Quantity

WordUseExample
How manythings you can countHow many apples?
How muchthings you can't countHow much sugar?
some / anysome (+), any (- and ?)some milk / not any juice
a lot ofa big quantity (both)a lot of fruit
a fewa small number (count)a few nuts
a littlea small amount (can't count)a little honey

Key idea: use How many + countable things (apples), and How much + uncountable things (sugar, water). Use a few for a small number (a few nuts) and a little for a small amount (a little honey). Use some in positive sentences and any in negatives and questions.

Common Mistake

Watch Out

"How much apples?" / "a few sugar"
"How many apples?" / "a little sugar"

Use How many + countable (apples) and How much + uncountable (sugar). Use a few for countable, a little for uncountable.
