Chapter 3

The Scene Rule

Past continuous for the scene, past simple for the moment

Scene Discovery

Listen and Notice

I was testing the system when it crashed.

While we were deploying, the network went down.

The team was watching the monitor when an alert appeared.

At eleven, I was monitoring the dashboard.

Which action sets the scene? Which one is the moment that interrupts it?

Scene Rule

Scene vs Moment

FormUseExample
was / were + verb-ingThe long action in progress (the scene)"I was testing the system."
past simpleThe short action that interrupts (the moment)"...when the server crashed."
while + long actionIntroduces the action in progress"While we were deploying..."
when + short actionIntroduces the interruption"...when the network went down."

Form: I / he / she / it = was; you / we / they = were; + verb + -ing.

Common Mistake

Wrong Form vs Right Form

✗ WRONG

"I was test the system when it crashed."

✓ RIGHT

"I was testing the system when it crashed." (was + verb-ing)

✗ WRONG

"I were monitoring the dashboard."

✓ RIGHT

"I was monitoring the dashboard." (I = was)

Past continuous = was/were + verb + -ing. Use "was" with I/he/she/it, "were" with you/we/they.

The Night It Crashed

Daniela and Marcus review the incident

Dialogue

Reviewing the Incident

M
Daniela, what was happening when the store froze last night?
D
We were monitoring the dashboard when an alert appeared.
M
And the update? Was it still running at that moment?
D
Yes. While the update was loading, the response time started to lag.
M
So what caused the freeze in the end?
D
A glitch in the payment step occurred and interrupted the checkout.
M
How did the team react while it was down?
D
While I was rebooting the server, Carla called the vendor. We noticed it early.
Comprehension

Did You Catch It?

1. What was the team doing when the alert appeared?
They were monitoring the dashboard (the long action) when the alert appeared (the moment).
2. What happened while the update was loading?
The response time started to lag, and then a glitch in the payment step occurred and interrupted the checkout.
3. What were Daniela and Carla doing while the system was down?
Daniela was rebooting the server while Carla was calling the vendor; they noticed the problem early.
Listening

The Coach’s Tip

A voice note about the past continuous. Sound only.

0:000:00
1. According to Sarah, what is the past continuous used for?
For a longer action that was already happening in the past, like "We were testing the new system."
2. What happens to each action when a short one interrupts the long one?
The long action stays in the past continuous and the short action goes to the past simple: "We were testing the system when the server crashed."
3. Which words come before the long action and the short action?
"While" comes before the long action and "when" comes before the short one.
