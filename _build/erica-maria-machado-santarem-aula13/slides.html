Chapter 3

The Code

Like + -ing — Want + to

Grammar Discovery

Listen and Notice

"I like reading books."

"I enjoy cooking on Saturday."

"I want to walk in the park."

"I want to learn to dance."

VerbStructureExample
like / love / enjoy+ verb + -ingI like reading.
want+ to + verbI want to walk.
RememberAfter like = -ing. After want = to. Never both.
Common Mistake

Careful Here

"I like to reading."
"I like reading." (after like = -ing)
"I want walk in the park."
"I want to walk in the park." (after want = to)
"I like read books."
"I like reading books." (after like = -ing)

After like use -ing. After want use to. Never both together.

Practice
