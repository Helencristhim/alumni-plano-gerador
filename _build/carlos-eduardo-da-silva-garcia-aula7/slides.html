Chapter 3: There Is / There Are

There Is, There Are

there is a warehouse · there are many farms

Discover the Grammar

Listen and Notice

"There is a warehouse near the road."

"There are many farms in the region."

"Is there a hotel in Varginha?"

"There is not an office in Virginia."

FormUse it forExample
There isone thing (singular)There is a warehouse.
There aremany things (plural)There are many farms.
Is there? / Are there?questionsIs there a hotel? Are there farms?
There isn't / aren'tnegativeThere isn't an office here.

In Portuguese we say "tem"; in English use there is / there are, NOT have/has. / Em portugues "tem" -> em ingles there is / there are.

Common Mistake

Wrong vs Right

✗ WRONG

"Has a warehouse near the road."

✓ RIGHT

"There is a warehouse near the road."

✗ WRONG

"There is many farms in the region."

✓ RIGHT

"There are many farms in the region."

To say a place exists, use there is / there are, not have/has. Use there are for plural. / "Tem" = there is / there are.
