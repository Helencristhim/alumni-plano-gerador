Stage 1.4: Grammar Tip -- Past Simple (Irregular Verbs)

GRAMMAR

Usado para falar de ações terminadas no passado. Verbos irregulares NÃO adicionam -ed. / Used to talk about finished past actions. Irregular verbs do not add -ed.

Base verbPast SimpleExample
eatateI ate a delicious dish.
drinkdrankWe drank fresh juice.
have / makehad / madeShe made a dessert; we had lunch.
go / givewent / gaveWe went out; he gave me the recipe.

Key point / Ponto-chave: verbos irregulares NÃO usam -ed. Cada um tem uma forma especial para memorizar: eat → ate, drink → drank, have → had, make → made, go → went, give → gave. Os sinais de passado são os mesmos: yesterday, last night, last weekend.
