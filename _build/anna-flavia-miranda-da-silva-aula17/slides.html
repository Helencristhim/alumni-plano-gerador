Chapter 3: Offers & Conditions

The Code

If you..., I can...

Discover the Code

What Word Comes First?

If you buy more, I can give a discount.

If you pay now, we can deliver fast.

Would you accept this offer?

Would you like a better price?

StructureExample
If + present, can + verbIf you buy more, I can give a discount.
If + present, will + verbIf you pay now, we will deliver fast.
Would you + verb...?Would you accept this offer?

For a condition, start with If + present (you buy), then use can or will for the result. To make a polite offer, use Would you...?

Common Mistakes

Wrong vs Right

✗ WRONG

"If you will buy more..."

✓ RIGHT

"If you buy more..."

✗ WRONG

"I give discount."

✓ RIGHT

"I can give a discount."

✗ WRONG

"You accept this offer?"

✓ RIGHT

"Would you accept this offer?"

Remember: after If use the present (you buy), and offer politely with Would you...?
