Language Focus

If This, Then That

The first conditional for real future results, plus modals to offer help.

Discovery

Listen and Notice

"If the part is missing, we will order a new one." (if + present)

"If we ship today, it will arrive on Monday." (will + base verb)

"If they confirm by Friday, we will keep the schedule." (real future)

"I can look into it, and I will fix it today." (modal -- offer help)

Language Rule

The First Conditional

PartFormExample
Condition (if)If + present simpleIf the part is missing,
Resultwill + base verbwe will order a new one.
Other orderresult + if + presentWe will ship it if they confirm.
Offer helpcan / will + base verbI can look into it. I will fix it.

Key idea: The first conditional talks about a real, possible future. The if part uses the present simple (If it arrives...), and the result uses will + base verb (we will start...). NEVER use will in the if part. To offer help, use can or will + base verb.

Watch Out

Watch Out

✗ "If the part will be missing, we will order a new one."

✓ "If the part is missing, we will order a new one." (if + present)

✗ "If they will confirm by Friday, we keep the schedule."

✓ "If they confirm by Friday, we will keep the schedule." (present + will)

Artefact

The Apology Email

To: Training Team  ·  From: Andreia Heins  ·  Subject: Update -- Console Delay & Plan

Dear team,

Apology: I apologize for the delay and the inconvenience this has caused.

Problem: The supplier had a missing part, so the new console is delayed by two weeks. I know this is urgent.

Solution: If the new date slips again, we will use a loan unit as a backup, so practice will not stop.

Reassurance: No real surgery is affected. If they confirm by Friday, we will keep the original schedule, and I will send a short update every week.
