Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Complaint -- when someone says they are not happy about a problem
"A surgeon filed a complaint about the delay." (Reclamação)
Apologize -- to say you are sorry about a problem or mistake
"I apologize for the inconvenience." (Pedir desculpas)
Inconvenience -- a small trouble or difficulty caused to someone
"Sorry for the inconvenience this caused you." (Transtorno / incômodo)
Delay -- when something happens later than planned
"The console has a two-week delay." (Atraso)
Urgent -- so important that it needs action right now
"This is urgent for the training team." (Urgente)
Reassure -- to make someone feel calm and less worried
"Let me reassure you: we have a plan." (Tranquilizar)
Solution -- a way to fix a problem or end a difficulty
"Let me offer a clear solution." (Solução)
Backup -- a second plan or unit you use if the first one fails
"We have a backup unit on loan." (Reserva / plano B)
Refund -- money given back when something goes wrong
"If the part is faulty, we will ask for a refund." (Reembolso)
Replacement -- a new item that takes the place of a broken or missing one
"They will send a replacement unit." (Substituição)

Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

The new console is delayed by two weeks, and a surgeon has filed a complaint. The matter is urgent, so Andreia calls to handle it. First she apologizes: "I apologize for the delay and the inconvenience." Then she offers a solution using the first conditional: "If the part is missing, we will order a new one." Note: the if part uses the present simple (is), and the result uses will + base verb (will order). She also explains the backup: "If the date slips again, we will use a loan unit." If the part is faulty, the supplier will send a replacement or give a refund. Finally, she wants to reassure the team: "If anyone is worried, please come and talk to me."

1. Andreia says "If the part is missing, we will order a new one." Why "is", not "will be"?
A In the first conditional, the if part uses the present simple.
B Because the part is already missing in the past.
C Both "is" and "will be" are correct here.
2. In the result part of a first conditional, which form do we use?
A The present simple: "we order a new one."
B Will + base verb: "we will order a new one."
C The past simple: "we ordered a new one."
3. What does Andreia do first when she handles the complaint?
A She apologizes for the delay and the inconvenience.
B She blames the supplier and ends the call.
C She says nothing about the problem.

Stage 1.4: Grammar Tip -- The First Conditional

GRAMMAR

Como falar de um resultado futuro real que depende de uma condição. / How to talk about a real future result that depends on a condition.

Parte / PartForma / FormExemplo / Example
Condição (if)If + presente simplesIf the part is missing,
Resultadowill + verbo basewe will order a new one.
Ordem invertidaresultado + if + presenteWe will ship it if they confirm.
Oferecer ajudacan / will + verbo baseI can look into it. I will fix it.

Atenção / Note: O first conditional fala de um futuro real e possível. A parte do if usa o presente simples (If it arrives...) e o resultado usa will + verbo base (we will start...). NUNCA use will na parte do if. Para oferecer ajuda, use can ou will + verbo base. / In the first conditional, the if part uses the present simple and the result uses will + base verb. Never use will in the if part.
