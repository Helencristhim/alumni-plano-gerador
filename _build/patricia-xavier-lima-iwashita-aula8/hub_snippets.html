Handling a Technical Issue

Report a technical issue and its result like a customer success manager. New words: issue, bug, ticket, root cause, workaround, resolve, escalate, log, reproduce, fix. Grammar: present perfect for results ("we have fixed", "it has resolved").

Stage 1.1: Vocabulary Cards

Vocabulary

Ouça cada palavra e leia o exemplo. Toque em Listen para escutar.

Issue -- a problem that stops something from working (problema/falha)
"We have identified the issue."
Bug -- an error in the software that makes it behave wrongly (erro de software)
"The team has fixed the bug."
Ticket -- a record of a customer problem in the support system (chamado)
"I have opened a ticket for you."
Root cause -- the real, underlying reason a problem happens (causa raiz)
"We have found the root cause."
Workaround -- a temporary way to keep working around a problem (solução temporária)
"We have shared a quick workaround."
Resolve -- to solve a problem completely so it does not return (resolver)
"Support has resolved the problem."
Escalate -- to pass a problem to a higher or more expert team (escalar/encaminhar)
"I have escalated this to our engineers."
Log -- a record of what the system has done, used to find errors (registro/log)
"We have checked the server logs."
Reproduce -- to make the same problem happen again, to study it (reproduzir)
"We have reproduced the error."
Fix -- a permanent solution that repairs the problem for good (correção/conserto)
"We have deployed a fix."

Stage 1.3: Grammar in Context

GRAMMAR

Leia o texto e responda às perguntas.

A customer opened a ticket: users could not log in. Patrícia has escalated the issue to the engineers. So far, the team has reproduced the bug and has checked the logs. They have found the root cause — a recent update changed one setting. This morning they have applied a fix, and support has resolved the problem. They tested it twice an hour ago, and the issue has not come back.

1. Por que "the team has reproduced the bug" usa o present perfect?
A Porque aconteceu num dia específico no passado.
B Porque é um resultado que importa agora (já foi feito).
C Porque é uma rotina de toda semana.
2. "They tested it twice an hour ago" usa o past simple porque:
A Há um tempo passado exato e fechado ("an hour ago").
B É um resultado sem hora definida.
C É uma ação acontecendo neste momento.
3. Which sentence reports a result correctly?
A "Support has resolved the problem yesterday."
B "Support has resolved the problem."
C "Support has resolve the problem."

Stage 1.4: Grammar Tip -- Present Perfect for Results

GRAMMAR

Como reportar um resultado que importa agora (explicação em inglês e português).

Form / FormaUse / UsoExample
Affirmative
have / has + particípio		Resultado que importa agora; sem hora exata. A result that matters now.We have fixed the bug.
Negative
have / has not + particípio		O problema não voltou. Something has not happened.The issue has not come back.
Question
Have / Has + sujeito + particípio		Perguntar sobre um resultado; com yet/already. Asking about a result.Have you checked the logs yet?
Past Simple
verbo-ed (contraste)		Hora passada exata e fechada. Finished past time (yesterday, an hour ago).We tested it an hour ago.

Use o present perfect com just, already, yet, so far (sem hora exata). Depois de have/has vem o particípio: fixed, resolved, found, run (NUNCA "have ran" nem "has fix").

Stage 4: Situational Quiz

Quiz

Escolha a melhor resposta para cada situação real de trabalho.

The customer asks for a status. The bug is solved now. You say:
A "Good news — we have fixed the bug."
B "Good news — we have fix the bug."
C "Good news — we fixing the bug."
You give the result with an exact past time. Which is correct?
A "We have resolved it an hour ago."
B "We resolved it an hour ago."
C "We are resolving it an hour ago."
The issue is still open. You give an honest status:
A "We have reproduced the bug and escalated it. A fix is coming."
B "We have reproduce the bug and escalate it."
C "We reproducing the bug since this morning."
You want to ask the engineer about progress. The best question is:
A "Have you found the root cause yet?"
B "Have you found the root cause already yesterday?"
C "Did you have found the root cause?"
