Incident Report #4821 — Login Failure on the Atlas Dashboard

On Monday morning, the Atlas team opened a support ticket. Several users could not log in to their dashboard, and the system showed the same error again and again. For about two hours, the issue stopped the sales team from working, so the customer marked it as urgent. The account manager has read the ticket and has escalated it to our technical team.

The engineers have worked on the problem all morning. First, they have reproduced the bug on a test account, so they could see exactly what the user saw. Then they have checked the server logs, and the logs have shown the real root cause: a recent update changed one security setting by mistake. While the team looked for a permanent fix, they have also shared a simple workaround, so most users could log in again.

By the afternoon, the team has resolved the issue. They have applied a small fix to the security setting and have tested it twice. The account manager has called the customer to confirm that everything works again. The login problem has not come back, and the team has added a note to the ticket so the same mistake does not happen next time.

Adapted for class use from a customer support incident report

Chapter 4: The Code

Results with the Present Perfect

Grammar Discovery

Listen and Find the Pattern

"We have identified the issue."

"The team has fixed the bug."

"Support has resolved the problem."

"We have deployed a fix this morning."

FormUseExample
have / has + past participlea result that matters NOW (no exact past time)We have fixed the bug.
has + past participlethird person: it / the team / supportIt has resolved the error.

Use the present perfect for results, with words like just, already, yet and so far. The verb after have / has is the past participle (fixed, resolved, found, run).

Common Mistake

Wrong vs Right

We have resolved the issue yesterday.
We resolved the issue yesterday.

With a finished past time (yesterday, last week, an hour ago), use the past simple, not the present perfect.

We have ran the test twice.
We have run the test twice.

After have / has, use the past participle (run, not ran; seen, not saw).

Chapter 5: On the Support Call

The Status Update

Dialogue

The Support Call

D
Hi Patrícia, thanks for the quick call. Can you give me an update on the login ticket?
P
Of course. Good news — we have resolved the issue. The dashboard is working again.
D
That is a relief. What was the root cause?
P
A recent update changed one security setting. We have reproduced the bug and checked the logs to be sure.
D
And it will not happen again?
P
We have applied a fix and tested it twice. I have also added a note to the ticket for the team.
Comprehension

About the Call

1. What has the team done about the login issue?
They have resolved it; the dashboard works again.
2. What was the root cause, according to the call?
A recent update changed one security setting.
3. How did the team make sure the fix works?
They applied a fix and tested it twice.
