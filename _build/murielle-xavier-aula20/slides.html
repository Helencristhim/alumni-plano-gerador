StructureFormExample
Comparatives-er than / more ... thanlower than, more efficient than
Superlativesthe ...-est / the mostthe highest, the most efficient
Quantifiersmany/much, a few/a littlemany roles, much data
Past continuouswas/were + verb-ingwe were preparing the report
Modalsshould / could / might + base verbwe should decide now
Common Mistake

Right vs Wrong

✗ WRONG

Turnover is more lower than last year.

✓ RIGHT

Turnover is lower than last year.

✗ WRONG

It is the most best result.

✓ RIGHT

It is the best result.

✗ WRONG

We have much vacancies.

✓ RIGHT

We have many vacancies.

✗ WRONG

We should to decide now.

✓ RIGHT

We should decide now.

Block 2 in one glance: -er than (compare), the best (rank), many/much (quantify), was/were + -ing (narrate), should + base verb (advise).
