Chapter 3 -- The Pattern

Grammar Discovery

asked + if / wh-word + statement order

Grammar Discovery

Listen and Notice

What changes when the direct question becomes a report?

"Where do you live?" → "She asked where I lived." (wh-, no do, verb back)

"Are you ready?" → "He asked if I was ready." (yes/no → if)

"What does he want?" → "She wondered what he wanted." (no does)

"Did they win?" → "They inquired whether we had won." (no did, verb back)

Grammar Rule

asked + if / wh- + statement

TypeDirect questionReported question
Yes/No"Are you ready?"She asked if I was ready.
Yes/No"Did they win?"She asked whether they had won.
Wh-"Where do you live?"She asked where I lived.
Wh-"What does he want?"She asked what he wanted.

Key idea: a reported question uses statement order (subject + verb), drops do / does / did, usually shifts the verb one step back (live → lived, want → wanted), and takes if or whether for yes/no questions or the wh-word for wh-questions. There is no question mark.

Common Mistake

Watch Out

"She asked where did I live?" (inversion + did + ?)
"She asked where I lived."

A reported question uses statement order (subject before verb), no do/does/did, and no question mark. It only looks like a question on the surface.

Comprehension

About the Tips

1. What word do we add to report a yes/no question?

"if" or "whether", for example "she asked if I was ready".

2. What happens to the word order in a reported question?

It goes back to normal statement order, so we say "where I lived", not "where did I live".

3. Why is there no question mark at the end?

Because a reported question is really a statement, not a direct question.
