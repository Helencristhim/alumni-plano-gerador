Chapter 3 -- The What-If Code

Language Discovery

The second conditional

Discovery

Listen and Notice

What do the highlighted words have in common? Is each situation real, or imaginary?

"If we acquired the Pittsburgh plant, we would double our capacity."

"If the board approved the budget, we could expand next year."

"If I were you, I would prioritize the Houston scenario."

"What would you do if the deal fell through?"

The Rule

The Second Conditional

FunctionPatternExample
Condition (if-clause)if + past simple"If we acquired the plant..."
Resultwould / could / might + base verb"...we would double our capacity"
Were for all subjectsif + I/he/she/it + were"If I were you, I would wait"
QuestionWhat + would + subject + base verb"What would you do if the deal fell through?"

Key idea: the second conditional is for hypothetical or unlikely situations. Use past simple in the if-clause and would / could / might + base verb in the result. This is different from the first conditional (if + present, will), which describes a real condition. Use were for all subjects: "If I were you...".

Diplomatic Proposals

Soften It with If I Were You

The second conditional makes advice and proposals more diplomatic. Use were for all subjects.

"If I were you, I would prioritize the Sao Paulo operation."

"If we were in their position, we would ask for more time."

"If the timeline were shorter, we might reconsider the budget."

Common Mistake

Watch Out

"If I would acquire the plant, we would double capacity."
"If I acquired the plant, we would double capacity."

Never put would in the if-clause. The condition uses the past simple: "if I acquired". The result uses would / could / might: "I would double".

Practice

Build the Conditional

"If we _____acquired the Pittsburgh plant, we would double our capacity." (if + past simple)
"If the board approved the budget, we _____could expand next year." (could + base verb)
"If I _____were you, I would wait for the forecast." (were for all subjects)
"_____What would you do if the deal fell through?" (question form)
