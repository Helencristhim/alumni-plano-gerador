Aula 13 -- Pre-class

Making Proposals -- Pitching Ideas to the Board

Pitch an idea to your partners in English: lead with the rationale, keep the scope tight, and propose a low-risk pilot with a clear value proposition. Use proposing frames ("I would like to suggest that we...", "Why don't we...?", "What if we tested it?", "I propose that we...") and report the outcome in the passive voice (the proposal was approved, the budget has been allocated). New vocabulary: pitch, rationale, scope, rollout, pilot, buy-in, value proposition, deliverable, viable, streamline, incentive, secure.

Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Pitch -- to present an idea to people in order to win their support
"On Thursday I will pitch the idea to the board." (apresentar / fazer um pitch)
Rationale -- the set of reasons that explain why an idea makes sense
"Lead with the rationale before any numbers." (justificativa)
Scope -- how large a project is -- what it includes and leaves out
"Keep the scope tight: one site, not all of them." (escopo)
Rollout -- the full launch of a plan or system to all users or sites
"A full rollout is risky, so we will start smaller." (implantação total)
Pilot -- a small first test of an idea before a full rollout
"What if we ran a three-month pilot at one site?" (projeto piloto)
Buy-in -- the agreement and support of the people who must approve a plan
"Secure buy-in from David before the meeting." (adesão / apoio)
Value proposition -- a short, clear statement of the benefit the partner gets
"Our value proposition: same output, lower cost." (proposta de valor)
Deliverable -- a concrete result a project must produce by a set date
"Each phase has a clear deliverable and a deadline." (entregável)
Viable -- realistic enough to work in practice and be worth doing
"The pilot shows the plan is viable and low-risk." (viável)
Streamline -- to make a process simpler, faster, and more efficient
"The plan would streamline approvals across both sites." (otimizar / simplificar)
Incentive -- a reason or reward that motivates someone to act or agree
"The cost saving is a strong incentive for the partners." (incentivo)
Secure -- to obtain or lock in something important, such as support or funding
"We need to secure approval and the budget." (garantir / assegurar)

Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

On Thursday I pitch my idea to the partners. I will lead with the rationale: the order process is slow and it costs us margin. Then I propose the plan. I would like to suggest that we run a three-month pilot at one site, with clear deliverables. What if we measured the cost saving before we scale? I propose that we keep the scope tight, because the board gets nervous with a full rollout. The value proposition is simple: the same output at a lower cost. That is the incentive, and the plan is viable and low-risk. If the pitch goes well, the proposal will be approved and the budget will be allocated -- and last year a similar pilot was approved in a single meeting. My job is to secure the partners' buy-in.

1. How does the writer introduce the pilot?
A "I would like to suggest that we run a pilot." -- a proposing frame.
B "We will run a pilot, end of story."
C "Maybe a pilot, I don't know."
2. Which tense follows "What if we..."?
A The present: "What if we measure".
B The past simple: "What if we measured".
C The infinitive: "What if we to measure".
3. How is the outcome reported -- "a similar pilot ___ approved"?
A "was approved" -- passive: was + past participle.
B "approved" -- with no "be".
C "is approving".

Stage 1.4: Grammar Tip -- Proposing Frames & the Passive

GRAMMAR

Para propor uma ideia, use frames claros: "I would like to suggest THAT we...", "I would recommend + -ing", "Why don't we...?", "What if we + passado?", "I propose THAT we...". Para relatar o resultado, use a voz passiva (be / was / have been + partícipio): "the proposal was approved", "the budget has been allocated". / To propose an idea, use clear frames; to report the outcome, use the passive (be / was / have been + past participle).

FunctionFormExample
Suggest / propose+ that + clause"I propose that we run a pilot."
Recommend+ -ing (or that)"I recommend starting smaller."
Float an ideaWhat if we + past simple"What if we tested it?"
Report the outcomebe / was / have been + participle"The proposal was approved."

Key point / Ponto-chave: proponha na voz ativa com um frame ("I propose that we..."), mas relate o resultado na passiva ("the proposal WAS approved", "the budget HAS BEEN allocated") -- a passiva SEMPRE precisa do BE. / Propose in the active voice with a frame, but report the result in the passive -- the passive always needs "be" (was / were / been).

Stage 2: Order the Pitch

Order

Listen first, then put the steps of a strong pitch in the correct order.

?You lower the risk: instead of a full rollout, you suggest a pilot at one site, with a short timeline and clear deliverables.
?You open with the rationale: you explain why the current process is a problem and why it matters now.
?You ask for the decision: you secure buy-in, confirm the next step, and agree on when the results will be reviewed.
?You state what you are proposing, in one clear line, and you keep the scope tight.
?You make the case on value: you give the return on the investment and a simple value proposition.
