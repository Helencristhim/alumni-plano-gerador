Chapter 3 -- Proposing Frames

Language Discovery

Frames that propose an idea -- and the passive for the outcome

Discovery

Listen and Notice

How does each sentence introduce the idea?

"I would like to suggest that we run a pilot first."

"I would recommend starting the rollout at one site."

"I propose that we measure the results after three months."

"What if we tested it as a pilot first?"

The Rule

Frames for Proposing

FrameRegisterExample
Why don't we...?Inviting, opens the talk"Why don't we start with one site?"
What if we + past?Tentative, floats an idea"What if we tested it as a pilot?"
I would suggest / recommend...Polite, professional"I would recommend starting smaller."
I propose that we...Formal, for the board"I propose that we run a pilot."

Key idea: after suggest and propose, use that + a clause ("I propose that we run a pilot"). After recommend, use -ing ("I recommend starting smaller") or that. What if we takes the past simple ("What if we tested it?"). Move up the ladder as the room gets more formal.

Language Focus

Reporting the Outcome

When you report what happened to a proposal, use be / was / have been + past participle. The doer is often the board, so you can leave it out.

"The proposal was approved at last week's meeting."

"The budget for the pilot has been allocated."

"The pilot was launched at the Houston site in March."

Pattern: subject + was / were or has / have been + past participle (approved, allocated, launched). Add "by + doer" only if it matters ("approved by the board"). You propose in the active voice; you report the result in the passive.

Common Mistake

Watch Out

"The proposal approved by the board."
"The proposal was approved by the board."

The passive always needs be (was / were / been). Say "the budget has been allocated", never "the budget has allocated". You propose in the active voice; you report the result in the passive.
