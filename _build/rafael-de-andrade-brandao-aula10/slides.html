Chapter 3 -- Should / Could / Would

Grammar Discovery

Should, could, would -- make a clear suggestion

Grammar Discovery

Listen and Notice

What pattern do the orange words follow?

"You should choose the pro plan."

"We could improve the old system first."

"I would recommend the new solution."

"You could consider an upgrade later."

Grammar Rule

Should / Could / Would

ModalUseExample
should + verbStrong suggestion / adviceYou should choose the pro plan
could + verbA soft option / possibilityWe could improve it first
would + recommendA polite recommendationI would recommend the new one
Key pointUse the base verb after the modal. Never add to and never add -s: say should choose, not "should to choose".

Key idea: A modal makes a suggestion polite and clear. Should is strong advice, could is a soft option, and would recommend is a polite tip. Always use the base verb after them.

Common Mistake

Watch Out

"You should to upgrade the system."
"You should upgrade the system."

After should, could, and would, use the base verb only. Never add to.
