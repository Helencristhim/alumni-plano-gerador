Chapter 3 -- The Ongoing Code

Still Going On with Have Been + -ing

How we say what started before and continues right now

Listen and Notice

What Shows an Ongoing Action?

Each sentence started in the past and is still happening. Notice the orange words.

"I have been spending more time on screens."

"My phone has been asking for an update."

"I have not been sleeping well lately."

"How long have you been using that app?"

The Rule

Have / Has + been + verb-ing

FormUseExample
have been + verb-ingI / you / we / they"I have been spending too much time online."
has been + verb-inghe / she / it"My phone has been asking for an update."
have not been + verb-ingnegative"I have not been sleeping well."
How long have you been ...?question"How long have you been using it?"

Tip: we use this tense for an action that started in the past and is still going on. Common time words: for, since, lately, recently, all day.

Common Mistake

Do Not Forget Been or -ing

"I have spending too much time on my phone."
"I have been spending too much time on my phone."

Always keep all three parts: have / has + been + verb-ing. Never drop been, and never use the base verb after it.

Practice

"How long _____have you been using that app?" (you have been / have you been)
