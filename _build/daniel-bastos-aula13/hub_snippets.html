Etapa 1.1: Tarjetas de Vocabulario

Vocabulario

Escucha cada palabra y lee el ejemplo. Toca Escuchar para oírla.

Tendencia -- la dirección general de los números (PT: tendência)
"La tendencia de las ventas es positiva."
Aumentar -- subir, hacerse más grande (PT: aumentar)
"La producción aumentó este año."
Disminuir -- bajar, hacerse más pequeño (PT: diminuir)
"Los costes disminuyeron un poco."
Cifra -- un número exacto, un dato (PT: cifra, número)
"La cifra subió a ciento treinta toneladas."
Eje -- cada línea del gráfico, vertical y horizontal (PT: eixo)
"El eje vertical muestra las toneladas."
Columna -- cada barra vertical del gráfico (PT: coluna, barra)
"La columna de marzo es la más alta."
Comparar -- ver dos cifras juntas (PT: comparar)
"Quiero comparar este mes con el anterior."
Promedio -- el valor del medio, la media (PT: média)
"El promedio mensual es más alto este año."

Etapa 1.3: Gramática en Contexto

Gramática

Lee la descripción del gráfico y responde las preguntas.

Daniel describe el gráfico: "La producción es más alta que el año pasado. La cifra subió de cien a ciento treinta toneladas. Marzo es el mes más alto del año. Es el mayor aumento de los últimos cinco años. En cambio, los costes son menos altos que antes."

1. "Más alta que" sirve para...
A Comparar dos cifras.
B Cerrar la presentación.
C Saludar al cliente.
2. "El mes más alto" sirve para...
A Comparar solo dos meses.
B Decir cuál es el primero de todos.
C Pedir un descuento.
3. "Los costes son menos altos que antes" significa que los costes...
A Aumentaron.
B Disminuyeron.
C Se quedaron igual.

Etapa 1.4: Consejo de Gramática -- Comparar y Superlativo

Gramática

Para comparar cifras usas más/menos... que. Para decir cuál es la primera de todas usas el/la más... (PT: para comparar use mais/menos... que; para o superlativo use o/a mais...).

EstructuraUso (PT)Ejemplo
más + adj. + quecomparar (mais... que)Es más alta que el año pasado.
menos + adj. + quecomparar (menos... que)El coste es menos alto que antes.
el / la más + adj.superlativo (o/a mais...)Marzo es el mes más alto.
el mayor / el menorsuperlativo irregular (o maior / menor)Es el mayor aumento de cinco años.
CuidadoEn español NO se dice "do que": solo que. (PT: em espanhol não existe "do que", só "que".)
