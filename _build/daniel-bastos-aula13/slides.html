Capítulo 3: El Código

Comparar Cifras

más que · menos que · el más alto

Descubre la Gramática

Escucha y Observa

"La producción es más alta que el año pasado."

"El coste es menos alto que antes."

"Marzo es el mes más alto del año."

"Es el mayor aumento de los últimos cinco años."

EstructuraSirve para
más + adjetivo + queComparar: A es mayor que B
menos + adjetivo + queComparar: A es menor que B
el / la más + adjetivoSuperlativo: el primero de todos
el mayor / el menorSuperlativo irregular (el más grande / pequeño)

Para comparar DOS cifras: más/menos... que. Para decir cuál es LA PRIMERA de todas: el/la más... o el mayor.

Error Común

Correcto vs Incorrecto

✗ INCORRECTO

"Es más alto do que el año pasado."

✓ CORRECTO

"Es más alto que el año pasado."

✗ INCORRECTO

"Marzo es más alto de todos."

✓ CORRECTO

"Marzo es el más alto del año."
