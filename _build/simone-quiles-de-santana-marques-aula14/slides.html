Chapter 3 -- First Conditional

If + Will

How to say what you will do if something happens

Listen and Notice

What Do They Share?

Look at the highlighted words. Where is "will", and where is the present?

"If the parties cannot agree, we will try mediation."

"If mediation fails, we will go to arbitration."

"We will save money if we settle the claim early."

"If the arbitrator decides, the award will be binding."

The First Conditional Code

First Conditional

PartFormExample
condition (if)if + present"if mediation fails"
resultwill + verb"we will go to arbitration"
orderboth ways"we will go to arbitration if mediation fails"
usereal futurea possible, real result -- not imaginary

Key idea: the first conditional talks about a real future result. Use the present after if, and will in the result -- never "if ... will".

Common Mistake

Watch Out

"If mediation will fail, we go to arbitration. ... If the parties will not agree, we settle the claim."
"If mediation fails, we will go to arbitration. ... If the parties do not agree, we will settle the claim."

After if, use the present. Put will in the result, not in the if part.

Artefact

The Dispute Clause

SERVICE AGREEMENT -- DISPUTES

DRAFT

Clause 22 -- Mediation

If a dispute arises, the parties will first try mediation in good faith. If they settle the claim in mediation, the dispute ends.

Clause 23 -- Arbitration

If mediation fails within sixty days, the parties will go to arbitration in London. The arbitrator will hear both sides and give an award.

Clause 24 -- Binding Award

The award will be final and binding on both parties. Neither party will appeal the award, except for a clear error of law.

The parties want to resolve any dispute quickly and keep a good business relationship.

Dialogue

The Dispute Call

S

"Thank you for the call. We know there is a dispute about the last payment. If we work together, we will resolve it quickly."

P

"I agree. We have a claim for the delay, but we prefer to talk. If you accept mediation, we will not start a court case."

S

"We accept mediation. If we settle the claim there, we will both save time. If mediation fails, we will go to arbitration, as the clause says."

P

"Good. And remember, if the arbitrator decides, the award will be binding. So mediation is better for both of us."

S

"Exactly. If we resolve the dispute in mediation, we keep a good relationship. Let us choose a mediator this week."

P

"Perfect. If you send me three names today, I will confirm one by Friday. I think we will settle this without arbitration."
