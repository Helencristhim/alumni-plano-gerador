Resolving Disputes -- Arbitration & the First Conditional

Learn the language of disputes and how to resolve them: mediation, arbitration, and a binding award. New vocabulary: dispute, mediation, arbitration, claim, settle, award, binding, resolve. Grammar: the first conditional -- if + present, will + verb ("If mediation fails, we will go to arbitration").

Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Dispute -- a disagreement between the parties about the contract
"There is a dispute about the last payment." (Disputa / litígio)
Mediation -- a neutral person helps the parties agree, without deciding
"They will try mediation first." (Mediação)
Arbitration -- a private judge decides the dispute outside court
"If mediation fails, we go to arbitration." (Arbitragem)
Claim -- a formal demand that one party makes against the other
"The partner has a claim for late delivery." (Reivindicação / pedido)
Settle -- to end a dispute by agreement, often before a decision
"If we settle the claim early, we save money." (Resolver por acordo)
Award -- the decision the arbitrator gives at the end of arbitration
"The award is final for both parties." (Sentença arbitral / laudo)
Binding -- a decision that both parties must obey
"The arbitration award is binding." (Vinculante / obrigatório)
Resolve -- to solve a dispute and bring it to an end
"We want to resolve the dispute quickly." (Resolver / solucionar)

Stage 1.4: Grammar Tip -- The First Conditional

GRAMMAR

Para falar de um resultado REAL no futuro, use o first conditional. Na condição, use if + presente ("if mediation fails"). No resultado, use will + verbo ("we will go to arbitration"). As duas partes podem trocar de ordem; quando o if vem primeiro, use vírgula. NÃO use will depois de if. / To talk about a real future result, use the first conditional. In the condition use if + present ("if mediation fails"). In the result use will + verb ("we will go to arbitration"). The two parts can swap order; use a comma when if comes first. Do not use will after if.

Part / ParteForm / FormaExample / Exemplo
condition / condiçãoif + present"if mediation fails"
result / resultadowill + verb"we will go to arbitration"
full / completaif + present, will + verb"If mediation fails, we will go to arbitration."
