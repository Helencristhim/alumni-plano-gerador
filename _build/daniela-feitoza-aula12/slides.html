Chapter 3

The Word-Builder

How tech terms are built from two nouns

Word Discovery

Listen and Notice

inventory + system = "inventory system"

payment + gateway = "payment gateway"

user + interface = "user interface"

sales + data = "sales data"

In each term, which noun is the main thing? Which one only describes it?

Word Rule

Compound Nouns

PatternHow it worksExample
noun + nounThe first noun describes the second; the second is the main thing"payment gateway" = a gateway for payments
first noun stays singularThe describing noun is almost always singular"inventory system" (NOT "inventories system")
Key questionAsk: what kind of [second noun]? The first noun answers it.
Common Mistake

Wrong Word vs Right Word

✗ WRONG

"the inventories system"

✓ RIGHT

"the inventory system" (first noun stays singular)

✗ WRONG

"the gateway payment"

✓ RIGHT

"the payment gateway" (describer first, main thing second)

Describer first, singular; main thing second. The opposite of Portuguese order.

Chapter 4

The Pitch Room

Daniela presents to Marcus, a stakeholder

Dialogue

The Stakeholder Meeting

M
Hi Daniela, thanks for the demo. Can you walk me through the new retail platform?
D
Of course. The customer opens the mobile app and browses the catalog. It connects to our inventory system in real time.
M
Nice. And how does the customer pay?
D
At the checkout, the payment gateway processes the card in two seconds. The user interface is very simple.
M
What about the store managers? Can they see the numbers?
D
Yes. The dashboard shows live analytics, and it also works with the point-of-sale terminals in the stores.
M
Impressive. Will it grow with us? And how long is the launch?
D
It is fully scalable, so it grows with the business. The full rollout takes about six weeks.
Comprehension

Did You Catch It?

1. What does the app connect to, and how often is it updated?
It connects to the inventory system in real time, so the stock count is always correct.
2. How do the store managers see the numbers?
On the dashboard, which shows live analytics, and which also works with the point-of-sale terminals.
3. According to Daniela, is the platform scalable, and how long is the rollout?
Yes, it is fully scalable, so it grows with the business, and the full rollout takes about six weeks.
