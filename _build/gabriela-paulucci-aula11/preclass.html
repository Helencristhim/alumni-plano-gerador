Stage 1.4: Grammar Tip -- There Is / There Are

GRAMMAR

Usamos there is para uma coisa e there are para várias. / We use there is for one thing and there are for many.

Form / FormaUse / UsoExample / Exemplo
There is + uma coisaUma coisa só (singular) / One thingThere is a sofa.
There are + várias coisasDuas ou mais (plural) / Two or moreThere are two windows.
There is no / There are noPara dizer que não existe / To say it does not existThere is no table.
Is there...? / Are there...?Para perguntar / To askIs there a window?

Key point / Ponto-chave: conte primeiro. Uma coisa → there is. Duas ou mais → there are. / Count first. One thing → there is. Two or more → there are.
