Chapter 3 -- The Code

Grammar Discovery

there is -- one thing -- there are -- many things

Grammar Discovery

Listen and Notice

When do we use there is and there are?

"There is a sofa in the living room."

"There is a bed in the bedroom."

"There are two windows in the room."

"There are three chairs in the kitchen."

Grammar Rule

There Is / There Are

FormWhen we use itExample
There is + one thingOne single thing (singular)There is a sofa.
There are + many thingsTwo or more things (plural)There are two windows.
There is no / There are noTo say something does not existThere is no table.
Is there...? / Are there...?To ask a questionIs there a window?

Key idea: use there is for one thing and there are for two or more things. Count first, then choose.

Common Mistake

Watch Out

"There is two windows in the living room."
"There are two windows in the living room."

For two or more things, use there are. For one thing, use there is.
