Stage 1.4: Grammar Tip -- Present Continuous (Now vs Routine)

GRAMMAR

Para falar do que está acontecendo AGORA, use am / is / are + verbo-ing. Para falar de uma ROTINA (toda hora, todo dia), use o present simple. I + am, he/she/it + is, we/you/they + are.

FormaUso (PT)Example
I am + -ingeu, agoraI am working right now
he/she/it is + -ingele/ela, agoraShe is typing the code
we/you/they are + -ingnós/vocês/eles, agoraWe are discussing it
negativaam/is/are + not + -ingI am not running it now
perguntaAm/Is/Are + sujeito + -ing?Are you working now?
routine (simple)rotina: every day, on MondaysI run the pipeline every day

Atenção (erro comum): nunca esqueça o verbo be ("I working" está errado; o certo é "I am working"). Use is com he/she/it. Para uma rotina (every day, every Monday), use o present simple, não o -ing. Ortografia do -ing: run → running (dobra o n), type / update → typing / updating (tira o e).
