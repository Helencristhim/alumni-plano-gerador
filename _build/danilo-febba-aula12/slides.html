Chapter 3

The -ing Code

am / is / are + verb-ing for right now

Grammar Discovery

Listen and Notice

I am working right now.

The script is running at the moment.

We are discussing the plan now.

But I run the pipeline every morning.

Subjectbeverb-ing
Iamworking, running
he / she / itisloading, typing
we / you / theyarediscussing, updating
NowcontinuousI am running it now
RoutinesimpleI run it every day

Now = am/is/are + verb-ing (right now, at the moment). Routine = present simple (every day, on Mondays, usually). Spelling: run → running, type → typing (drop the e).

Common Mistake

Right vs Wrong

✗ WRONG

"I working right now."

✓ RIGHT

"I am working right now."

✗ WRONG

"She typing the code."

✓ RIGHT

"She is typing the code."

✗ WRONG

"I am running the pipeline every Monday."

✓ RIGHT

"I run the pipeline every Monday."

Always use be + verb-ing for now. He/she/it takes is. For a routine (every day, every Monday), use the present simple, not -ing.

Build the Sentence

Now or Routine?

Say the sentence, then click to compare

1. now: the script + run
"The script is running right now."
2. routine: every morning + I + run it
"I run the pipeline every morning."
3. now: my team + discuss + the bug
"My team is discussing the bug now."
4. now: I + focus + the report
"I am focusing on the report."
5. routine: I + check + emails + every day
"I check my emails every day."
