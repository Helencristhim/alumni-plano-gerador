Capítulo 3: El Código

Cómo Iban las Cosas

era · había · teníamos · vendíamos

Descubre la Gramática

Escucha y Observa

"Antes el proceso era manual."

"El año pasado teníamos menos clientes."

"Había muchos errores en los datos."

"Cada trimestre vendíamos más."

EstructuraUsoEjemplo
Imperfecto -ar: -abaDescribir el pasadotrabajaba, presentaba
Imperfecto -er/-ir: -íaHábitos y tendenciastenía, vendía, crecía
IrregularesEstado o fondoera (ser), había (haber)
Imperfecto vs IndefinidoImperfecto = cómo era / fondo. Indefinido = una acción puntual: "presenté".

Usa el imperfecto para describir cómo eran las cosas: "antes vendíamos poco". Usa el indefinido para una acción terminada: "ayer presenté el informe".

Error Común

Correcto vs Incorrecto

✗ INCORRECTO

"Antes el proceso fue manual."

✓ CORRECTO

"Antes el proceso era manual."

✗ INCORRECTO

"Ayer presentaba el informe."

✓ CORRECTO

"Ayer presenté el informe."

El imperfecto describe cómo eran las cosas (fondo). El indefinido cuenta una acción terminada en un momento concreto (ayer, el lunes).

Autoevaluación

Lo Que Aprendí

Puedo presentar resultados y cifras de un trimestre.
Uso el pretérito imperfecto: era, había, teníamos, vendíamos.
Diferencio el imperfecto (fondo) del indefinido (acción puntual).
Hablo de tendencias, porcentajes y promedios.
Conozco las palabras: gráfico, tendencia, porcentaje, dato, promedio, indicador, trimestre, destacar.
