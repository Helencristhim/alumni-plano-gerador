Chapter 3: Can and Can't

The Code

can + verb. can't + verb.

Discover the Code

What Comes Before?

I can hear you.

I can't see your screen.

Can you hear me?

I can share my screen.

FormExample
can + verb (possible)I can hear you.
can't + verb (not possible)I can't see your screen.
Can + you + verb? (question)Can you hear me?

Use can + verb when it is possible. Use can't + verb when it is not possible. The verb never changes (no -s, no to).

Common Mistakes

Wrong vs Right

✗ WRONG

"I can to hear you."

✓ RIGHT

"I can hear you."

✗ WRONG

"I no can see you."

✓ RIGHT

"I can't see you."

✗ WRONG

"He cans share."

✓ RIGHT

"He can share."

Remember: can + verb, no "to". Negative is can't. Never add -s (no "cans").
