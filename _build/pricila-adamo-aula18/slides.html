Chapter 3: Deducing the Past

How Sure Are You About the Past?

must have · might have · can't have

Discover the Code

Listen and Find the Pattern

"She must have misunderstood me." (I'm almost sure)

"He might have been confused by the custom." (it's possible)

"They can't have meant it rudely." (I'm almost sure NOT)

"The card must have been offered with one hand." (passive)

Certainty about the pastStructureExample
Almost certain it WAS truemust have + past participleShe must have misunderstood me.
It was possiblemight / could have + past participleHe might have been confused.
Almost certain it was NOT truecan't have + past participleThey can't have meant it rudely.
Passive deductionmust / might / can't have BEEN + past participleThe card must have been offered with one hand.

Past modals of deduction = how sure you are about the past. The form never changes for he/she/it (never "must has"). Note: this is a REAL past you're guessing about -- different from the third conditional (Lesson 17), which imagines a past that never happened.

Common Mistake

Wrong vs Right

✗ WRONG

"They mustn't have meant it."

✓ RIGHT

"They can't have meant it."

✗ WRONG

"She must of misunderstood me."

✓ RIGHT

"She must have misunderstood me."

For "almost sure it did NOT happen", use can't have, never "mustn't have". And it is "must have", never "must of" (they just sound the same).

Expressions & Meanings

How Sure Are You?

The Certainty Scale

MUST HAVE -- almost certain it WAS true

"He went quiet -- I must have broken a custom." (~95% sure)

MIGHT / COULD HAVE -- it was possible

"He might have been surprised, or tired." (maybe, maybe not)

CAN'T HAVE -- almost certain it was NOT true

"He can't have been angry -- he smiled after." (~95% sure NOT)

Same moment, three levels of certainty -- all with modal + have + past participle.
