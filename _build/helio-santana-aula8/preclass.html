Etapa 1.1: Tarjetas de Vocabulario

Vocabulario

Escucha cada palabra y lee el ejemplo. La traducción al portugués está entre paréntesis. Toca Escuchar para oírla.

Proceso -- conjunto de pasos para hacer algo (PT: processo)
"El proceso tiene cuatro pasos."
Paso -- cada parte de un proceso (PT: passo / etapa)
"El primer paso es analizar."
Analizar -- estudiar algo con atención (PT: analisar)
"Primero se analiza la operación."
Detectar -- encontrar algo, como un riesgo (PT: detectar)
"Luego se detecta el riesgo."
Riesgo -- la posibilidad de un problema o peligro (PT: risco)
"El sistema detecta el riesgo."
Revisar -- mirar algo otra vez para comprobar (PT: revisar)
"Después se revisa el caso."
Aprobar -- dar el sí, aceptar algo (PT: aprovar)
"Finalmente se aprueba el resultado."
Resultado -- lo que sale al final del proceso (PT: resultado)
"El resultado da mucha confianza."

Etapa 1.3: Gramática en Contexto

Gramática

Lee el texto y responde las preguntas.

Helio le explica el proceso antifraude a Carolina. "Tiene cuatro pasos", dice. "Primero se analiza cada operación. Luego se detecta el riesgo de forma automática. Después se revisa el caso con un especialista y finalmente se aprueba el resultado." Carolina queda muy tranquila: "Es un proceso claro y seguro. Me gusta que se revisan todos los casos con cuidado."

1. "Primero se analiza la operación." ¿Para qué sirve el "se"?
A Para describir el proceso sin decir quién lo hace.
B Para hablar del futuro.
C Para dar una orden a alguien.
2. "Se revisan todos los casos." ¿Por qué el verbo va en plural?
A Porque siempre es plural.
B Porque concuerda con "los casos" (plural).
C Porque lleva "se".
3. ¿Qué palabras marcan el orden de los pasos?
A claro, seguro, automático.
B primero, luego, después, finalmente.
C proceso, paso, riesgo.

Etapa 1.4: Consejo de Gramática -- El Se Impersonal y los Secuenciadores

Gramática

Cómo describir un proceso paso a paso (ES + PT).

EstructuraUsoEjemplo
se + verboDescribir un proceso sin decir quiénSe analiza la operación.
un objeto → verbo singularConcordancia en singularse revisa el caso
varios objetos → verbo pluralConcordancia en pluralse revisan los casos
primero / luego / después / finalmenteOrdenar los pasos del flujoPrimero se analiza, finalmente se aprueba
Em português"se analiza / se detecta" = "analisa-se / detecta-se" (descrever um processo sem dizer quem faz). O verbo concorda com o objeto: se revisa el caso (singular), se revisan los casos (plural). Os secuenciadores primero, luego, después, finalmente colocam os passos em ordem.
