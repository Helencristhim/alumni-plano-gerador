Capítulo 3: El Código

Describir un Flujo

se + verbo · primero · luego · después · finalmente

Descubre la Gramática

Escucha y Observa

"Primero se analiza la operación."

"Luego se detecta el riesgo."

"Después se revisa el caso."

"Finalmente se aprueba el resultado."

EstructuraUsoEjemplo
se + verboDescribir un proceso sin decir quién lo hacese analiza, se detecta
un objeto → verbo singularConcordancia en singularse revisa el caso
varios objetos → verbo pluralConcordancia en pluralse revisan los casos
primero / luego / después / finalmenteOrdenar los pasos del flujoPrimero se analiza, finalmente se aprueba

Usamos se + el verbo para describir un proceso sin decir quién. En portugués es como "primeiro analisa-se / depois revisa-se". El verbo concuerda con el objeto: se revisa el caso, se revisan los casos.

Error Común

Correcto vs Incorrecto

✗ INCORRECTO

"Primero analiza la operación."

✓ CORRECTO

"Primero se analiza la operación."

✗ INCORRECTO

"Se aprueba los resultados."

✓ CORRECTO

"Se aprueban los resultados."

No olvides el se (sin él parece una orden). Y el verbo concuerda: un objeto → singular (se revisa), varios objetos → plural (se aprueban).

Práctica

Completa el Verbo

Dilo primero, luego haz clic para comprobar

"Después se ___revisa (revisar) el caso."

Lo Que Aprendí

Puedo describir el proceso antifraude paso a paso.
Uso el se impersonal: se analiza, se detecta, se revisa.
Uso los secuenciadores: primero, luego, después, finalmente.
Hago la concordancia: se revisa el caso / se revisan los casos.
Conozco las palabras: proceso, paso, analizar, detectar, riesgo, revisar, aprobar, resultado.
