Chapter 3 -- The Code

Grammar Discovery

should, could and might: suggesting and recommending

Grammar Discovery

Listen and Notice

What comes after the orange words?

"We should start with the budget." (strong advice)

"We could brainstorm ideas for the launch." (a possible option)

"The client might join us later." (not sure)

"Every attendee should read the agenda." (strong advice)

Grammar Rule

should, could and might

ModalUseExample
shouldadvice / recommendationWe should start on time.
couldsuggestion / optionWe could postpone the launch.
mightpossibility (less sure)The client might join late.
Negativeadd notWe should not be late.

Key idea: modal + base verb. Never use to and never add -s after should, could or might. Use should for strong advice, could for a softer option, and might for something that is not certain.

Common Mistake

Watch Out

"We should to start with the budget."
"We should start with the budget."

After should, could and might, use the base verb with no to and no -s.

Grammar Practice

Choose the Modal
