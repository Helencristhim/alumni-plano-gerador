Grammar Rule

be going to + verb

FormStructureExample
Affirmativesubject + am/is/are + going to + verbI am going to play.
he / she / itis going toShe is going to bake.
we / you / theyare going toWe are going to ride bikes.
Negativeam/is/are + not + going to + verbHe is not going to come.
Questionam/is/are + subject + going to + verbAre you going to study?

Key idea: use be going to to talk about plans for the future. After "going to", always use a verb in the base form (play, have, bake, ride). In speaking we say I'm going to, she's going to, they're going to.

Common Mistake

Watch Out

"I going to play" / "She is going to plays."
"I am going to play." / "She is going to play."

Always use the verb be (am / is / are) before "going to", and use the base verb after it: play, have, bake. Never add -s or -ing to the verb after "going to".
