Polite Requests

Ask or Offer?

Look at the orange words. When do we ask for something, and when do we offer something?

I would like a coffee, please.

I would like some water.

Would you like a dessert?

Would you like a drink?

Grammar Rule

I Would Like vs Would You Like

UseStructureExample
Ask for foodI would like + a / someI would like a coffee.
Offer foodWould you like + a / someWould you like a dessert?
Say yesYes, please.Yes, please.
Say noNo, thank you.No, thank you.

Key idea: use I would like to ask for something in a polite way, and Would you like to offer something. It is more polite than "I want".

Common Mistake

Watch Out

"Give me a coffee."
"I would like a coffee, please."

Use I would like and please to be polite, not "Give me".

Grammar Practice

Ask or Offer?

You ask for a coffee.  =  I would like a coffee.
You offer a dessert.  =  Would you like a dessert?
You ask for some water.  =  I would like some water.
You offer a drink.  =  Would you like a drink?
