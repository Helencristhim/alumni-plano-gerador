Vocabulary

Eight words you hear in almost every negotiation. Listen to each one and notice how it is used.

"Please send me a quote for the project."
Bargain -- pechinchar / negociar o preço
"Let us bargain a little on the price."
Installment -- parcela (um de vários pagamentos)
"We will pay in three installments."
Deposit -- sinal / entrada (pagamento inicial para garantir)
"They asked for a ten percent deposit."

The seller is flexible: if the buyer pays a deposit today, the seller will accept three installments, but the discount will be smaller. If both sides agree, they will close the deal and shake hands.

B It will be smaller.

Stage 1.4: Grammar Tip -- The First Conditional

Grammar

Em negociações, usamos o first conditional para fazer ofertas sobre o futuro: uma condição possível e o seu resultado. Use if + presente simples na condição (if you sign, if you pay) e will + verbo na base no resultado (we will give, we will offer). Nunca use "will" depois de "if" (errado: "if you will sign"). As duas metades podem trocar de ordem: "If you sign, we will give..." = "We will give... if you sign". / In negotiations, use the first conditional: if + present simple for the condition, will + base verb for the result. Never use "will" after "if".

Condição (if + presente)If you sign this month,Se você assinar este mês,
Resultado (will + base)we will give you a discount.nós daremos um desconto.
Ordem trocadaWe will give a discount if you sign.Daremos um desconto se você assinar.
ErradoIf you will sign, we give...(nunca "will" depois de "if")
Regra de ouroif + present, ... will + base verbif + presente, ... will + verbo base
