Chapter 3 -- The Code

Grammar Discovery

would like with a, an and some

Grammar Discovery

Listen and Notice

What do the orange words have in common?

"I would like a juice, please."

"I would like an apple, please."

"I would like some water, please."

"Would you like any dessert?"

Grammar Rule

would like + a / some

FormExample
Polite requestI would like a salad.
Countable (one)I would like a juice / an apple.
Uncountable / pluralI would like some water.
QuestionWould you like any dessert?

Key idea: use would like to order politely. Use a / an with one countable thing, and some with uncountable words or plurals. In questions, use any.

Common Mistake

Watch Out

"I would like a water, please."
"I would like some water, please."

Water is uncountable. Use some, not a.

Grammar Practice

a, an or some?

"I would like _____a orange juice." (one countable)
"I would like _____some water." (uncountable)
"I would like _____an apple." (countable, starts with a vowel)
"Would you like _____any dessert?" (question)
