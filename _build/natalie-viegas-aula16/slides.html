Chapter 3 -- The Duration Code

Grammar Discovery

Present perfect continuous -- ongoing work and duration

Grammar Discovery

Listen and Notice

What are the orange words? When did the action start, and is it still going on?

"We have been working on phase two for three weeks."

"The team has been testing the system since Monday."

"I have been monitoring one blocker closely."

"We have completed the audit." (result, finished)

Grammar Rule

Present Perfect Continuous

FormUseExample
have/has + been + -ingongoing action, started in the past, still happening"We have been working on it."
+ for + periodhow long (duration)"...for three weeks."
+ since + start pointwhen it started"...since Monday."
present perfect simplefinished result, not the process"We have completed the audit."

Key idea: use the present perfect continuous (have/has been + -ing) for work that started in the past and is still going on, especially to stress how long. Use for + a period (for three weeks) and since + a start point (since Monday). Use the present perfect simple (have completed) for a finished result.

Common Mistake

Watch Out

"We work on phase two since three weeks."
"We have been working on phase two for three weeks."

Use have/has been + -ing for ongoing work, not the present simple. Use for with a period (for three weeks) and since with a start point (since Monday).
