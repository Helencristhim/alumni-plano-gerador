Project Updates -- Reporting Progress

Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Metrics -- numbers used to measure and track progress or results
"The metrics show we are ahead of schedule." (Métricas)
To date -- from the beginning of the project until now
"To date, we have completed two phases." (Até agora)
Ahead of schedule -- earlier or faster than the planned time
"Good news: we are ahead of schedule this month." (Adiantado)
Behind schedule -- later or slower than the planned time
"One workstream is slightly behind schedule." (Atrasado)
Momentum -- the energy and speed that keep progress moving forward
"The team is building real momentum." (Impulso)
Forecast -- a prediction about the future based on current data
"Our forecast shows we will finish in May." (Previsão)
Dependency -- a task that must be finished before another can start
"The vendor sign-off is a dependency for the next phase." (Dependência)
Blocker -- a problem that stops the work from moving forward
"There is one blocker: we are waiting for approval." (Impedimento)
Escalate -- to raise an issue to a more senior person for help
"I will escalate the delay to the project sponsor." (Escalar)
Mitigate -- to reduce the size or impact of a risk or problem
"We have a backup plan to mitigate the risk." (Mitigar)

Stage 1.4: Grammar Tip -- Present Perfect Continuous

GRAMMAR

Usamos o present perfect continuous (have/has been + verbo-ing) para ações que começaram no passado e continuam agora, enfatizando a DURAÇÃO. Use for + período e since + ponto de início. / We use the present perfect continuous for actions that started in the past and are still happening, stressing duration. Use for + a period and since + a start point.

FormUseExample
have/has + been + -ingongoing, still happening"We have been working on it."
+ for + periodhow long (duration)"...for three weeks."
+ since + start pointwhen it started"...since Monday."
present perfect simplefinished result"We have completed the audit."

Key point / Ponto-chave: use have/has been + -ing para o PROCESSO em andamento (have been testing) e o present perfect simples para o RESULTADO concluído (have completed). for = período (for three weeks); since = ponto de início (since Monday). Terceira pessoa: has been (she has been monitoring), nunca "have been".

Stage 2: Build a Progress Update in Order

Order

Listen first, then put the steps of a clear progress update in the correct order.

?You explain what the team has been doing and the momentum so far.
?You state the headline, so leadership knows the status in one sentence.
?You confirm the next steps and whether you are still ahead of schedule.
?You report the results you have completed, with clear metrics.
?You flag any blocker or dependency honestly, before it grows.
