GRAMMAR

Stage 1.4: Grammar Tip -- Want to / Need to

GRAMMAR

Em inglês, usamos want to (desejo) e need to (necessário) + verbo. / In English, we use want to (desire) and need to (necessary) + verb.

IdeiaStructure / EstruturaExample / Exemplo
desejowant to + verboI want to plan a lunch.
necessárioneed to + verboWe need to choose a date.
He / She (com S)wants to / needs to + verboShe wants to help.
negativodo not / does not want toI do not want to wait.

Key point / Ponto-chave: use want to para desejo e need to para algo necessário, sempre com o verbo base. Com he/she, acrescente um S: wants to, needs to. Nunca diga "I want choose" -- diga I want to choose.
