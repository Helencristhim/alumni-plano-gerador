Chapter 3 -- The Code

Grammar Discovery

Polite requests: I'd like... / Would you mind + -ing?

Grammar Discovery

Listen and Notice

How do we state what we want and ask for a favor politely?

"I would like a quiet room, please." (stating what you want)

"I'd like to check in early, please." (I would like + to + verb)

"Would you mind giving me a wake-up call?" (asking a favor)

"Would you mind sending housekeeping up?" (favor + -ing form)

Grammar Rule

I'd like... / Would you mind + -ing

FormExample
Want a thing: I would like + nounI would like a quiet room, please.
Want to do: I'd like + to + base verbI'd like to check in early, please.
Ask a favor: Would you mind + verb-ing?Would you mind giving me a wake-up call?
Ask a favor: Would you mind + verb-ing?Would you mind sending housekeeping up?

Key idea: I would like (short form I'd like) is a polite way to STATE what you want -- much softer than "I want". Use it with a noun or with to + base verb. To ask someone for a favor, use Would you mind + the -ing form (giving, sending, helping).

Common Mistake

Watch Out

"Would you mind to give me a wake-up call?" / "I would like check in early."
"Would you mind giving me a wake-up call?" / "I would like to check in early."

After Would you mind, use the -ing form. After I would like, use a noun or to + base verb.
