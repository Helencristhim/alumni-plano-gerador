Capítulo 3: El Código

Por y Para

gracias por su tiempo · es para usted

Descubre la Gramática

Escucha y Observa

"Gracias por su confianza."

"Esta propuesta es para usted."

"Lo hacemos por el cliente."

"Lo tengo listo para el viernes."

PreposiciónUsoEjemplo
PorCausa o motivoGracias por su tiempo
PorIntercambioun reembolso por el error
ParaDestino o finalidadesta propuesta es para usted
ParaPlazo (cuándo)lo tengo para el viernes

Por mira hacia atrás: la causa, el motivo, el intercambio. Para mira hacia adelante: el destino, la finalidad, el plazo. Memoriza: "gracias por" (causa) y "es para usted" (destino).

Error Común

Correcto vs Incorrecto

✗ INCORRECTO

"Gracias para su tiempo."

✓ CORRECTO

"Gracias por su tiempo."

✗ INCORRECTO

"Este informe es por usted."

✓ CORRECTO

"Este informe es para usted."

Usa por para la causa o el motivo (gracias por, un reembolso por el error) y para para el destino o el plazo (es para usted, listo para el viernes).

Capítulo 7: Cierre

Objeción Manejada

Autoevaluación

Lo Que Aprendí

Puedo atender una objeción de un cliente con calma.
Uso por para la causa y el motivo: gracias por su tiempo.
Uso para para el destino y el plazo: es para usted, para el viernes.
Sé ceder en parte, ofrecer un reembolso y mantener mi postura.
Conozco las palabras: objeción, reclamar, malentendido, insatisfecho, ceder, postura, reembolso, plantear.
