Manejar Quejas y Objeciones

Etapa 1.1: Tarjetas de Vocabulario

Vocabulario

Escucha cada palabra y lee el ejemplo. La traducción al portugués está entre paréntesis. Toca Escuchar para oírla.

Objeción -- una razón en contra de una propuesta (PT: objeção)
"La cliente plantea una objeción sobre el precio."
Reclamar -- pedir algo con derecho o quejarse formalmente (PT: reclamar)
"Quiero reclamar una solución para el viernes."
Malentendido -- cuando dos personas entienden cosas distintas (PT: mal-entendido)
"Hubo un malentendido con las fechas de entrega."
Insatisfecho -- no contento con un servicio o producto (PT: insatisfeito)
"El cliente está insatisfecho con el retraso."
Ceder -- aceptar parte de lo que pide el otro (PT: ceder)
"Puedo ceder en el plazo de entrega."
Postura -- la posición que mantienes en una negociación (PT: postura, posição)
"Mantengo mi postura sobre el alcance del proyecto."
Reembolso -- devolver el dinero a un cliente (PT: reembolso)
"Le ofrecemos un reembolso parcial por el error."
Plantear -- presentar o levantar un problema o una idea (PT: propor, levantar)
"Quiero plantear una objeción sobre el precio."

Etapa 1.3: Gramática en Contexto

Gramática

Lee el texto y responde las preguntas.

La cliente plantea una objeción y está insatisfecha por el retraso. Helio responde con cortesía: "Gracias por su confianza. Lo siento, hubo un malentendido. Puedo ceder en el plazo: lo tengo listo para el viernes. Por el error, le ofrecemos un reembolso parcial. Mantengo mi postura sobre el alcance, pero todo el trabajo lo hacemos para usted." Carolina acepta: "Gracias por la solución."

1. "Gracias por su confianza." ¿Por qué usamos "por" y no "para"?
A Porque indica un destino futuro.
B Porque indica la causa o el motivo del agradecimiento.
C Porque es un plazo de entrega.
2. "Lo tengo listo para el viernes." ¿Qué expresa "para" aquí?
A La causa de un error.
B El plazo: cuándo va a estar listo.
C Un intercambio de dinero.
3. "Por el error, le ofrecemos un reembolso." ¿Para qué sirve "por" aquí?
A Para indicar el destino del reembolso.
B Para indicar el motivo del reembolso (a cambio del error).
C Para indicar una opinión personal.

Etapa 1.4: Consejo de Gramática -- Por y Para

Gramática

Cuándo usar por y cuándo usar para con un cliente (ES + PT).

PreposiciónUsoEjemplo
PorCausa o motivoGracias por su tiempo.
PorIntercambio (a cambio de)Un reembolso por el error.
ParaDestino o finalidadEsta propuesta es para usted.
ParaPlazo (cuándo)Lo tengo para el viernes.
Em portuguêsUse por para a causa, o motivo ou a troca: "gracias por su tiempo", "un reembolso por el error". Use para para o destino, a finalidade ou o prazo: "es para usted", "listo para el viernes". Dica: por olha para trás (o porquê), para olha para frente (o objetivo).

Etapa 3: Pronunciación

Pronunciación

Escucha cada frase y luego grábate diciéndola. La nota en portugués indica para qué sirve.

PT: Para abrir com cortesia quando o cliente faz uma objeção (por = causa).
Creo que hubo un malentendido con las fechas.
PT: Para aclarar a confusão sem culpar o cliente.
Puedo ceder en el plazo: lo tengo para el viernes.
PT: Para ceder em parte e dar um prazo (para = plazo).
Por el error, le ofrecemos un reembolso parcial.
PT: Para oferecer uma compensação pelo erro (por = motivo).

Etapa 4: Cuestionario Situacional

Cuestionario

Elige la mejor respuesta para cada situación profesional.

El cliente plantea una objeción. Quieres agradecer su confianza. Dices:
A "Gracias para su confianza."
B "Gracias por su confianza, la escucho con atención."
C "La objeción no es importante."
Quieres decir cuándo estará lista la solución. Dices:
A "Lo tengo listo por el viernes."
B "Lo tengo listo para el viernes."
C "Lo tengo listo en el viernes pasado."
Quieres ofrecer una compensación por el error. La mejor opción es:
A "Le ofrecemos un reembolso para el error."
B "No hay reembolso, es su problema."
C "Por el error, le ofrecemos un reembolso parcial."
