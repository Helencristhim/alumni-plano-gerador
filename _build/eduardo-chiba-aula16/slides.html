Chapter 3 -- Softening the Tone

From Order to Polite Request

The same ask, made courteous

Discover the Pattern

What Makes It Polite?

"Could you send me the contract?"

"I was wondering if you could confirm the date."

"I would appreciate it if you could reply by Friday."

"Would you mind sending the attachment?"

The Rule

Softening a Request

RegisterSoftenerExample
Direct / informalimperative"Send me the contract."
PoliteCould you...?"Could you send me the contract?"
FormalI was wondering if you could..."I was wondering if you could confirm the date."
Very formalI would appreciate it if you could..."I would appreciate it if you could reply by Friday."

Key idea: the longer and more indirect the softener, the more formal the request. Match the register to the recipient: a close colleague is fine with "Could you...?", but a senior partner you have never met deserves "I was wondering if you could..." or "I would appreciate it if you could...".

Watch Out

Common Mistake

Send me the signed contract today.
Could you send me the signed contract by the end of the day?

A bare imperative is fine for a close colleague, but to a senior recipient it reads as an order. Soften it with "Could you...?" and a courteous deadline, and the same request sounds professional.

Soften the Line
