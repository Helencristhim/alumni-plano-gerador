GRAMMAR

Stage 1.4: Grammar Tip -- Softening Requests & Refusals

GRAMMAR

Para um pedido educado, troque a ordem direta por uma pergunta indireta: "I was wondering if you could..." / "Would it be possible to...?" / "Could you possibly...?". Para um NÃO educado, comece com "I am afraid..." ou "Unfortunately...", use "cannot / we are unable to", DÊ UMA RAZÃO (as / because) e ofereça uma alternativa. / For a polite request, use an indirect question. For a polite refusal, start with "I am afraid", give a reason, and offer an alternative.

Direct (rude) / Direto (rude)Softened / Suavizado
Send the report.I was wondering if you could send the report.
Do it by Friday.Would it be possible to do it by Friday?
No, we cannot.I am afraid we cannot, as our budget is fixed.
We will not do it.We must respectfully decline, but we could review it later.

Key point / Ponto-chave: um pedido suave vira pergunta indireta ("I was wondering if you could...") e um NÃO educado SEMPRE traz uma razão e, quando possível, uma alternativa. Cuidado: diga "I am afraid we cannot" (sem "that"); e "decline the request" (sem "of"). / A soft request becomes an indirect question, and a polite no always carries a reason and, if possible, an alternative.
