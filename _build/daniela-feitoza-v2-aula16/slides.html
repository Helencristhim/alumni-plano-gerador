Chapter 3

The Used-To Rule

Used to + base verb for habits that have changed

Habit Discovery

Listen and Notice

We used to work with the old legacy system.

The team used to update the prices by hand.

I didn’t use to trust the data.

Did you use to work on weekends?

Each one is true in the past, not today. What comes right after "used to"?

Used-To Rule

Before vs Now

FormStructureExample
Affirmativeused to + base verb"We used to work by hand."
Negativedidn’t use to + base verb"I didn’t use to trust the data."
QuestionDid + subject + use to + base verb?"Did you use to work late?"
MeaningA habit or state that was true before, but is not true now.

In the negative and question the d disappears: use to, not "used to".

Common Mistake

Wrong Form vs Right Form

✗ WRONG

"I used to working with the old system."

✓ RIGHT

"I used to work with the old system." (used to + base verb)

✗ WRONG

"Did you used to work on weekends?"

✓ RIGHT

"Did you use to work on weekends?" (no "d" in the question)

Used to + base verb (never -ing). In questions and negatives, drop the "d": use to.

Listening

The Coach’s Tip

A voice note about used to. Sound only.

0:000:00
1. According to Sarah, what is "used to" used for?
For something that was true before but is not true now, like "I used to work with the old legacy system."
2. What is the form, and what is the common mistake?
It is always "used to" plus the base verb — "I used to work", never "used to working" and never "use to work" in the affirmative past.
3. What happens in questions and negatives?
The "d" disappears: "Did you use to work late?" and "I did not use to work on weekends."
