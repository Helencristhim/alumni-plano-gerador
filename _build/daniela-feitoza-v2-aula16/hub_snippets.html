Warm-up — Callback da Aula 15

Review

Na Aula 15 você aprendeu a montar a cena no passado — o que estava acontecendo (was/were + verbo-ing) quando um momento interrompeu (past simple): "We were monitoring the dashboard when an alert appeared". Aquilo descreveu uma noite. Agora afaste a câmera: antes daquela noite, como as coisas costumavam ser todo dia? Hoje você vai falar de hábitos e rotinas do passado com used to — algo que era verdade antes, mas não é mais. (A past habit that has changed — true before, not true now.)

Stage 1.1: Vocabulary Cards

Vocabulary

Ouça cada palavra, leia o exemplo e a tradução. Toque em Listen para ouvir.

Legacy -- an old system still in use but outdated (PT: sistema legado / antigo)
"We used to work with the old legacy system."
Manual -- done by hand, without automation (PT: manual / feito à mão)
"The manual process took three full days."
Spreadsheet -- a document with rows and columns, like Excel (PT: planilha)
"They updated the prices on a huge spreadsheet."
Routine -- a regular, repeated way of doing your work (PT: rotina)
"Our daily routine used to be very different."
Automate -- to make a system do a task by itself (PT: automatizar)
"Gradually, we started to automate everything."
Replace -- to put a new thing in the place of an old one (PT: substituir / trocar)
"The new platform replaced the old spreadsheets."
Switch -- to change from one system or option to another (PT: mudar / trocar)
"We switched from paper to a digital platform."
Gradually -- slowly and step by step (PT: gradualmente / aos poucos)
"Gradually, the manual work disappeared."
Nowadays -- at the present time, in contrast with the past (PT: hoje em dia / atualmente)
"Nowadays the new platform does it in minutes."

Stage 1.3: Grammar in Context

GRAMMAR

Leia o texto e responda às perguntas.

Two years ago, the routine was very different. The team used to work with an old legacy system, and they used to update the prices by hand on a spreadsheet — these are past habits that have changed, so we use used to + base verb. Daniela didn’t use to trust the data; notice that in the negative the "d" disappears (didn’t use to, not "didn’t used to"). When Marcus asks "Did you use to work late?", the question also drops the "d". Then, gradually, the company started to automate the work. Nowadays the new platform replaces the old routine and does everything in minutes.

1. Why does the text say "the team used to work with a legacy system"?
A It is a past habit that is not true now.
B It is something happening right now.
C It is a plan for the future.
2. In the negative and the question, what happens to "used to"?
A It stays the same: "didn’t used to".
B The "d" disappears: "use to".
C It changes to "using to".
3. Which sentence is correct?
A "We used to working by hand."
B "We used to work by hand, but nowadays we automate it."
C "Did you used to work by hand?"

Stage 1.4: Grammar Tip — Used To (used to + base verb)

GRAMMAR

Como falar de hábitos do passado (bilingue EN + PT).

FormStructureExample
Afirmativoused to + verbo base"We used to work by hand."
Negativodidn’t use to + verbo base"I didn’t use to trust the data."
PerguntaDid + sujeito + use to + verbo base?"Did you use to work late?"
Regra (PT)Use used to + verbo base para um hábito do passado que mudou (era verdade antes, não é mais). NUNCA "used to working" (sempre verbo base). Na pergunta e na negativa o d some: Did you use to..? / didn’t use to... Contraste com hoje usando nowadays.
