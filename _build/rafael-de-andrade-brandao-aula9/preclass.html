Stage 1.4: Grammar Tip -- Comparatives

GRAMMAR

Usado para comparar dois produtos ou soluções. / Used to compare two products or solutions.

Form / FormaUse / UsoExample / Exemplo
adjetivo curto + er + thanComparar com adjetivo curto / Compare with a short adjectivefaster than the old one
more + adjetivo longo + thanComparar com adjetivo longo / Compare with a long adjectivemore reliable than the others
forma irregularAdjetivos especiais / Special adjectivesgood -> better, bad -> worse
Key point / Ponto-chaveUse than para nomear a segunda coisa. NUNCA use more com um adjetivo terminado em -er: diga "faster", não "more faster".
