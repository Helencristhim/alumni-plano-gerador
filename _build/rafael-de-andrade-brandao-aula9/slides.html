Grammar Discovery

Comparatives -- show why your solution is better

Grammar Discovery

Listen and Notice

What pattern do the orange words follow?

"Our system is faster than the old one."

"This plan is cheaper than the others."

"Our solution is more reliable than the current one."

"The new model is more efficient than the old model."

Grammar Rule

Comparatives with -er / more

TypeRuleExample
Short adjectiveadjective + er + thanfast -> faster than
Long adjectivemore + adjective + thanreliable -> more reliable than
Irregularspecial formgood -> better, bad -> worse
Key pointUse than to compare two things. Never use more with an -er adjective: say faster, not "more faster".

Key idea: A comparative shows that one thing is more than another. Short adjectives add -er (cheaper, faster). Long adjectives use more (more reliable, more efficient). Use than to name the second thing.

Common Mistake

Watch Out

"Our solution is more faster than the old one."
"Our solution is faster than the old one."

With a short adjective, add -er only. Never use more together with an -er word.
