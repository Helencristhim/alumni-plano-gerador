The Post-Mortem -- What We Would Have Done Differently

Running a project review of a launch that fell short of target — the Halo sensor line. Key words: hindsight, setback, root cause, overlook, misjudge, underestimate, backfire, turning point, salvage, write off, shortfall, momentum. Structure: third conditional (If + had + past participle, would have + past participle) and the mixed conditional (past cause, present result: "If we had hired her, we would not be firefighting now").

Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Hindsight -- understanding of an event only after it has happened
"In hindsight, the shortfall was avoidable."
Setback -- a problem that delays or reverses progress
"The firmware defect was the setback that hurt the most."
Root cause -- the fundamental reason a problem happens
"The root cause was speed: we rushed the decision."
Overlook -- to fail to notice or consider something
"We overlooked how fast the competition would cut prices."
Misjudge -- to form a wrong opinion or estimate of something
"We misjudged the market because we were in a hurry."
Underestimate -- to think something is smaller or weaker than it really is
"We underestimated the competition."
Backfire -- to produce the opposite result to the one intended
"If we do not learn from this, the next launch will backfire."
Turning point -- the moment when a situation changes decisively
"This review is the turning point for the next launch."
Salvage -- to rescue something of value from a failure
"There is real momentum, so we can salvage the line."
Write off -- to accept that something is a total loss
"I am not writing the product off; it can be saved."
Shortfall -- the amount by which something falls short of the target
"The headline is a thirty percent shortfall against target."
Momentum -- the force that keeps progress moving forward
"If we had not built the brand, there would be no momentum now."

Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

Izabel opens the post-mortem on the Halo launch, which missed its target. "In hindsight, the shortfall was avoidable," she says. "If we had run a regional pilot, we would have caught the pricing problem. And if we had tested with real customers, we would have found the firmware defect in a week." The root cause was speed: they misjudged the market and underestimated the competition. Andy asks whether they should write the line off. "No," she answers. "There is real momentum. If we had not built the brand so carefully, there would be nothing to salvage now — but there is. And if we had hired an operations lead a year earlier, we would not be firefighting today."

1. "If we had run a pilot, we would have caught it." Did AVRT run a pilot?
A No — the third conditional imagines a past that did not happen.
B Yes, and it worked well.
C Yes, but only in one region.
2. "If we had hired her earlier, we would not be firefighting now." Which conditional is this?
A A pure third conditional (past result).
B A mixed conditional — past cause, present result.
C A second conditional.
3. Why "had run" and not "would have run" in the if-clause?
A The if-clause uses had + past participle; "would" never goes in the if-clause.
B Because "run" is irregular.
C Because the sentence is a question.

Stage 1.4: Grammar Tip -- Third & Mixed Conditionals

GRAMMAR

Talking about an imagined past and its results.

TypeStructureExample
Third conditional
imagined past		If + had + past participle, ... would/could/might have + past participleIf we had tested it, we would have found the bug.
Mixed conditional
past cause, present result		If + had + past participle, ... would + base (now)If we had hired her, we would not be firefighting now.
OrderBoth orders work. If-clause first → use a comma. Result first → no comma: "We would have caught it if we had run a pilot."
Watch outNEVER use "would" in the if-clause — use had + past participle. NEVER drop "have" in the result. Wrong: "If we would have tested", "we would caught it", "we would not have been firefighting now" (when the result is happening now).
