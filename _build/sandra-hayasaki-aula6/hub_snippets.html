Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

Hello! My name is Anna. I live in a small house. There is a kitchen and a living room. In the living room, there is a sofa and a table. There are two bedrooms and one bathroom. In my bedroom, there is a big window. I like my house very much. It is small but comfortable.

1. How many bedrooms are there in Anna's house?

B Two bedrooms.
C Three bedrooms.
2. What is there in the living room?
A A sofa and a table.
B A bed and a window.
C A kitchen and a bathroom.
3. Which sentence is correct?
A "There is two bedrooms."
B "There are two bedrooms."
C "There are one bathroom."

Stage 1.4: Grammar Tip -- there is / there are

GRAMMAR

Usado para dizer o que existe num lugar. / Used to say what exists in a place.

Form / FormaStructure / EstruturaExample / Exemplo
One (+) / umThere isThere is a kitchen.
Two or more (+) / dois ou maisThere areThere are two bedrooms.
Question (?) / perguntaIs there...?Is there a table?
Negative (-) / negativaThere is noThere is no table.
Key point / Ponto-chaveUse there is para uma coisa e there are para duas ou mais. Diga "There are two bedrooms", não "There is two bedrooms".
