Chapter 3 -- Putting It Together

Grammar Discovery

The five moves and the transitions that join them

Grammar Discovery

Where does each orange phrase belong in the talk?

"As you can see in the chart, costs dropped."

"To sum up, the project is on track. Thank you."

Grammar Rule

Open to Close

MoveTransition phrase
1. Open"Let me start with..."
2. Signpost"First... then... finally..."
3. Data"As you can see in the chart..."
4. Q&A"Are there any questions?"
5. Close"To sum up... thank you."

Key idea: A full talk is five moves in the same order. Between each move you use a transition phrase, so the talk has a smooth flow and moves like one story. With a calm pace and poise, the talk feels polished.

Common Mistake

Watch Out

"Good morning. Costs dropped fifteen percent."
"Good morning. Let me start with where we are. Now, let me show you the numbers. As you can see in the chart, costs dropped."

Do not jump from the opening straight to the data. Use a transition so the talk has a smooth flow and does not feel abrupt.
