Chapter 3 -- Advice

Grammar Discovery

Giving advice with should and shouldn't

Grammar Discovery

Listen and Notice

What do the orange words show about advice?

"You should make a study plan."

"You shouldn't worry so much."

"She should ask her teacher for help."

"Should I talk to her about it?"

Grammar Rule

Should for Advice

TypeRuleExample
Positiveshould + base verbYou should rest
Negativeshouldn't + base verbYou shouldn't worry
QuestionShould + subject + verb?Should I ask for help?
Soft adviceWhy don't you + verb?Why don't you take a break?
Same for allverb never changesShe should ask

Key idea: use should + base verb to give advice (You should rest). The negative is shouldn't (You shouldn't worry). After should, the verb never changes: she should ask, not "she should asks". To ask for advice, say Should I...? For softer advice, try Why don't you...?

Common Mistake

Watch Out

"She should rests more." / "You should to rest tonight."
"She should rest more." / "You should rest tonight."

After should, always use the base verb (should rest, should ask). Never add -s and never add to.
