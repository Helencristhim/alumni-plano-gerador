Chapter 3: The Code

Have Been Doing

Duration, with for and since

Grammar Discovery

Listen and Notice

"I have been restoring furniture for six years."

"How long have you been doing this?"

"She has been working since morning."

"I have restored twenty chairs."

The green ones answer how long. The blue one answers how many. What changed in the verb?

FormUseExample
Present Perfect Continuous
have / has been + verb-ing		An activity that started in the past and is STILL going. How long.I have been restoring furniture.
for / sincefor + a period · since + a starting pointfor six years / since 2019
QuestionHow long + have/has + subject + been + -ingHow long have you been doing this?
vs SimpleSimple = a finished result, how many.I have restored twenty chairs.

The rule in one line: how long → have been doing. how many → have done.

Common Mistake

Right vs Wrong

"I am restoring furniture for six years."
"I have been restoring furniture for six years."
"I have been doing this since six years."
"I have been doing this for six years."
"How long you have been doing this?"
"How long have you been doing this?"

For a hobby you still do, use have been doing, not the present continuous. Use since for a starting point and for for a period. And in questions, have comes before you.

For or Since?

for = a period of time · since = a starting point

"I have lived here ___for six years."
"I have lived here ___since 2019."
"She has been tinkering in the workshop ___since lunchtime."
"He has been playing the guitar ___for nearly twenty years."

Dialogue

How Long Have You Been at This?

T
This is beautiful work, Ana. How long have you been restoring furniture?
A
I have been doing it for about six years now, since we moved to the countryside.
T
And how did you learn? Did you take a course?
A
No, I picked it up on my own. I have been teaching myself from videos the whole time.
T
Have you ever thought about selling your pieces?
A
I have been thinking about it for a while, honestly. But I am not ready yet. I still lose track of time and forget it is a business.
