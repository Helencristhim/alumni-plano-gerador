Chapter 4: The Code

The Language of What If

For imagined situations and hard choices

Grammar Discovery

Notice the Second Conditional

"If I had more time, I would automate the checkout first."

"If we tested everything by hand, we would miss the deadline."

"If the screen failed, nobody would lose money."

"What would you do if the tests failed on Friday?"

PartFormExample
If-part (condition)If + past simpleIf I had more time...
Result-partwould + base verb...I would automate the checkout.
QuestionWhat would + subject + base verb + if...?What would you do if it failed?

Second conditional = If + past simple, would + base verb. The past form does not mean the past — it marks the situation as imagined.

Grammar Practice

Language Focus

Plan It in the Conditional

Common Mistake

Wrong vs Right

✗ WRONG

"If I will have more time, I will automate it."

✓ RIGHT

"If I had more time, I would automate it."

✗ WRONG

"If it failed, nobody will lose money."

✓ RIGHT

"If it failed, nobody would lose money."

In the second conditional, use the past simple after "if" and would + base verb in the result. Never "will" in the if-part.

Dialogue

The Plan with David

D
Ana, I read your test plan for Friday. If you had one more day, what would you change?
A
If I had one more day, I would automate the reporting screen too. Right now, it is a trade-off, so I am focusing on the checkout.
D
Makes sense. And if the reporting screen broke in production, how bad would it be?
A
Low. If it failed, nobody would lose money, so I can justify postponing it to the next sprint.
D
Good. Leadership will ask about the worst case. What would you do if the tests found a serious bug on Friday morning?
A
If the tests found a serious bug, we would postpone the release by a few hours. And to mitigate the risk, I keep the old version ready.
D
So if the new feature failed badly, you would roll back?
A
Exactly. We would roll back, and nobody would notice. It is a reasonable contingency, and I can explain every choice.
