Chapter 3: The Code

If This, Then Will

The first conditional for real situations

Grammar Discovery

Listen and Notice

"If the server gets overloaded, the app will crash."

"If we skip this test, a bug will reach production."

"If the file is too big, the upload will fail."

"If you click Submit twice, the system will create two orders."

After if, which tense is the verb? And which part uses will?

PartFormExample
Cause (if-clause)If + present simpleIf the load is high,
Effect (result)will + base verbthe app will slow down.
Key rulePresent simple after if — never "will" after if. The order can swap (no comma when the if-clause is second).

We use the first conditional for a real cause and its likely effect — exactly what you do in a risk analysis.

Common Mistake

Right vs Wrong

"If the server will crash, we will lose data."
"If the server crashes, we will lose data."
"If the test fail, I will open a ticket."
"If the test fails, I will open a ticket."
"If you click twice, it creates two orders and crash."
"If you click twice, the system will create two orders."

Use the present simple after "if" (with the third-person -s), and will for the effect.

Chapter 4: Planning the Rollout

Mapping Risks with Diego

A risk call before Friday's release

Dialogue

The Risk Call

D
Hi Ana, we're planning the rollout for Friday. Can you look at the risks with me?
A
Sure, Diego. If we deploy on Friday afternoon, and something breaks, who will fix it over the weekend?
D
Good point. If a bug appears, I'll be on call, but the team will be smaller.
A
Right. And if the load is high on Monday morning, will the new feature handle it?
D
We added a fallback. If the main service times out, the app will switch to the backup.
A
Nice. But if the database becomes a bottleneck, the fallback won't help much, will it?
D
True. If the queries get too slow, the whole system will cascade into errors.
A
So, if we test the load first, we'll catch that before the rollout.
D
Exactly. If we do the load test tomorrow, we'll know the threshold by Friday.
