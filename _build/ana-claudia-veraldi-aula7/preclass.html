Lesson 07 -- Pre-class

Explaining a Test Scenario -- Sequencing

Walking a new teammate through a test case step by step: the precondition, the input, what you do to trigger the check, the outcome, and one edge case. Key words: test case, precondition, input, outcome, edge case, sequence, to walk through, to set up, to trigger, to proceed, to submit, straightforward. Structure: sequencers and time linkers (first, then, next, after that, finally).

Stage 1.1: Vocabulary Cards

Vocabulary

Listen to each word and read the example. Tap Listen to hear it.

Test case -- a written set of steps and the expected result for one function
"Let me walk you through the test case for the login page."
Precondition -- something that must be ready or true before the test starts
"The precondition is that the test account already exists."
Input -- the data or information you enter into the system
"The input is the email and the password."
Outcome -- the final result you get after the steps
"The expected outcome is that the dashboard opens."
Edge case -- an unusual situation at the limit of normal use
"As an edge case, I try a wrong password."
Sequence -- the fixed order in which the steps happen
"Follow the sequence and the test is easy."
To walk through -- to explain something step by step, slowly and clearly
"Can you walk me through this test scenario?"
To set up -- to prepare something before you use it
"First, I set up the test account."
To trigger -- to make an action or event happen
"I click the button to trigger the check."
To proceed -- to continue to the next step
"Once the page loads, you can proceed."
To submit -- to send information or a form to the system
"Finally, I submit the report."
Straightforward -- simple, clear, and easy to follow
"The whole sequence is straightforward."

Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

Ana is helping Lucas, a new teammate. She walks him through a test case for the login page. First, she checks the precondition: the test account must already exist. Then she opens the login page and enters the input, the email and the password. Next, she clicks the button to trigger the check. After that, she waits for the outcome: the user reaches the dashboard. Before she writes the report, she also tries one edge case, a wrong password. Finally, she saves the results and submits the report. The whole sequence is straightforward once you follow the steps in order.

1. Which word introduces the very FIRST step in the sequence?
A "First"
B "Finally"
C "After that"
2. "After that, she waits for the outcome." What does "after that" tell us?
A This is the first step.
B This step comes right after the previous one.
C This step happens at the same time as the input.
3. Which word marks the LAST step in the sequence?
A "Next"
B "Before"
C "Finally"

Stage 1.4: Grammar Tip -- Sequencers and Time Linkers

GRAMMAR

How to put the steps of a process in a clear order.

SequencerUseExample
First, / To start,The very first step of the process.First, I set up the test account.
Then / Next, / After that,The following steps, one after another in order.Then I enter the input. After that, I click the button.
Before / OnceOne action in relation to another (which happens first).Before you proceed, check the precondition. Once it loads, read the outcome.
Finally,The last step. Use a comma after First, Next, After that and Finally at the start of a sentence; "then" usually takes no comma.

Stage 4: Situational Quiz

Quiz

Choose the best answer for each real work situation.

A new teammate asks you to explain the test. You start with:
A "First, I set up the test account and open the login page."
B "Finally the test account I set up."
C "I set up finally the account first then."
You want to describe the next two steps in order. You say:
A "Finally I enter the input, then first I click the button."
B "Then I enter the input. After that, I click the button to trigger the check."
C "After that first I trigger then the input."
You reach the last step of the walk-through. You say:
A "Finally, I save the results and submit the report."
B "First, I save the results and submit the report."
