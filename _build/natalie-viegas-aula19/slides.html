Chapter 3 -- The Condition Code

Grammar Discovery

Conditional connectors -- setting clear conditions

Grammar Discovery

Listen and Notice

What are the orange words? How do they set a condition?

We can move on the timeline, as long as the volume stays the same.

Provided that you sign this week, we will hold the price.

Unless we sign a longer contract, we cannot lower the rate.

Neutral: "We can lower the rate if you sign a longer contract."

Grammar Rule

Conditional Connectors

ConnectorMeaningExample
as long as + presentonly if this is true"We can move as long as the volume stays the same."
provided that + presentonly on this condition (formal)"Provided that you sign this week, we will hold the price."
unless + presentif NOT (the negative condition)"Unless we sign longer, we cannot lower the rate."
if + present (neutral)plain condition, less weight"We can lower it if you sign longer."

Key idea: use as long as / provided that to say "only if this is true", and unless to say "if not". After the connector, use the present (stays, sign); in the other part, use can / will / cannot. These connectors let you stand firm and stay flexible at the same time.

Common Mistake

Watch Out

"Unless you do not sign, we cannot lower the rate."
"Unless you sign, we cannot lower the rate."

Unless already means "if not", so never add another "not". And after a connector, use the present (sign, pay), never "will".

Grammar Practice

Set the Condition

"We can move on the timeline, _____as long as the volume stays the same." (only if this is true)
"_____Unless we sign a longer contract, we cannot lower the rate." (if not)
"Provided that you _____sign this week, we will hold the price." (use the present, not 'will')
"We have some leeway, _____as long as the deadline does not move." (only if this is true)
