Chapter 3 -- The Code

Grammar Discovery

How much is it? -- Can I try it on?

Grammar Discovery

Listen and Notice

How do we shop for clothes?

"How much is this jacket?"

"I need new boots for the winter."

"Can I try it on? What size do you have?"

"In summer I wear a light dress."

Grammar Rule

Shopping for Clothes

FormWhen we use itExample
I wear + clothesTo say what you put onI wear a jacket.
I need + clothesTo say what you want to buyI need new boots.
How much is it?To ask the priceHow much is this scarf?
Can I try it on?To try the clothing onCan I try it on?

Key idea: ask the price with How much is it? and ask to try clothes with Can I try it on?

Common Mistake

Watch Out

"How many is this jacket?"
"How much is this jacket?"

For the price of one item, use How much is it?, not How many.
