Stage 1.4: Grammar Tip -- like + verb-ing

GRAMMAR

Usado para falar de hobbies e do que gostamos de fazer. / Used to talk about hobbies and what we enjoy doing.

Form / FormaStructure / EstruturaExample / Exemplo
Positive (+) / afirmativolike + verb-ingI like playing games.
Negative (-) / negativodon't like + verb-ingI don't like cooking.
Question (?) / perguntaDo you like + verb-ing?Do you like running?
Spelling / OrtografiaDepois de like, o verbo termina em -ing: play → playing, watch → watching. Run → running (dobra o n). Dance → dancing (tira o e).
