Chapter 3 -- The Code

Grammar Discovery

like + verb-ing

Grammar Discovery

Listen and Notice

Look at the orange words. What comes after "like"?

"I like playing games." (+)

"I don't like cooking." (-)

"Do you like running?" (?)

"She likes dancing." (+)

Grammar Rule

Like + verb-ing

FormStructureExample
Positive (+)like + verb-ingI like playing games.
Negative (-)don't like + verb-ingI don't like cooking.
Question (?)Do you like + verb-ing?Do you like running?

Key idea: after like, the verb ends in -ing: play → playing, run → running, dance → dancing.

Common Mistake

Watch Out

"I like play games."
"I like playing games."

After like, the verb ends in -ing, not the base form.

Grammar Practice

Add the -ing

"I like _____playing games." (play)
"I don't like _____cooking." (cook)
"Do you like _____running?" (run)
"She likes _____dancing." (dance)
