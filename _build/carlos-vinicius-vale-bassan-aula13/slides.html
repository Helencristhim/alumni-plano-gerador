Chapter 3 -- Progress Over Time

The "It Has Been Happening" Language

Present perfect continuous: have been + verb-ing

Listen and Notice

What Is the Pattern?

Each sentence reports progress over time. Notice the orange words.

"We have been tracking five metrics since day one."

"Adoption has been climbing every week."

"The teams have been using the new system for three months."

"Engagement has been rising since the town hall."

The Rule

Have / Has + Been + Verb-ing

FormExample
I / we / they + have been + -ing"We have been tracking five metrics since day one."
He / she / it + has been + -ing"Adoption has been climbing every week."
since + a starting point"Engagement has been rising since the town hall."
for + a length of time"The teams have been using it for three months."

Tip: use the present perfect continuous for an action that started in the past and is still going on -- it stresses the duration or the progress ("adoption has been climbing"). Use the present perfect simple for the finished result ("we have hit the target"). Use since for a start point and for for a length of time.

Common Mistake

Do Not Drop Been, and Watch Since vs For

"Adoption has climbing every week. The teams have used the system since three months."
"Adoption has been climbing every week. The teams have been using the system for three months."

The full form is have/has + been + verb-ing: never drop "been". Use for with a length of time (for three months) and since with a start point (since the town hall).
