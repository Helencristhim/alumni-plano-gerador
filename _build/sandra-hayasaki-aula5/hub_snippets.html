Stage 1.3: Grammar in Context

GRAMMAR

Hello! My name is Anna. I love food. In the morning, I drink coffee and eat bread. I like a big breakfast. For lunch, I eat chicken with rice. I really like chicken. I don't like fish, so I never eat it. I eat a lot of fruit because it is healthy. When I am at a restaurant, I always ask for water. I am happy when I eat good food.

3. Which sentence is correct?
A "I no like fish."
B "I don't like fish."
C "I don't likes fish."

Stage 1.4: Grammar Tip -- like / don't like

GRAMMAR

Usado para dizer o que você gosta e não gosta. / Used to say what you like and do not like.

Form / FormaStructure / EstruturaExample / Exemplo
Positive (+) / gostoI likeI like rice.
Negative (-) / não gostoI don't likeI don't like fish.
Question (?) / você gosta?Do you like...?Do you like coffee?
Short answer / respostaYes, I do. / No, I don't.Yes, I do.
Key point / Ponto-chavePara dizer não, use don't (do + not), nunca "no". Diga "I don't like fish", não "I no like fish".
