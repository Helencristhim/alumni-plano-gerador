Grammar Rule

Like and Don't Like

FormStructureExample
Positive (+)I likeI like rice.
Negative (-)I don't likeI don't like fish.
Question (?)Do you like...?Do you like coffee?
Short answerYes, I do. / No, I don't.Yes, I do.

Key idea: to say no, use don't (do + not), not "no". Say "I don't like fish", not "I no like fish".

Common Mistake

Watch Out

"I no like fish."
"I don't like fish."

To say no, use don't (do + not), not "no".
