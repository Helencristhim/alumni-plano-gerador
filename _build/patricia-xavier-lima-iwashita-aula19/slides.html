Chapter 3: The Code

We've Grown...

Present perfect for results · since / for · that's why...

Grammar Discovery

Listen and Notice

"We've grown adoption by forty percent."

"Your team has saved many hours this year."

"We've reduced churn since January."

"The partnership has delivered value for six months."

What two words build each result, and which words follow — since or for?

StructureUseExample
have / has + past participleA result that is still true nowWe have grown adoption.
since + point in timeWhen the result started...since January.
for + period of timeHow long the result has lasted...for six months.
That's why... / This means...Turn a result into persuasionThe ROI is clear. That's why renewing makes sense.

The present perfect (have/has + past participle) shows a result that is true today. Use since for a point in time and for for a period. Then add that's why / this means to persuade.

Common Mistake

"We grew... since" vs "We've grown... since"

"We grew adoption by 40% since January."
"We've grown adoption by 40% since January."
"We've worked together since six months."
"We've worked together for six months."

For a result still true now, use the present perfect (we've grown), not the past simple. Use since + a point in time (since January) and for + a period (for six months). Keep the past simple only for a finished time: we grew adoption last quarter.
