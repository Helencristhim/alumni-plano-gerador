Chapter 3 — The Code

If + Had... Would Have

Look at the purple words. How do we talk about a different past — one that did NOT happen?

If we had caught it sooner, we would have avoided the delay.
If they had escalated earlier, we would not have lost two weeks.
If the downturn had been deeper, the outcome could have been very different.
What would you have done differently?
PartFormExample
If-clause (imagined past)If + had + past participle"If we had caught it sooner..."
Result (imagined past)would / could have + past participle"...we would have avoided the delay."
QuestionWhat would + subject + have + done"What would you have done differently?"
Grammar Rule

The Third Conditional

FunctionPatternExample
Imagined past conditionIf + had + past participle"If we had caught it sooner..."
Imagined past resultwould have + past participle"...we would have avoided the delay."
Softer / possible resultcould have + past participle"...it could have been worse."
Ask about a past choiceWhat would + subject + have done?"What would you have done differently?"
Common Mistake

Watch Out!

"If we would have caught it sooner, we would have avoided the delay."
"If we had caught it sooner, we would have avoided the delay."

The if-clause takes had + past participle — never "would have". Only the result uses would / could have.
