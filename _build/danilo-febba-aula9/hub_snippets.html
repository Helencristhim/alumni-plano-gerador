Skills & Abilities -- What You Can Do at Work

Fale do que você consegue fazer no trabalho usando can / can't. Palavras-chave: ability, code, design, build, operate, handle, troubleshoot, present. Gramática: can / can't para habilidade (I can code, I can't drive; Can you...? Yes, I can / No, I can't; she can -- sem -s no verbo).

Stage 1.3: Grammar in Context

GRAMMAR

Leia o texto e responda às perguntas. (Repare em can / can't e no verbo base depois deles.)

Danilo is a data engineer, and he has many skills. He can code in Python, and he can build data pipelines. He can handle large datasets every day. When the system breaks, he can troubleshoot the problem. He can't make beautiful graphics, but that is okay. His colleague Sofia can design dashboards, and she can present the results. Can they work together? Yes, they can. A good team can do many things.

1. Na frase "He can code", por que NÃO dizemos "He can codes"?
A Porque "code" nunca leva -s.
B Porque depois de "can" o verbo fica na forma base, sem -s.
C Porque "he" nunca usa -s.
2. Como se forma a pergunta "Você consegue programar?"
A "Can you code?" (Can vem primeiro, sem "do")
B "Do you can code?"
C "You can code?"
3. Qual frase está correta?
A "She can to design dashboards."
B "She can design dashboards."
C "She can designs dashboards."

Stage 1.4: Grammar Tip -- Can / Can't for Ability

GRAMMAR

Use can para o que você consegue fazer e can't para o que não consegue. Depois de can/can't, sempre o verbo na forma base.

FormEstrutura (PT)Example
Positivecan + verbo baseI can code
Negativecan't (= cannot) + verbo baseI can't drive
QuestionCan + sujeito + verbo?Can you code?
Short answerresposta curtaYes, I can. / No, I can't.

Atenção (erro comum): Com he/she/it, "can" NÃO muda e o verbo NÃO leva -s: "She can speak" (não "She can speaks"). Nunca use "to" depois de can: "I can code" (não "I can to code"). A pergunta começa com Can, sem "do": "Can you build it?" (não "Do you can build it?").

Stage 4: Situational Quiz

Quiz

Escolha a melhor frase para cada situação no trabalho.

Você fala de uma habilidade que tem. Qual frase usar?
A "I can to code in Python."
B "I can code in Python."
C "I can codes in Python."
Você pergunta a um colega sobre uma habilidade dele. Você diz:
A "Do you can build a pipeline?"
B "You can build a pipeline?"
C "Can you build a pipeline?"
Você fala de algo que NÃO consegue fazer. A frase correta é:
A "I can't make beautiful graphics."
B "I don't can make beautiful graphics."
C "I can't to make beautiful graphics."
Você fala da habilidade de uma colega (she). Você diz:
A "She cans design dashboards."
B "She can design dashboards."
C "She can designs dashboards."

Stage 5: Free Production

Reflection

Grave você respondendo à pergunta abaixo. Não existe resposta certa ou errada.

Fale do que você consegue fazer no trabalho. Diga três coisas que você can fazer e uma coisa que você can't fazer ainda. Por exemplo: "I can code in Python and I can build data pipelines, but I can't make beautiful graphics yet." Fale com calma.
