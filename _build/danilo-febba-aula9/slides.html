Chapter 3

The Can-Do Code

can / can't for ability

Grammar Discovery

Listen and Notice

I can code in Python. / I can't drive a truck.

She can speak English. / He can design dashboards.

Can you build a pipeline? / Yes, I can. No, I can't.

We can troubleshoot the system. / They can't work today.

FormStructureExample
Positivecan + base verbI can code
Negativecan't (= cannot) + base verbI can't drive
QuestionCan + subject + base verb?Can you code?
Short answerYes, I can. / No, I can't.Yes, I can.

Same for he / she / it: no -s on the verb. She can speak (NOT speaks). And no "to" after can: I can code (NOT I can to code).

Common Mistake

Right vs Wrong

✗ WRONG

"She can speaks English."

✓ RIGHT

"She can speak English."

✗ WRONG

"I can to code in Python."

✓ RIGHT

"I can code in Python."

✗ WRONG

"Do you can build it?"

✓ RIGHT

"Can you build it?"

After can, use the base verb: no -s, no "to". For questions, start with Can, never with "do".

R
Hi Danilo! We are building the new project team. What can you do well?
D
I can code in Python and SQL, and I can build data pipelines.
R
Great. Can you design the dashboards too?
D
I can design simple dashboards, but I can't make beautiful graphics.
R
That is fine. Can you handle large datasets?
D
Yes, I can. I handle big datasets every day.
R
Perfect. And when the system breaks, can you troubleshoot it?
D
Yes, I can troubleshoot the problem and fix it fast.
R
Excellent. Can you present the results to the client on Friday?
D
Yes, I can. I can present our results in a clear way.
