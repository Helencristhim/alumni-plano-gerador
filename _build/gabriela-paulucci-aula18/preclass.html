Stage 1.4: Grammar Tip -- Do you have...? / I would like...

GRAMMAR

No hotel, o recepcionista PERGUNTA com "Do you have...?" e você RESPONDE com "Yes, I have...". Para pedir algo de forma educada, use "I would like...". / At the hotel, the receptionist ASKS with "Do you have...?" and you ANSWER with "Yes, I have...". To request something politely, use "I would like...".

Function / FunçãoStructure / EstruturaExample / Exemplo
Receptionist asks / Recepcionista perguntaDo you have + a + coisa?Do you have a reservation?
You answer / Você respondeYes, I have + a + coisa.Yes, I have a reservation.
You request / Você pede (educado)I would like + a + coisa.I would like a double room.

Key point / Ponto-chave: para perguntar, comece com Do you have...?; para pedir com educação, diga I would like... (mais gentil que "I want"). / To ask, start with "Do you have...?"; to request politely, say "I would like..." (softer than "I want").
