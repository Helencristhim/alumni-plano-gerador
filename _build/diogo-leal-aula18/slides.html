Chapter 3 -- The Skills

Grammar Recap

Signpost the parts, present the data, and say WHY it matters

Grammar Recap

Listen and Notice

What job do the orange phrases do in the talk?

"First the status, then the numbers, finally the next steps."

"Costs dropped by fifteen percent, and revenue rose."

"I built one tool to save the team two hours."

"I automate the report so that managers save time."

Grammar Rule

The Three Patterns

Job in the talkPatternExample
Signpostfirst ... then ... finally"First the status, then the numbers, finally the next steps."
Datasubject + rose / dropped (by + number)"Costs dropped by fifteen percent."
Whyto + verb  /  so that + clause"I built it to save time, so that managers work less."

Key idea: A great talk joins all three. You signpost the parts so the audience can follow, you present the data clearly, and you always say WHY it matters with to or so that. End with a short recap and one takeaway.

Common Mistake

Watch Out

"Costs dropped of fifteen percent, so that to save money."
"Costs dropped by fifteen percent, so that the team saves money."

For data, use by + the number. After so that, use a small sentence with a subject and a verb -- never "to".
