Chapter 3

The Habits Code

How to talk about the way things used to be

Discovery

Spot the Pattern

Each line describes a past habit that has changed. What do the orange words do?

I used to freeze on every English call.

I would rehearse "hello" before I picked up the phone.

I used to apologize for my English every two minutes.

Use used to and would to describe past habits — things you did again and again before, but don't anymore. The base verb follows: used to freeze, would rehearse.

used to + verb = a regular past habit OR a past state, now changed — I used to freeze on calls. (works for both actions and states like be, have)

would + verb = a repeated past action only — I would rehearse before every call. (do NOT use would for states: say I used to be nervous, not "I would be nervous")

They're perfect for reflecting on how far you've come.

The Structure

Used To vs Would

FormUseExample
used to + verbpast habit OR past state, now changedI used to freeze on calls. / I used to be nervous.
would + verbrepeated past action only (not states)I would rehearse before every call.
didn't use to + verba habit you did NOT have beforeI didn't use to speak up in meetings.
Did you use to...?ask about a past habitDid you use to avoid English calls?
Watch Out

Common Mistake

I would be very nervous on calls.
I used to be very nervous on calls.

Use used to — not would — for past states like be, have or feel: I used to be nervous, not "I would be nervous". And in the negative and question, drop the "d": I didn't use to speak up and Did you use to...?, never "didn't used to".

Practice

How Things Used to Be

"A year ago, I ___ freeze whenever the phone rang." → used to / would freeze whenever the phone rang.
"I ___ be nervous before every meeting." → used to be nervous (state → used to, not would).
"I ___ rehearse every sentence before a call." → used to / would rehearse every sentence.
"I ___ (negative) speak up in English — now I lead the meeting." → didn't use to speak up (not "didn't used to").

Spot the Error

Fix the Line

"I would be very nervous on every call."
"I used to be very nervous on every call." (used to for states, not would)
"I didn't used to speak up in meetings."
"I didn't use to speak up in meetings." (drop the "d" in the negative)
"My objective is sustain this growth."
"My objective is to sustain this growth." (objective + to + verb)
"For the first time the ambition feels reach."
"For the first time the ambition feels reachable." (adjective form)
Build

Sentence Building

Say each line out loud, then click to compare

1. Contrast then and now (used to + achievement).
"I used to dread English calls; now closing a deal in English is my proudest achievement."
2. Describe an old routine (would).
"Before a call I would rehearse every sentence, and I'd still freeze."
3. Name your aim (objective + aspiration).
"My objective is to enter Europe, and my aspiration is a brand Paris knows."
4. Talk about keeping going (sustain + momentum).
"To get there I have to sustain this momentum and grow the team."
5. Own your confidence (breakthrough + ambition + vision).
"The real breakthrough was losing the fear; now the vision is clear and the ambition feels reachable."
