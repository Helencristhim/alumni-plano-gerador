"I am writing regarding the supplier report." (A respeito de / sobre)

Stage 1.4: Grammar Tip -- The Present Perfect

GRAMMAR

Usamos o present perfect (have/has + particípio passado) para uma ação concluída cujo resultado importa agora, sem dizer a hora exata: "I have attached the report" (o arquivo está aqui agora). / We use the present perfect for a finished action with a result now, without an exact time.

Forma / FormPatternExample
Afirmativo / PositiveI/you/we/they have + particípioI have attached the report.
Afirmativo (he/she)he/she/it has + particípioKaren has sent the email.
Negativo / Negativehave not / has not + particípioI have not replied yet.
Pergunta / QuestionHave/Has + sujeito + particípio?Have you received my email?

Key point / Ponto-chave: com hora exata (yesterday, at 9) use o past simple -- "I sent it yesterday". O particípio de write é written e de send é sent. Use just / already / yet com o present perfect. / With an exact time use the past simple. Past participles: write -> written, send -> sent.
