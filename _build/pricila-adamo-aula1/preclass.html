Stage 1.4: Grammar Tip -- Present Perfect Continuous

Grammar

Para falar de uma acao que comecou no passado e continua ate agora (muitas vezes com for / since), use have / has + been + verbo-ing. Ela destaca a DURACAO e o esforco recente. / To talk about an action that started in the past and is still going on (often with for / since), use have / has + been + verb-ing. It highlights the duration and recent effort.

Forma / FormEnglishPortugues
AffirmativeI have been learning English for a year.Estudo ingles ha um ano.
Affirmative (he/she)She has been thriving since she retired.Ela vem prosperando desde que se aposentou.
NegativeI have not been wasting my time.Eu nao venho perdendo tempo.
InterrogativeWhat have you been doing lately?O que voce vem fazendo ultimamente?
Common mistakeNOT "I study English since 2000." Say "I have been studying since 2000."Com since / for, use have been + -ing.
