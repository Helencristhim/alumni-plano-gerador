Etapa 1.1: Tarjetas de Vocabulario

Vocabulario

Escucha cada palabra y lee el ejemplo. La traducción al portugués está entre paréntesis. Toca Escuchar para oírla.

Reportar -- informar los resultados a un grupo (PT: reportar)
"Hoy reporto los resultados del trimestre."
Logro -- algo bueno que se consigue (PT: conquista / feito)
"Bajar el fraude es un gran logro."
Meta -- el objetivo que se quiere alcanzar (PT: meta)
"Cumplimos la meta del trimestre."
Cifra -- un número o un dato (PT: cifra / número)
"Las cifras son muy claras."
Aumentar -- hacer más grande, subir (PT: aumentar)
"Las ventas aumentaron mucho."
Disminuir -- hacer más pequeño, bajar (PT: diminuir)
"Los fraudes disminuyeron."
Coincidir -- pensar igual que otra persona (PT: concordar / coincidir)
"Coincido contigo en la meta."
Sugerencia -- una idea que se propone (PT: sugestão)
"Tengo una sugerencia para el equipo."

Etapa 1.3: Gramática en Contexto

Gramática

Lee el texto y responde las preguntas.

Helio abre la reunión de equipo. "El trimestre pasado reportamos muy buenos resultados", dice. "Las ventas aumentaron un veinte por ciento y los fraudes disminuyeron mucho. El equipo logró la meta." Carolina responde con su opinión: "Me parece excelente. Estoy de acuerdo con tu informe y coincido contigo. Tengo una sugerencia para el próximo trimestre."

1. "Las ventas aumentaron." ¿De qué tiempo habla el verbo?
A Del pasado (un trimestre que ya terminó).
B Del futuro.
C De algo que pasa siempre.
2. "Me parece excelente." ¿Para qué sirve esta frase?
A Para reportar una cifra.
B Para dar una opinión.
C Para hacer una pregunta.
3. "Estoy de acuerdo." ¿Qué verbo se usa?
A El verbo ser (soy de acuerdo).
B El verbo estar (estoy de acuerdo).
C El verbo tener (tengo de acuerdo).

Etapa 1.4: Consejo de Gramática -- El Pretérito Indefinido y la Opinión

Gramática

Cómo hablar del pasado y dar tu opinión (ES + PT).

EstructuraUsoEjemplo
-ar → nosotros -amosPasado, nosotrosReportamos los resultados.
-ar → él -ó / ellos -aronPasado, 3ª personalogró, aumentaron
-er/-ir → -ió / -ieronPasado, 3ª personacoincidió, disminuyeron
creo que / me parece que + (no) estoy de acuerdoDar opinión y acuerdoCreo que las cifras son buenas. Estoy de acuerdo.
Em portuguêsO pretérito indefinido é o passado terminado: "reportamos / aumentaram / conseguiu / diminuíram". Para opinar use creo que / me parece que + frase. Atenção: diz-se estoy de acuerdo (verbo estar), nunca "soy de acuerdo".
