Stage 1.1: Recommendation Vocabulary

Vocabulary

Listen to each word and read the example. These words help you give your opinion and recommend places.

Recommend -- recomendar
"I would recommend that little cafe."
Suggest -- sugerir
"I suggest the old museum first."
Worth it -- vale a pena
"The view is amazing -- it is worth it."
Must-see -- imperdível / que você tem que ver
"The old bridge is a must-see."
Overrated -- superestimado / não é tudo isso
"The big tower is a little overrated."
Brilliant -- ótimo / excelente
"The coffee there is brilliant."
Gorgeous -- lindíssimo(a) / deslumbrante
"The sunset was absolutely gorgeous."
Charming -- charmoso(a) / encantador(a)
"It is such a charming old street."
Definitely -- com certeza / definitivamente
"You should definitely try it."
Fantastic -- fantástico(a) / maravilhoso(a)
"The fish there is fantastic."

Stage 1.4: Grammar Tip -- So & Such a

Grammar

Para dar ênfase, use so antes de um adjetivo sozinho ("so good", "so peaceful") e such a antes de um substantivo, com ou sem adjetivo ("such a charming place"). Para recomendar, diga I would recommend... ou You should... (sem "to"). / For emphasis, use so before an adjective alone ("so good"), and such a before a noun, with or without an adjective ("such a charming place"). To recommend, say I would recommend... or You should... (no "to").

Estrutura / StructureEnglishPortuguês
so + adjectiveThe food is so fresh.A comida é tão fresca.
such a + (adj) + nounIt is such a charming place.É um lugar tão charmoso.
I would recommend + -ing/nounI would recommend going early.Eu recomendaria ir cedo.
You should + verb (no "to")You should visit the market.Você deveria visitar o mercado.

Stage 4: Situational Quiz

Quiz

Choose the best response for each situation.

You want to recommend a restaurant to a friend. How do you say it?
A "You should try the rooftop restaurant."
B "You should to try the rooftop restaurant."
C "You try should the rooftop restaurant."
You want to say a place is great, before the noun "place".
A "It is so a great place."
B "It is such a great place."
C "It is such great place."
A friend asks about a famous spot that is not really that good.
A "It is a must-see, definitely worth it."
B "Honestly, it is a little overrated, so I would skip it."
C "It is so a fantastic place."

Reflection
