Chapter 3 -- The Code

Grammar Discovery

used to for past habits and states

Grammar Discovery

Listen and Notice

How do we talk about habits we had in the past but not now?

"I used to smoke, but I quit two years ago." (past habit, not now)

"I used to have high blood pressure." (past state)

"I did not use to exercise, but now I do." (negative)

"Did you use to take any regular medicine?" (question)

Grammar Rule

used to + base verb

FormExample
Affirmative: used to + base verbI used to smoke. / She used to live in Rio.
Negative: did not use to + base verbI did not use to exercise.
Question: Did + subject + use to + base verb?Did you use to take any medicine?
MeaningA habit or state in the past that is not true now.

Key idea: used to + base verb describes a past habit or state that is over. In the negative and the question, did carries the past, so we go back to use to (no "d"): "did not use to", "Did you use to...?".

Common Mistake

Watch Out

"I used to smoked." / "Did you used to exercise?"
"I used to smoke." / "Did you use to exercise?"

After used to, always use the base verb. With did (negative / question), go back to use to.
