Chapter 3: The Code

Be + Going To

the easy way to talk about plans

Discover the Grammar

How Do We Talk About Plans?

"I am going to send the agenda."

"We are going to confirm the shipment."

"She is going to meet the supplier."

FormStructureExample
Affirmativeam/is/are + going to + verbI am going to send it.
Negativeam/is/are + not + going toI am not going to be here.
QuestionAm/Is/Are + ... + going to?Are you going to join?

am/is/are + going to + verbo no infinitivo (sem -s). Usamos para PLANOS do futuro.
