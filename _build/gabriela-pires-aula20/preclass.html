Stage 1.1: Vocabulary Cards

Vocabulary

Ouça cada palavra e leia o exemplo. Toque em Listen para ouvir. / Listen to each word and read the example.

So far -- from the beginning until now
"My trip has been amazing so far." (Até agora)
Amazing -- very good, wonderful, that surprises you in a good way
"Paris is an amazing city." (Incrível)
Unforgettable -- so special that you remember it forever
"This is an unforgettable trip." (Inesquecível)
Highlight -- the best or most special part of an event
"The highlight of my trip was the museum." (Ponto alto)
Memory -- a thing you remember from the past
"I have a beautiful memory of Paris." (Lembrança / memória)
Experience -- a thing you do or that happens to you and you learn from
"Travel is an amazing experience." (Experiência)
Miss -- to feel sad because a person or place is not with you
"I miss my family a little." (Sentir falta / sentir saudade)
Capture -- to keep a moment forever, for example in a photo
"I want to capture every memory." (Capturar / registrar)
Journal -- a book where you write about your days and feelings
"I write about my trip in my journal." (Diário)

Stage 1.4: Grammar Tip -- Present Perfect (already / yet / just / ever)

GRAMMAR

Para falar de experiências até agora, usamos "have / has" + o verbo na 3a forma (visited, seen, written, been). / To talk about experiences until now, we use "have / has" + the past participle.

Use / UsoStructure / EstruturaExample / Exemplo
Done / Feito (already)have + already + verbI have already visited the museum.
Not done / Não feito (yet)have not + verb + yetI have not seen the tower yet.
A moment ago / Há pouco (just)have + just + verbI have just written a postcard.
Ask / Perguntar (ever)Have + you + ever + verb?Have you ever tried a croissant?
3rd form / 3a formavisit→visited, see→seen, go→beenwritten, done, been, seen

Key point / Ponto-chave: use have + verb (3a forma) para experiências até agora. already = já feito, yet = ainda não, just = há pouco, ever = já, na sua vida (em perguntas). Diga "I have visited" (não "I have visit").
