Chapter 3: Real Promises

If, When & As Soon As

Grammar Discovery

Listen and Find the Pattern

"If I get there early, I 'll call you."

"As soon as I land, I 'll text you."

"I won't leave until I hear back from you."

"Once we set a date, I 'll book the tickets."

Part 1: condition / timePart 2: result
if / when / as soon as / once / until / before / after + PRESENTwill (or won't) + base verb
As soon as I land (present)I 'll text you (will)

The future idea is in BOTH halves, but only the result clause uses will. After if / when / as soon as / once / until we use the present. When the time clause comes first, add a comma: When you arrive, call me.

Common Mistake

Wrong vs Right

As soon as I will land, I'll text you.
As soon as I land, I'll text you.

After when / as soon as / once / until / if, use the present -- never will.

If she will come to Brazil, we'll catch up.
If she comes to Brazil, we'll catch up.

The will belongs only in the result clause, not after if.
