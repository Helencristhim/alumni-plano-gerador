Chapter 3 -- Used to / Would

Talking About Past Habits

used to + verb & would + verb

Listen & Repeat

Used to & Would in Action

"My family used to visit this museum every summer."

"We would spend the whole afternoon in the galleries."

"The artist used to paint outside, in the fields."

"I did not use to enjoy sculpture, but now I love it."

Grammar Rule

Past Habits: used to / would

StructureExampleNote
used to + base verbI used to visit the museum.a past habit or state that is not true now
would + base verbWe would spend all day there.a repeated past ACTION (not states like be / like)
negative: did not use toI did not use to enjoy art.no -d: "use to", not "used to"
question: Did you use to...?Did you use to draw as a child?again "use to", no -d

Key idea: used to + base verb describes a past habit or situation that is finished now (I used to visit = I do not visit now). would + base verb also tells repeated past actions (We would spend all day there), but NOT states: say "I used to be shy", never "I would be shy". In the negative and question, drop the -d: "did not use to", "Did you use to...?"

Common Mistake

Watch Out

"I did not used to enjoy sculpture."
"I did not use to enjoy sculpture."

In the negative and the question, drop the -d: did not use to, Did you use to...? And use used to (not would) for past states: "I used to be shy", never "I would be shy".

About the Tour

1. What does the admission ticket include?

It includes all four galleries on the floor, as well as the special exhibit on the second floor.

2. How did the artists in the main gallery used to work, and how often would they return to the same spot?

They used to paint in the open air, outside in the fields and gardens, and many of them would return to the same spot again and again to capture the changing light.

3. What is the museum's most valuable work, and where is it?

The masterpiece is a portrait that is almost two hundred years old, at the far end of the portrait gallery.
