Chapter 3 -- The Code

Grammar Discovery

The first conditional and polite opinion frames

Grammar Discovery

Listen and Notice

What do the orange words have in common?

"If we add this feature, we will miss the deadline."

"If the vendor is late, we may need a backup plan."

"If we cut the scope, the client will not be happy."

"If you agree, I will update the plan today."

Grammar Rule

First Conditional + Opinion Frames

If-clause (condition)Result clause
If we add this feature,we will miss the deadline.
If the vendor is late,we may need a backup plan.
If you agree,I will update the plan today.

Key idea: If + present simple, ... will / may + base verb. Use it to disagree softly: open with "I see your point, but...", add a "concern", and explain the consequence with a first conditional.

Common Mistake

Watch Out

"If we will add this feature, we will miss the deadline."
"If we add this feature, we will miss the deadline."

After if, use the present simple, not will. The will or may belongs in the result clause.

Grammar Practice

Build the Conditional

"If we _____add this feature, we will miss the deadline." (present simple after if)
"If the vendor is late, we _____may need a backup plan." (possible result)
"If you _____agree, I will update the plan today." (present simple after if)
"If we cut the scope, the client _____will not be happy." (certain result)

Artefact

The Disagreement Email

NEW MESSAGE

FromRafael Pelizaro
ToSandra, Product Owner
SubjectQuick note on the new login feature

Hi Sandra,

Thank you for the idea. I see your point about the login feature, but I have a concern about the timing. If we add it to this sprint, we will miss the milestone on Friday.

My objection is not the feature itself; it is the constraint on time. There is also a trade-off with quality, because the team will not have time to test it well.

As a compromise, could we move it to the next sprint? If we do that, we will protect the deadline and still deliver it. I would like us to align before we talk to the client.

Best regards,
Rafael Pelizaro

"Perfect. I am glad we align on this. I will send the summary in one hour."
