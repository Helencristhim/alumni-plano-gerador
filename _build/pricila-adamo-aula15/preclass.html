Vocabulary

Stage 1.3: Grammar in Context

Grammar

Read Pricila's advice about handling problems and answer the questions.

"I always tell people: if something went wrong on a trip, I would simply keep my cool. If I lost my passport, I would go straight to the embassy. If our hotel fell through, we would make do with somewhere simpler. Honestly, if the whole day went pear-shaped, I would just roll with the punches and take it in my stride. If I did not have a backup plan, I would feel far more anxious -- so I always prepare one."

1. In "If I lost my passport, I would go to the embassy", is the situation real or imaginary?
A A real thing that is happening now.
B Imaginary / hypothetical.
C Something that always happens.
2. Which verb form comes after "if"?
A Past simple ("lost").
B would + verb ("would lose").
C Present continuous ("losing").
3. Which form comes in the result clause?
A Past simple ("went").
B would + base verb ("would go").
C will + base verb ("will go").

Stage 1.4: Grammar Tip -- Second Conditional

Grammar

Use o second conditional para situacoes HIPOTETICAS ou improvaveis: If + past simple na condicao, would + verbo base no resultado. Depois de if NUNCA use would. / Use the second conditional for HYPOTHETICAL or unlikely situations: If + past simple in the condition, would + base verb in the result. Never use would after if.

Parte / PartEnglishPortugues
If + past simple (condition)If I lost my passport, ...Se eu perdesse meu passaporte, ...
would + base (result)... I would go to the embassy.... eu iria a embaixada.
Imaginary / unlikelyIf the plan fell through, we would make do.Se o plano furasse, a gente se viraria.
Advice (If I were you)If I were you, I would keep my cool.Se eu fosse voce, manteria a calma.
Common mistakeNOT "If I would lose my passport." Say "If I lost..."Nunca use would depois de if.
