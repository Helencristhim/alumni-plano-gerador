Chapter 3 -- The Code

Grammar Discovery

There is and There are

Grammar Discovery

Listen and Notice

When do we say There is and There are?

"There is a lot of sun today."

"There are many clouds in the sky."

"There is a storm tonight."

"There are no clouds tomorrow."

Grammar Rule

There is / There are

FormStructureExample
SingularThere is + 1 thingThere is a storm.
PluralThere are + 2+ thingsThere are many clouds.
NegativeThere is no / There are noThere are no clouds.
QuestionIs there ...? / Are there ...?Is there rain today?

Key idea: use There is with one thing (sun, rain, wind, a storm) and There are with two or more things (clouds). To ask, turn it around: "Is there rain?", "Are there clouds?".

Common Mistake

Watch Out

"There is many clouds."
"There are many clouds."

With plural things (two or more), use There are. Use There is only with one thing.
