Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

Dienane looks out the window in the morning. The weather is good: there is a lot of sun and the sky is blue. There are only two small clouds. In the afternoon, there is rain and there is a strong wind near the river. At night, her friends say there is a big storm, so she stays at home. She likes the sun, but she does not like the storm. Tomorrow, there are no clouds and the weather is good again.

1. How do we talk about the sun in the morning?
A "There are a lot of sun."
B "There is a lot of sun."
C "There is a lot of suns."
2. How do we talk about the two clouds?
A There are only two small clouds.
B There is only two small clouds.
C There are only two small cloud.
3. Why does she stay at home at night?
A Because there is a big storm.
B Because there is a lot of sun.
C Because there are no clouds.

Stage 1.4: Grammar Tip -- There is / There are

GRAMMAR

Usado para dizer que algo existe ou está presente. / Used to say that something exists or is present.

Form / FormaStructure / EstruturaExample / Exemplo
singularThere is + 1 coisaThere is a storm.
pluralThere are + 2+ coisasThere are many clouds.
negativoThere is no / There are noThere are no clouds.
perguntaIs there ...? / Are there ...?Is there rain today?

Key point / Ponto-chave: use there is com uma coisa (singular: sun, rain, wind, a storm) e there are com duas ou mais coisas (plural: clouds, storms). Para perguntar, inverta: Is there rain? / Are there clouds?
