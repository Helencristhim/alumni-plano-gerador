Warm-up — Callback da Aula 17

Review

Na Aula 17 você aprendeu a comparar passado e presente com used to / would e "but now" ("We used to deploy manually, but now it is automated"). Agora você explica o porquê: liga uma causa ao seu resultado. Use because e since antes da causa, e so, as a result ou therefore antes do resultado. (Link the cause to its result, clearly.)

Stage 1.1: Vocabulary Cards

Vocabulary

Ouça cada palavra, leia o exemplo e a tradução. Toque em Listen para ouvir.

Latency -- the delay between a request and the response (PT: latência / atraso de resposta)
"Since the latency was high, requests timed out."
Capacity -- the maximum amount a system can handle (PT: capacidade / limite máximo)
"The server reached its capacity during the surge."
Threshold -- the limit at which something changes or triggers (PT: limiar / limite)
"Because the load passed the threshold, latency went up."
Overload -- to put more work on a system than it can handle (PT: sobrecarregar)
"The traffic surge overloaded the server."
Congestion -- a slowdown caused by too much traffic at once (PT: congestionamento)
"One failure caused congestion across the system."
Throttle -- to deliberately limit the number of requests (PT: limitar / restringir a vazão)
"We had to throttle the requests to protect the system."
Dependency -- something a system relies on to work (PT: dependência)
"The real cause was a dependency on a slow database."
Redundancy -- an extra backup that takes over if one part fails (PT: redundância / reserva)
"Since the database had no redundancy, one failure was enough."
Resilience -- the ability to keep working when something fails (PT: resiliência)
"Because we lacked resilience, a small overload became a big outage."
Consequence -- the direct result or effect of an action or event (PT: consequência / resultado)
"The consequence of the surge was a full outage."

Stage 1.3: Grammar in Context

GRAMMAR

Leia o texto e responda às perguntas.

In the briefing, Daniela linked each cause to its result. "There was a surge in traffic, so the server reached its capacity," she said — so goes before the result. "Because the load passed the threshold, the latency went up" — because goes before the cause. As a result, requests timed out: as a result is a more formal way to introduce the effect. She also used since, which means the same as because: "Since there was no redundancy, one failure caused congestion." Remember: never use because and so together in the same sentence — choose one.

Stage 1.4: Grammar Tip — Cause & Result Connectors

GRAMMAR

Conectores de causa e de resultado (bilingue EN + PT).

ConnectorIntroduzExample
because / sinceA causa (o motivo)"...because the server was overloaded."
soO resultado (informal)"...so the service crashed."
as a result / thereforeO resultado (formal)"As a result, the latency went up."
Regra (PT)Use because e since antes da CAUSA; use so, as a result e therefore antes do RESULTADO. "X happened because Y" = "Y, so X" (mesma ideia, ordem inversa). NUNCA use because e so juntos na mesma frase.
