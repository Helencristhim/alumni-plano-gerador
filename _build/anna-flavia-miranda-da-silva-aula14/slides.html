Chapter 3: Ordering

I'd Like, Please

The polite way to ask for food

Discover the Code

Listen and Find the Pattern

"I would like the fish, please."

"Can I have a coffee, please?"

"I would like the bill, please."

"Can I have the menu, please?"

Polite startExample
I would like + foodI would like the fish, please.
Can I have + food?Can I have a coffee, please?
Always end with please..., please.

To order food, use I would like or Can I have...? and always add please. This is polite. We say I'd like in fast speech.

Common Mistake

Wrong vs Right

✗ WRONG

"I want the fish."

✓ RIGHT

"I would like the fish, please."

✗ WRONG

"I like a coffee."

✓ RIGHT

"I would like a coffee, please."

Use I would like, not "I want", to order. "I like" means you enjoy it. Always add please.
