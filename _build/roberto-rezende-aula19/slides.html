Chapter 3 -- The Code

Grammar Discovery

Used to + base verb: past habits, not true now

Grammar Discovery

Listen and Notice

What do the orange words have in common?

"I used to be nervous before every call."

"I used to read every line from a script."

"We used to lose deals on price alone."

"I used to panic when a client pushed back."

Grammar Rule

Used To + Base Verb

FormSubject + used to + base verb
PositiveI used to be nervous on calls.
PositiveWe used to lose deals on price.
NegativeI did not use to speak up.
QuestionDid you use to read from a script?

Key idea: use used to + base verb for a habit or state that was true in the past but is not true now. The form is the same for every subject. In the negative and question, the verb becomes use to (no -d), because "did" already carries the past: did not use to, did you use to.

Common Mistake

Watch Out

"I used to being shy and we did not used to win."
"I used to be shy and we did not use to win."

After used to, use the base verb (be, win), never the -ing form. In the negative and question with did, write use to without the -d.
