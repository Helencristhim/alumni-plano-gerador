El Código

Demostrativos: este / esta

SingularPlural
Masculinoeste pantalónestos pantalones
Femeninoesta camisaestas camisas

Para preguntar el precio: ¿Cuánto cuesta? (una cosa) / ¿Cuánto cuestan? (varias). Para pagar: ¿Puedo pagar con tarjeta?
