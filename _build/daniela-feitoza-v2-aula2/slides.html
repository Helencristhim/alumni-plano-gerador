Chapter 3

The Tag Rule

Add a short question to invite a reply

Tag Discovery

Listen and Notice

Great event, isn't it?

The keynote was great, wasn't it?

You're an attendee here, aren't you?

You aren't from here, are you?

When the statement is positive, what happens to the tag? And when it's negative?

Tag Rule

Statement + Tag

StatementTagExample
Positive (is / was / are)Negative tag"Great event, isn't it?"
Negative (isn't / aren't)Positive tag"You aren't from here, are you?"
Same auxiliary verbRepeat is / was / are"It was good, wasn't it?"
Pronoun matches subjectit / you / they"You're an attendee, aren't you?"

Positive statement → negative tag. Negative statement → positive tag. Keep the same verb.

Common Mistake

Wrong Tag vs Right Tag

✗ WRONG

"Great event, is it?"

✓ RIGHT

"Great event, isn't it?" (positive → negative tag)

✗ WRONG

"Great event, doesn't it?"

✓ RIGHT

"Great event, isn't it?" (keep the same verb: is)

Flip the sign and keep the same verb. Positive statement takes "isn't / wasn't / aren't".

I can use event words: networking, attendee, keynote, approach, small talk, opener, icebreaker, first time, keep in touch.
