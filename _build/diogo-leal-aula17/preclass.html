Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

Diogo meets a director at a coffee break and has ninety seconds. He starts with a hook: "Have you ever lost an afternoon to a report a computer could write?" That one line can spark interest fast. Then he says what he does: he builds small tools to remove slow, manual work. Next he gives the value: he automates the monthly report so that managers save two hours a week. To sound credible, he uses one real example, not big promises. The pitch feels compelling because every line has a reason. Finally, he keeps the momentum with a clear ask: "Could we set up a short call next week?" His niche is simple automation, and a solution for a real problem is easy to remember.

1. Why does Diogo build small tools?
A To remove slow, manual work.
B To make the report longer.
C To work more hours.
2. What is the value he gives?
A Managers save two hours a week.
B The report has more pages.
C He talks for longer.
3. How does Diogo sound credible?
A He uses one real example, not big promises.
B He makes very big promises.
C He says nothing at all.

Stage 1.4: Grammar Tip -- Saying WHY with "to" and "so that"

GRAMMAR

Como dizer o PROPÓSITO (o porquê) com "to" + verbo e "so that" + frase. / How to say the purpose (the why) with "to" + verb and "so that" + clause.

Form / FormaPattern / EstruturaExample / Exemplo
to + verboação + to + verbo base"I build tools to remove manual work."
so that + fraseação + so that + sujeito + verbo"I automate it so that managers save time."
Negative / Negativaso that + sujeito + do not + verbo"...so that teams do not copy by hand."
Question / PerguntaWhy do you...?"Why do you build it? -- To save time."
Key point / Ponto-chaveNum pitch, não diga apenas O QUE você faz; diga POR QUE importa. Depois de to use o verbo base ("to remove"); depois de so that use uma frase com sujeito e verbo ("managers save time"). Errado: "so that to save time".
