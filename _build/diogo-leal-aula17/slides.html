Grammar Rule

The Word "Why"

To say WHYPatternExample
to + verbaction + to + base verb"I build tools to remove manual work."
so that + clauseaction + so that + subject + verb"I automate it so that managers save time."

Key idea: In a pitch, do not only say WHAT you do. Say WHY it matters. Use to + a verb, or so that + a small sentence. This turns a task into a value, and your pitch becomes compelling.

Common Mistake

Watch Out

"I automate the report so that to save time."
"I automate the report so that managers save time."

After so that, use a small sentence with a subject and a verb. After to, use the base verb: "to save time".
