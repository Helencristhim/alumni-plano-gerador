Capítulo 3: El Código

El Condicional con Si

si + presente, + futuro

Descubre la Gramática

Escucha y Observa

"Si el barco llega tarde, perderemos el plazo."

"Si enviamos por avión, el café llegará a tiempo."

"Si hablamos con el cliente hoy, evitaremos el problema."

"Si pedimos un reembolso, cubriremos el coste."

Parte de la fraseTiempo verbal
Si + ... (la condición)Presente (llega, enviamos)
, ... (la consecuencia)Futuro (perderemos, llegará)
Estructura completaSi + presente, + futuro

Después de SI va el presente, nunca el futuro. La consecuencia va en futuro (-é, -emos, -á).

Error Común

Correcto vs Incorrecto

✗ INCORRECTO

"Si el barco llegará tarde, perderemos el plazo."

✓ CORRECTO

"Si el barco llega tarde, perderemos el plazo."

✗ INCORRECTO

"Si enviaremos por avión, llegará a tiempo."

✓ CORRECTO

"Si enviamos por avión, llegará a tiempo."
