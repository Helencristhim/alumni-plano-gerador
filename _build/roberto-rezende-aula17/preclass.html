Vocabulary

Persuade -- to make someone want to agree or do something
"I want to persuade the client to order more." (Persuadir / convencer)
Hesitant -- unsure and slow to decide or commit
"The client is hesitant about the price." (Hesitante / inseguro)
Incentive -- something that encourages someone to act or buy
"A free trial is a strong incentive to order." (Incentivo)
Trial -- a short test of a product before buying it
"We can offer a free trial of the new model." (Período de teste)
Guarantee -- a firm promise that something will happen
"We would guarantee delivery in March." (Garantir / garantia)
Worthwhile -- worth the money, time, or effort it takes
"For a large order, the discount is worthwhile." (Que vale a pena)
Convince -- to make someone believe or agree with you
"The free trial convinced the client to sign." (Convencer)
Win-win -- a deal that is good for both sides
"A bigger order with a discount is a win-win." (Vantajoso para ambos)

Stage 1.3: Grammar in Context

GRAMMAR

Read the text and answer the questions.

The client loves the engines but is hesitant about the price. Roberto does not push -- he tries to persuade them with smart offers. He uses the second conditional to imagine a deal: "If you ordered 500 units, we would offer a five percent discount." Then he adds: "If you signed today, we would guarantee delivery in March." Finally, he gives an incentive: "If we added a free trial, it would be a win-win." The trial is worthwhile, so the client is almost convinced. Notice: to make a polite, imaginary offer, English uses if + past simple in the condition and would + base verb in the result.

Stage 1.4: Grammar Tip -- The Second Conditional

GRAMMAR

Use o segundo condicional para ofertas hipotéticas e polidas: if + passado simples, would + verbo. / Use the second conditional for polite, hypothetical offers: if + past simple, would + verb.

If + passado / If + pastwould + verbo / would + verb
If we lowered the price,you would save a lot.
If you ordered more,we would offer a discount.
If you signed today,we would guarantee delivery.
If we added a trial,it would be a win-win.

Key point / Ponto-chave: o segundo condicional fala de uma situação imaginária e seu resultado. No if, use o passado simples (ordered, signed); no resultado, use would + verbo base (would offer, would be). NUNCA "would offered" nem "if you would order". É polido porque você oferece, não exige.
