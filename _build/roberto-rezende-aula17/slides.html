Chapter 3 -- The Code

Grammar Discovery

Second conditional: if + past, would + verb

Grammar Discovery

Listen and Notice

What do the orange words have in common?

"If we lowered the price, you would save a lot."

"If you ordered more, we would offer a discount."

"If you signed today, we would guarantee delivery."

"If we added a free trial, it would be a win-win."

Grammar Rule

The Second Conditional

If + past simple (condition)would + verb (result)
If we lowered the price,you would save a lot.
If you ordered more,we would offer a discount.
If you signed today,we would guarantee delivery.
If we added a free trial,it would be a win-win.

Key idea: the second conditional talks about an imaginary situation and its result. The if part uses the past simple (lowered, ordered, signed); the result uses would + base verb (would save, would offer). It is polite because nothing is demanded -- you are just offering a deal.

Common Mistake

Watch Out

"If you would order more, we would lowered the price."
"If you ordered more, we would lower the price."

After if, use the past simple (ordered), not "would". After would, use the base verb (lower), never the past or "to".
