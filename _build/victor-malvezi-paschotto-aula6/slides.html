Grammar Rule

The Block Cheat Sheet

JobStructureExample
Achievementhave / has + past participle"We have made headway."
Unreal pastIf + had + p.p., would have + p.p."If we had launched earlier, we would have won."
Speculationmust / might / could + have + p.p."The delay must have affected Q4."
Debriefreported speech (backshift)"She said the budget would be cut."
RegisterI would appreciate it if you could + base"I would appreciate it if you could confirm."

Watch the contrasts:

Present perfect = result up to now (no finished time). Third conditional = a past that did NOT happen. First conditional (real future) is for live risks. Modal perfects = how sure you are about the past.

Common Mistake

Right vs Wrong

✗ WRONG

"We have reached the milestone last quarter."

✓ RIGHT

"We reached the milestone last quarter."

✗ WRONG

"If we would have launched earlier, we would have won."

✓ RIGHT

"If we had launched earlier, we would have won."

A finished time word (yesterday, last quarter) needs past simple, not present perfect. In a third conditional, the "if" clause uses "had + past participle" -- never "would have".

Precision Drill

All five tools, at speed

Quick Fire

Pick the Right Tool

1 / 5

1. Achievement: say what the team has done with the platform (present perfect).

"We have made significant headway on the new platform."

2. Unreal past: explain the slip (third conditional, "align earlier").

"If we had aligned the leads earlier, we would have avoided the slip."

3. Speculation: guess why Q4 slipped (modal perfect, "must / might").

"The delay must have affected Q4, or the client might have misunderstood the scope."

4. Debrief: report what the director said about the budget (reported speech).

"She confirmed that the budget would be cut by fifteen percent."

5. Register: ask the board to review the figures (hedged request).

"I would appreciate it if you could review the figures at your earliest convenience."

Detective

Spot the Error

0 / 4 errors found

"We have outperformed the benchmark last quarter."
"We outperformed the benchmark last quarter." (finished time = past simple)
"If we would have launched earlier, we would have won."
"If we had launched earlier, we would have won." (if-clause = had + p.p.)
"The delay must affected the Q4 results."
"The delay must have affected the Q4 results." (modal perfect = must + have + p.p.)
"She confirmed that the budget will be cut."
"She confirmed that the budget would be cut." (backshift: will -> would)
