If This, Then That

Stage 1.1: Vocabulary Cards

Vocabulary

Ouça cada palavra e leia o exemplo. Toque em Listen para ouvir.

Situation -- the set of things happening around you at one time
"I always think about every situation before I go." (situação)
Prepare -- to get ready for something before it happens
"If you prepare well, you will enjoy the trip more." (preparar / se preparar)
In case -- in order to be ready if something happens
"I take a paper map, in case my phone runs out of battery." (caso / para o caso de)
Backup -- a second plan or copy you use if the first one fails
"I also prepare a backup, just in case." (plano reserva / cópia de segurança)
Unless -- except if; if not
"Unless something serious happens, nothing will stop this trip." (a menos que / a não ser que)
As long as -- only if; on the condition that
"As long as I leave early, I will arrive before dark." (contanto que / desde que)
Otherwise -- if not; or else (the other result)
"If it is nice, we will eat outside; otherwise, we will stay inside." (senão / caso contrário)
Consequence -- a result that follows from an action or event
"I think about every consequence before I decide." (consequência)

Stage 1.3: Grammar in Context

GRAMMAR

Leia o texto e responda às perguntas.

I am planning a small dinner this weekend, and I always think about every situation. If the weather is nice, we will eat in the garden; otherwise, we will stay inside. I always prepare a backup for the food. In case someone does not eat meat, I will cook a vegetable dish too. As long as I start early, I will not feel rushed, and I can enjoy the evening. Unless something serious happens, this dinner will be wonderful. I think about every consequence, because one small problem can change everything.

1. What form do we use after "if", "unless" and "as long as"?
A the present simple (is, happens, start)
B will + verb (will be, will start)
C the past simple (was, happened)
2. Are these situations real and possible, or impossible?
A Real and possible in the future.
B Impossible and imaginary.
C Finished a long time ago.
3. What is the speaker's backup in case someone does not eat meat?
A Cancel the dinner.
B Cook a vegetable dish as well.
C Buy meat for everyone.

Stage 1.4: Grammar Tip -- The First Conditional (If / Unless / As Long As)

GRAMMAR

Para falar de uma situação real e possível no futuro, usamos o first conditional: if + presente simples, will + verbo. Depois de if, unless e as long as NUNCA usamos will — o verbo fica no presente. Unless = se não / a menos que. As long as = contanto que. Otherwise = senão (o outro resultado). / To talk about a real and possible future situation, use if + present simple, will + verb. After if, unless and as long as, NEVER use will — keep the present simple. Unless already means if not.

FormMeaningExample
If + present, will + verba real, possible result"If it rains, I will stay home."
Unless + present, will + verbunless = if not"Unless it rains, we will go out."
As long as + present, will + verbonly if; on the condition that"As long as I leave early, I will arrive on time."
..., otherwise, will + verbotherwise = if not"Start early; otherwise, you will be late."
