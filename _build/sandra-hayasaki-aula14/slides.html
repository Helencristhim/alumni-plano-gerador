Chapter 3 -- The If Code

Real Situations with If + Will

How we say what will happen if something is true

Listen and Notice

What Shows a Real Condition?

Each sentence is a real, possible situation. Notice the orange words.

"If it rains tomorrow, I will stay home."

"As long as I leave early, I will arrive before dark."

"Unless something serious happens, nothing will stop the trip."

"What will you do if the plan fails?"

The Rule

If + present, will + verb

FormMeaningExample
If + present, will + verba real, possible result"If it rains, I will stay home."
Unless + present, will + verbunless = if not"Unless it rains, we will go out."
As long as + present, will + verbonly if; on the condition that"As long as I leave early, I will arrive on time."
..., otherwise, will + verbotherwise = if not"Start early; otherwise, you will be late."

Tip: the if part uses the present simple, never will. When the if part comes first, we add a comma. Unless already means if not, so do not add not after it.

Common Mistake

No Will After If

"If it will rain, I will stay home."
"If it rains, I will stay home."

After if, unless, and as long as, use the present simple. Keep will only in the result part.

Practice
