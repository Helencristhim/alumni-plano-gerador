Etapa 1.1: Cartoes de Vocabulario

Vocabulary

Ouca cada palavra e leia o exemplo. Toque Listen para ouvir. (Listen = Ouvir)

Available -- disponivel (livre para algo)
"Are you available on Monday?" (Disponivel na segunda?)
Discuss -- discutir / conversar sobre
"Let's discuss the order." (Vamos conversar sobre o pedido.)
Status -- situacao / andamento (como esta algo)
"What is the status of the order?" (Qual a situacao?)
Report -- relatorio (documento com informacoes)
"Please send a short report." (Envie um relatorio curto.)
Issue -- problema / questao
"Is there any issue?" (Ha algum problema?)
Handle -- cuidar de / lidar com
"Who handles the contract?" (Quem cuida do contrato?)
Update -- atualizacao (informacao nova)
"Can you give me an update?" (Pode me dar uma atualizacao?)
Contact -- contatar / contato (pessoa responsavel)
"Who is the contact for this order?" (Quem e o contato?)

Etapa 1.3: Grammar in Context

Grammar

Leia o texto e responda as perguntas. (Read the text and answer.)

The email is not complete, so I call the supplier. I ask: "What do you ship this month? When does the delivery arrive? Who handles the contract? Is there any issue with the schedule? Are you available for a call?" I need a clear status and a short report before the meeting.

1. We say "What DO you ship?" but "When DOES the delivery arrive?" Why? (Por que do/does?)
A "does" with he/she/it (the delivery = it). (does com it.)
B They are the same. (Sao iguais.)
C "does" is the past. (does e passado.)
2. "Who handles the contract?" What do you want to know? (O que voce quer saber?)
A The person responsible for it. (A pessoa responsavel.)
B The date of the contract. (A data.)
C The price. (O preco.)
3. "Is there any issue?" What is an "issue"? (O que e um issue?)
A A greeting. (Uma saudacao.)
B A problem. (Um problema.)
C A report. (Um relatorio.)

Etapa 1.4: Grammar Tip -- Questions (Do/Does & Wh-)

Grammar

Como fazer perguntas no presente simples. / How to ask questions.

Type / TipoPattern / EstruturaExample / Exemplo
Yes/No (I/you/they)Do + subject + verbDo you ship coffee?
Yes/No (he/she/it)Does + subject + verbDoes the delivery arrive Monday?
Wh- questionWhat/Where/Who/When/How + do/doesWho handles the contract?
Key idea / Ideia-chavedo = I/you/we/they; does = he/she/it (o verbo volta ao normal). / do vs does.

Etapa 4: Quiz Situacional

Quiz

Escolha a melhor resposta para cada situacao profissional.

You want to know the order status. You ask: (Voce quer saber a situacao.)
A "You status the order what?"
B "What is the status of the order?"
C "Status is the order?"
You ask about the responsible person. You ask: (Voce pergunta o responsavel.)
A "Who handle the documents?"
B "Who handles the documents?"
C "Who the documents handle?"
You ask if the supplier is free Monday. You ask: (Disponivel na segunda?)
A "You available Monday is?"
B "Are you available on Monday?"
C "Do you available Monday?"

Etapa 5: Producao Livre

Reflection
