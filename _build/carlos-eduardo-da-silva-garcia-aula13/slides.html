Chapter 3: There Is / There Are

There is a unit. There are two areas.

there is + one · there are + many · in, on, near, between

Discover the Grammar

Listen and Notice

"There is a processing unit on the property."

"There are two storage areas near the house."

"There is no certified warehouse yet."

"Is there a warehouse on the farm?"

FormHowExample
One thingThere is + a / oneThere is a processing unit.
Many thingsThere are + number/pluralThere are two storage areas.
NegativeThere is no / There are noThere is no certified warehouse.
QuestionIs there...? / Are there...?Are there storage areas?
Wherein / on / near / betweena unit on the property; a tank between the areas.
one: there is many: there are

There is = one thing exists here. There are = two or more exist here. / There is = existe uma coisa; There are = existem duas ou mais.

Common Mistake

Wrong vs Right

✗ WRONG

"There is two storage areas."

✓ RIGHT

"There are two storage areas."

✗ WRONG

"The property have a processing unit."

✓ RIGHT

"There is a processing unit on the property."

One thing = there is; two or more = there are. To say something exists in a place, use there is / there are (not have). / Uma coisa = there is; duas ou mais = there are. Para dizer que algo existe num lugar, use there is/there are, nao have.

Dialogue

Amara Asks: What Is on the Property?

A
Hi Carlos. I am visiting your region next week. Can you describe the coffee property?
C
Of course, Amara. The farm is in a rural area, near the municipality of Varginha.
A
Is there a processing unit on the property?
C
Yes, there is a small processing unit. There, the coffee cherries become green beans.
A
Are there storage areas for the coffee bags?
C
Yes, there are two storage areas near the main house. Between them, there is a water tank.
A
And is there a certified warehouse on the property?
C
No, there is no certified warehouse yet. But the producer is building one near the boundary.
