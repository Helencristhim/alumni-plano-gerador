Chapter 3 -- The There Code

Grammar Discovery

Asking and reporting with there is / there are

Grammar Discovery

Grammar Rule

Asking & Reporting with There

UseFormExample
Ask (one thing)Is there + a/an + singular?Is there a gym?
Ask (more than one)Are there + plural?Are there two beds?
Report (one thing)There isn't a + singularThere isn't a towel.
Report (no / none)There's no + nounThere's no internet.

Key idea: use Is there for one thing and Are there for more than one. To report a problem, use There isn't a + singular or There's no + noun. "There's no" works for both singular and uncountable things (no internet, no water).

Common Mistake

Watch Out

"Are there a gym in the hotel?"
"Is there a gym in the hotel?"

Use Is there for one thing (a gym, a pool) and Are there for more than one (two beds, three restaurants).
