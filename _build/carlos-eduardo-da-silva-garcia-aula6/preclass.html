Etapa 1.1: Cartoes de Vocabulario

Vocabulary

Ouca cada palavra e leia o exemplo. Toque Listen para ouvir. (Listen = Ouvir)

Document -- documento (papel ou texto digital com informacao)
"I have six documents to produce each cycle." (Tenho seis documentos para produzir por ciclo.)
Spreadsheet -- planilha (tabela de linhas e colunas, como o Excel)
"The farm member spreadsheet has all the names." (A planilha de produtores tem todos os nomes.)
Certificate -- certificado (papel oficial que prova algo)
"The farm has an Impact certificate." (A fazenda tem um certificado Impact.)
Access -- acesso (permissao para abrir ou usar um sistema)
"I have access to the IMS." (Tenho acesso ao IMS.)
Version -- versao (uma forma do documento; a 2 e mais nova que a 1)
"I send the new version of the checklist." (Envio a nova versao da lista.)
File -- arquivo (conjunto de informacao salvo com um nome no computador)
"Each producer has a file in the database." (Cada produtor tem um arquivo no banco de dados.)
Database -- banco de dados (grande armazenamento de dados no computador)
"The producer files are in the database." (Os arquivos dos produtores ficam no banco de dados.)
Tool -- ferramenta (programa ou coisa que voce usa para fazer um trabalho)
"The IMS is my main tool at work." (O IMS e minha principal ferramenta no trabalho.)

Etapa 1.3: Grammar in Context

Grammar

Leia o texto e responda as perguntas. (Read the text and answer.)

I have access to the IMS, and each producer has a file in the database. Fazenda Sao Joao has an Impact certificate, but it doesn't have a corrective action plan. Fazenda Boa Vista doesn't have IMS access yet. I have six documents to produce each cycle. I check who has what, and I report it to my manager.

1. Why do we say "the producer HAS" and not "have"? (Por que "has"?)
A Because with he/she/it we use has. (he/she/it = has.)
B Because it is plural. (Porque e plural.)
C Because it is a question. (Porque e pergunta.)
2. What does "it doesn't have a plan" mean? (O que significa "doesn't have"?)
A A question. (Uma pergunta.)
B A negative: it does NOT have it. (Uma negativa.)
C The past. (O passado.)
3. Which farm doesn't have IMS access? (Qual fazenda nao tem acesso ao IMS?)
A Fazenda Sao Joao.
B Fazenda Boa Vista.
C Every farm.

Etapa 1.4: Grammar Tip -- Have & Has

Grammar

O verbo have para posse e responsabilidade. / The verb have for possession.

Subject / SujeitoVerb / VerboExample / Exemplo
I / you / we / theyhaveI have access. (Tenho acesso.)
he / she / ithasThe farm has a certificate. (A fazenda tem um certificado.)
Negative / Negativadon't / doesn't haveIt doesn't have a plan. (Nao tem um plano.)
Question / PerguntaDo / Does + haveDo you have it? Does the farm have it?
Have vs Behave = posse (I have six documents). to be = descricao (Sucafina is Swiss). NAO "I am six documents".

Etapa 3: Pronunciation

Speaking

Para dizer o que voce tem (com "I" = have).
The farm has an Impact certificate.
Para a 3a pessoa (the farm = it = has).
It doesn't have a corrective action plan.
Para a forma negativa (doesn't have).
Does the producer have all the documents?
Para fazer uma pergunta (Does + have).

Etapa 4: Quiz Situacional

Quiz

Escolha a melhor resposta para cada situacao.

You talk about the farm's certificate. Which is correct? (Fale do certificado da fazenda.)
A "The farm have a certificate."
B "The farm has a certificate."
C "The farm is a certificate."
You say a farm does NOT have a plan. You say: (A fazenda nao tem um plano.)
A "It no have a plan."
B "It doesn't have a plan."
C "It doesn't has a plan."
You count your documents. Which is correct? (Voce fala dos seus documentos.)
A "I am six documents."
B "I have six documents."
C "I has six documents."
