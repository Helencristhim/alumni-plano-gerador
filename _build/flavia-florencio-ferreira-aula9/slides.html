Chapter 3: Want To / Would Like To

The Language of Plans

want to and would like to + verb

Grammar Discovery

Listen and Notice

"I want to improve my English."

"She wants to grow her business."

"I would like to work abroad."

"We want to achieve our goals."

One word is always there: to + a base verb. Why want to with I and we, but wants to with she?

StructureUseExample
want to + verbstrong goalI want to improve.
would like to + verbpolite wishI would like to work abroad.
he / she / itwant → wantsShe wants to grow.

Always to + base verb. Negative: I don't want to stop. Question: Would you like to start?

Common Mistake

Right vs Wrong

"I want improve my English."
"I want to improve my English."
"She want to grow her business."
"She wants to grow her business."
"I would like work abroad."
"I would like to work abroad."

Always to + base verb. And with he / she / it: wants to.

Practice

Detail

want / would like · then to · then the base verb

"I want ___to travel."

"They want to ___grow the company."
