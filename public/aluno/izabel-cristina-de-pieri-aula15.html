Chapter 1: The Post-Mortem

From Making the Case to Looking Back

Last class you built the written case for the deal and separated fact from opinion, giving emphasis with cleft sentences: it was the speed that convinced them; what makes the case compelling is the logic. Making the case looks forward. Today we look back. One year ago, AVRT launched the Halo sensor line into a new market, and it fell short of target. Now you run the post-mortem: what went wrong, and what would you have done differently? Welcome to the language of hindsight: if we had tested it, we would have caught it.

Chapter 3

The Code

Third conditional and mixed conditional

Grammar Discovery

Listen and Notice

"If we had run a pilot, we would have caught the problem."

"If the forecast had been realistic, we would have ordered less stock."

"If we had hired her earlier, we would not be firefighting now."

"We could have salvaged it if we had acted sooner."

TypeStructureMeaning
Third conditionalIf + had + past participle, ... would/could/might have + past participleAn imagined PAST — the opposite of what really happened.
Mixed conditionalIf + had + past participle, ... would + base (now)A past cause with a PRESENT result — "we would not be firefighting now".
OrderEither clause can come first. If-clause first → use a comma. Result first → no comma. NEVER put "would" in the if-clause.

The "if" clause uses had + past participle — never "would have". The result uses would have + past participle for a past result, or would + base for a result you still feel today.

Common Mistake

Right vs Wrong

✗ WRONG

"If we would have tested it, we would have found the bug."

✓ RIGHT

"If we had tested it, we would have found the bug."

✗ WRONG

"If we had run a pilot, we would caught it."

✓ RIGHT

"If we had run a pilot, we would have caught it."

✗ WRONG

"If we had hired her earlier, we would not have been firefighting now."

✓ RIGHT

"If we had hired her earlier, we would not be firefighting now."

Keep "would" out of the if-clause — use "had" + past participle. The result needs "have" + past participle for a past outcome. When the result is about now, switch to "would" + base (the mixed conditional).

Dialogue

The Post-Mortem

A
Izabel, thanks for running this. The Halo launch missed target by thirty percent. Where do you land on the root cause?
I
Speed. We rushed. In hindsight, if we had run a regional pilot, we would have caught the pricing problem before the full rollout.
A
And the firmware defect? That was the setback customers actually noticed.
I
Exactly. If we had tested with real customers, we would have found it in a week. We underestimated how visible it would be.
A
Should we write off the whole line, then?
I
No. There is real momentum in the technology. If we had not built the brand so carefully, there would be nothing to salvage — but there is.
A
So what is the turning point here — what changes for the next launch?
I
Process, not luck. If we had hired an operations lead a year earlier, we would not be firefighting now. So we fix the process, and the next launch will not backfire the same way.

Self-Assessment

What I Learned

I can form the third conditional: If + had + past participle, would have + past participle.
I never put "would" in the if-clause, and I never drop "have" in the result.
I can use the mixed conditional for a past cause with a present result ("we would not be firefighting now").
I can name a root cause, a setback and a shortfall in a project review.
I can run a post-mortem out loud, saying what we would have done differently and what we can salvage.
