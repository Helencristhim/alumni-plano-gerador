Stage 1.4: Grammar Tip -- Present Perfect vs Present Simple

GRAMMAR

Use the present perfect to connect your past experience to now. Use the present simple for current facts. / Use o present perfect para ligar sua experiência ao presente; o present simple para fatos atuais.

FormUseExample
have / has + past participleExperience up to now (no exact time)I have led global HR projects.
... for + periodHow long the situation has lastedI have worked here for twelve years.
... since + start pointWhen the situation startedI have been here since 2019.
... ever / neverLife experience (or none)I have never worked abroad.
Present simpleFacts and routines true now: I report to the VP. / My expertise is people development.

Stage 1.4: Grammar Tip -- Relative Clauses & the Passive for Processes

GRAMMAR

Use relative clauses (who / which / that) to add information, and the passive (is / are + past participle) to describe processes when the doer is not important.

FormUseExample
... who + verbRelative clause for people (pessoas)the managers who lead each unit
... which / that + verbRelative clause for things / processes (coisas)the policy that we implement
is / are + past participlePassive: focus on the process (voz passiva)Recruitment is handled by my team.

always / usually / often / sometimes / rarely / neverFrequency adverb before the main verb: I usually streamline slow processes.

GRAMMAR

Use o present continuous (am/is/are + verbo-ing) para o que acontece AGORA, e o present simple para rotinas. Para uma ação que começou no passado e ainda continua, use o present perfect continuous (have/has been + verbo-ing). (Now vs routine vs still in progress.)

FormUse (PT)Example
present simple: subject + verbRotina, sempre / routine, alwaysI usually lead the weekly meeting.
present continuous: am / is / are + verb-ingAgora, temporário / now, temporaryRight now I am taking notes / she is presenting.
negative & question (continuous)Negativa e pergunta / negative & questionWe are not hiring now. Is she presenting?
present perfect continuous: have / has been + verb-ingComeçou antes e ainda continua / started before, still goingWe have been working on the policy all morning.
now vs routine vs still in progressI usually review plans (routine), today I am reviewing the backlog (now), and I have been reviewing it since Monday (still in progress).

Stage 1.4: Grammar Tip -- Present Simple vs Present Continuous (+ Stative Verbs)

GRAMMAR

Use o present simple (subject + verb) para rotinas e fatos, e o present continuous (am/is/are + verbo-ing) para o que acontece AGORA. Alguns verbos de estado (stative verbs: prefer, notice, involve, know, want) ficam SEMPRE no present simple, mesmo para agora. (Routine vs now vs stative.)

FormUse (PT)Example
present simple: subject + verbRotina, fatos / routine, factsI usually run the Monday meeting.
present continuous: am / is / are + verb-ingAgora, temporário / now, temporaryThis week Liam is running it.
question & negative (simple)Do / does para rotina / routineDo you attend it? We do not assign it.
stative verbs: prefer, notice, involve, know, wantNunca com -ing / never take -ingI prefer email. It involves four steps. (não "am preferring")
