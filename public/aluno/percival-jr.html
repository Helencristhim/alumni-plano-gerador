Stage 1.4: Grammar Tip -- Polite Email Phrases

GRAMMAR

Num email de trabalho você aponta para um anexo com Please find attached + o arquivo, e faz um pedido educado com I would be grateful if you could + verbo na forma base. Depois de "if you could" usamos o verbo base, sem "to" e sem "-ing". / In a work email you point to a file with Please find attached + the file, and make a polite request with I would be grateful if you could + the base verb. After "if you could" we use the base verb.

Key point / Ponto-chave: escrevemos Please find attached (não "find in attach") e, depois de if you could, usamos o verbo base: send, reply, confirm (não "to send" nem "sending"). / We write Please find attached, and after if you could we use the base verb.
