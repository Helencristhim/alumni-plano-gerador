Chapter 3: The Code

Look at the orange words.
Three structures, three uses.

I will call the airline right now.
I am going to talk to the manager.
You should file a report.
You should ask for a replacement.
The Code

Reveal the Rule

StructureUseExample
will + verbInstant decision (right now)"I will call the airline right now."
going to + verbPlanned action"I am going to talk to the manager."
should + verbAdvice (best action)"You should file a report."
I am not satisfied with...Polite complaint"I am not satisfied with this room."
What can you do about...?Asking for a solution"What can you do about this?"
Common Mistake

Watch Out!

"You should to call the airline."
"You should call the airline."
"I should reported the problem."
"I should report the problem."
Grammar Practice

Complete with will, going to, or should

My luggage is missing. I ___will call the airline right now. (instant decision)
You ___should file a report at the desk. (advice)
I ___am going to talk to the manager about the problem. (planned action)
The room is damaged. You ___should ask for a replacement. (advice)
OK, I ___will take the next flight. (instant decision)

Spot the Error

Find the Mistake

"You should to call the airline."
"You should call the airline." (no "to" after "should")
"I should reported the problem yesterday."
"I should report the problem." (should + base verb, never past)
"I am satisfy with the new room."
"I am satisfied with the new room." (satisfied = adjective)
"What I should do about this?"
"What should I do about this?" (should before subject in questions)
"My suitcase is damage."
"My suitcase is damaged." (damaged = adjective form)
