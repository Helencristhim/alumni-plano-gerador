Grammar Stage 1.4: Grammar Tip

Revise os Modal Verbs para negociação: could, would, should.

Modal Verbs — Polite Proposals, Suggestions, and Advice
MODAL USE EXAMPLE
Could
Poderia		Polite requests and suggestions"Could we discuss the terms?"
"Could you consider a lower price?"
Poderíamos discutir os termos? / Você poderia considerar um preço menor?
Would
Gostaria / Faria		Hypothetical/polite offers"Would you accept a larger volume?"
"I would suggest a 10% discount."
Você aceitaria um volume maior? / Eu sugeriria um desconto de 10%.
Should
Deveria		Advice and recommendations"We should review the deadline."
"You should consider our proposal."
Devemos revisar o prazo. / Você deveria considerar nossa proposta.
Can vs Could: "Can" expressa habilidade, "Could" expressa polidez:
"I can offer 5%." (direto, fato) vs "I could offer 5%." (educado, sugestão)
"Can you lower the price?" (direto) vs "Could you lower the price?" (mais polido)

✗ "Can we discussing the terms?" → ✓ "Could we discuss the terms?" (modal + base form)

Stage 1.4: Grammar Tip -- Used To & Would

GRAMMAR

Use these forms to describe past habits and states that are no longer true -- perfect for reflecting on how far you've come.

FormUseExample
used to + verbpast habit OR past state, now changedI used to freeze on calls. / I used to be nervous.
would + verbrepeated past action only (NOT states)I would rehearse before every call.
didn't use to / Did you use to...?negative and question (drop the "d")I didn't use to speak up. / Did you use to...?
Common mistake: use "used to" -- not "would" -- for past states like be, have or feel (I used to be nervous, not "I would be nervous"); and drop the "d" in the negative and question (I didn't use to, not "didn't used to"). PT: use "used to" e "would" para habitos do passado que mudaram. "Used to" serve para acoes E estados; "would" so para acoes repetidas, nunca para estados. No negativo e na pergunta, tire o "d": "didn't use to", "Did you use to...?".
