Chapter 3 -- The Code

Grammar Discovery

The first conditional: if + present, will + verb

Grammar Discovery

Listen and Notice

What do the orange words have in common?

"If the dose is higher, the outcome will improve."

"If we control for age, we will know the truth."

"If the variable confounds the data, the result will not be reliable."

"If the effect passes the threshold, it will be significant."

Grammar Rule

The First Conditional

PartExample
Cause: if + presentIf the dose is higher, ...
Effect: will + verb..., the outcome will improve.
Negative effect..., the result will not be reliable.
Reversed orderThe result will improve if the dose is higher.
QuestionWill the effect be significant if we control for age?

Key idea: the first conditional is if + present (the cause) and will + verb (the likely effect). After if, use the present, never will. We use it to predict a real, probable result.

Common Mistake

Watch Out

"If the dose will be higher, the outcome will improve."
"If the dose is higher, the outcome will improve."

After if, use the present tense for the cause. Keep will only in the result clause.

Grammar Practice

Build the Conditional

"If the dose is higher, the outcome _____will improve." (improve, result)
"If we _____control for age, we will know the truth." (control, cause / present)
"The result will not be reliable if a variable _____confounds the data." (confound, cause / present)
"If the effect passes the threshold, it _____will be significant." (be, result)
