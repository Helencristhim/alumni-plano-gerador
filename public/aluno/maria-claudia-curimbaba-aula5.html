What is the pattern?

How do we order adjectives and express opinions?

OSASCOMP -- Adjective Order

OrderCategoryExample
1Opinionbeautiful, striking, stunning
2Sizelarge, small, tiny
3Ageold, modern, contemporary
4Shaperound, square, rectangular
5Colorred, blue, vibrant
6OriginBrazilian, French, Japanese
7Materialwooden, ceramic, oil
8Purposedecorative, ceremonial

I find it + adjective: Subject + find(s) + it + adjective. NEVER "I find it is beautiful." Just "I find it beautiful."

Chapter 3 -- The Code

Common Mistake

"It is a red big painting."

Wrong order: size comes before color

"It is a big red painting."

Correct: size (big) before color (red)

"I find it is beautiful."

Never add "is" after "find it"

"I find it beautiful."

Correct: find it + adjective directly
