Chapter 3 -- The Code

What do these structures have in common?

Both use the base form of the verb. No "to", no conjugation.

Imperatives + Let's

StructureUseExample
Base verbDirect command (you)Send the report. Take note.
Please + verbPolite request (you)Please review the agenda.
Do not + verbProhibition (you)Do not forget the action items.
Let's + verbGroup suggestion (we)Let's begin with the revenue update.
Let's not + verbNegative suggestion (we)Let's not skip any items.

Key rule: Both imperatives and "let's" use the BASE form of the verb. Never add "to" after "let's" or in imperatives.

Chapter 3 -- The Code

Common Mistake

"She suggest we adjourn."

Missing third-person -s

"She suggests we adjourn."

Correct: third-person singular + subjunctive "we adjourn"

"Let's to begin the meeting."

Extra "to" after "let's"

"Let's begin the meeting."

Correct: let's + base verb (no "to")
