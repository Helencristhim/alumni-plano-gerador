Grammar Rule

Fronting and Inversion

Fronted ExpressionNormal OrderInverted (Emphatic)
Rarely / SeldomWe rarely see this.Rarely do we see this.
Not only... but alsoWe missed + we lost.Not only did we miss, but we also lost.
Under no circumstancesThis should not happen.Under no circumstances should this happen.
Never / Never againWe should never wait.Never should we wait.
Only after / Only whenWe acted after the complaint.Only after the complaint did we act.
Little / Had (conditional)We didn't realize.Little did we realize.
Common Mistake

Watch Out

"Rarely we see such a disconnect between teams."
"Rarely do we see such a disconnect between teams."

After fronted negatives, ALWAYS invert: auxiliary + subject + main verb (like a question).

Grammar Practice

Rewrite with Inversion

"We have never faced this level of scrutiny." → _____Never have we faced this level of scrutiny.
"We seldom get a second chance with a client." → _____Seldom do we get a second chance with a client.
"We did not realize how much this would affect the relationship." → _____Little did we realize how much this would affect the relationship.
"We only understood the gap when we reviewed the timeline." → _____Only when we reviewed the timeline did we understand the gap.

Comprehension

Reconstruct the Inversions

1. "_____ have I seen such a disconnect." (Rarely)

2. "_____ did we miss the deadline, _____ we also lost credibility." (Not only / but)

3. "_____ should this happen again." (Under no circumstances)

Spot the Error

Find the Mistake

0 / 4 errors found

"Rarely we get a second chance with this client."
"Rarely DO WE get a second chance." (inversion after rarely)
"Not only we missed the deadline, but we also lost trust."
"Not only DID WE miss the deadline, but we also lost trust." (inversion after not only)
"Never again we should wait for the client to complain."
"Never again SHOULD WE wait for the client to complain." (inversion: should before we)
"Only after the complaint we acknowledged the problem."
"Only after the complaint DID WE acknowledge the problem." (inversion after only after)
