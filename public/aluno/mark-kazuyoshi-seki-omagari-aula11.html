Chapter 3 -- The Pattern

Grammar Discovery

used to & would: how to talk about past habits

Grammar Discovery

Listen and Notice

What do the orange words tell us about the past?

"I used to live in a small town." (state)

"Every summer, we would visit my grandparents." (action)

"I used to be really shy as a kid." (state)

"After school, we would play until dark." (action)

Grammar Rule

used to vs would

StructureUseExample
used to + verbpast habits AND past states (no longer true)I used to live there. / I used to be shy.
would + verbrepeated past ACTIONS only (not states)We would play outside every day.
didn't use tonegative and questions (drop the -d)I didn't use to like it. / Did you use to play?

Key idea: "used to" works for everything, both actions and states. "would" works only for repeated actions, never for states. So we say "I used to be shy," never "I would be shy."

Common Mistake

Watch Out

"I would be shy as a child."
"I used to be shy as a child."

For past states (be, have, like, know) use used to, never would. And in the affirmative it is always used to with a -d, not "use to".

"used to" (for example, "I used to be shy").

"would" (for example, "we would play in the street").
