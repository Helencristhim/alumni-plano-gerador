Stage 1.4: Grammar Tip

Grammar

Aqui está a regra. Leia com calma e veja os exemplos.

Present Perfect com "been to" e "visited" (Experiências de viagem)

Use o Present Perfect para falar de experiências de vida (viagens, lugares visitados). Estrutura: Subject + have/has + past participle. "Been to" = visitou um lugar e voltou. / Use the Present Perfect to talk about life experiences (travel, places visited). Structure: Subject + have/has + past participle. "Been to" = visited a place and returned.

StructureExampleTradução
AffirmativeI have been to Portugal.Eu já estive em Portugal.
NegativeI have never been to Asia.Eu nunca estive na Ásia.
QuestionHave you ever visited Japan?Você já visitou o Japão?
3rd personShe has been to many countries.Ela já esteve em muitos países.
Common Mistake / Erro Comum:
"I have been in Portugal." ✗ → "I have been to Portugal." ✓
"Did you ever visit Japan?" ✗ → "Have you ever visited Japan?" ✓
Lembre: "been to" (não "been in") para lugares visitados. Use "Have/Has" (não "Did") com o Present Perfect.

Stage 1.4: Grammar Tip

Grammar

Aqui está a regra. Leia com calma e veja os exemplos.

Would Like — Pedidos e Preferências Educados (Polite Requests and Preferences)

Usamos "Would like" para expressar desejos e preferências de forma educada. NUNCA use "I want" em pedidos! / We use "Would like" to express wishes and preferences politely. NEVER use "I want" in requests!

FormStructureExample
Request (pedido)I would like + noun...I would like a room with a view, please.
Question (pergunta)Would you like + noun...?Would you like breakfast included?
Negative (negativa)I would not like + noun...I would not like a room near the elevator.
Short answer (resposta curta)Yes, I would. / No, thank you.Yes, I would. / No, thank you.
Compare with Aula 2:
Could I = asking permission/ability. "Could I have a window seat?"
Would like = expressing preference/desire politely. "I would like a room with a view."
Common Mistake / Erro Comum:
"I want a room." ✗
"I would like a room, please." ✓
Use "I would like..." para expressar desejos com educação. NUNCA "I want" em pedidos!
