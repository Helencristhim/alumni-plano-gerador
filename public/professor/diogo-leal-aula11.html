Grammar Rule

Should and Should Not

FormExample
Good rule -- should + base verbYou should mute your microphone.
Good rule -- should + beYou should be punctual.
Avoid -- should not + base verbYou should not interrupt the speaker.
Avoid -- should not + beWe should not be late.

Key idea: Use should for good advice and should not for things to avoid. After both, use the base verb with NO "to" -- say "You should mute", never "You should to mute". The verb never takes -s: "He should join", not "He should joins". In speech, should not is often shouldn't.

Common Mistake

Watch Out

"You should to mute your microphone."
"You should mute your microphone."

After should, use the base verb with no "to": "You should mute", "He should join". For the negative, say "should not" (or "shouldn't"), never "don't should".
