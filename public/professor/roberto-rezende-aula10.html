Chapter 3: The Code

Grammar
Review

Grammar

Discover the Pattern

1. We have eighteen signed contracts. (fact)
2. We are working with five new prospects. (ongoing now)
3. I will send a summary after this meeting. (quick decision)
4. We are going to expand into agriculture next quarter. (planned intention)

WHY does each sentence use a different tense?

TenseUse in MeetingsExample
Present SimpleFacts, data, permanent situations"We have 18 contracts."
Present ContinuousCurrent projects, ongoing actions"We are targeting 5 prospects."
WillQuick decisions, promises, offers"I will send the report by Friday."
Going toPlanned intentions, strategies"We are going to expand."

Common Mistakes

Watch Out

"We are having eighteen contracts."
"We have eighteen contracts." (fact = present simple)
"Q2 is more better than Q1."
"Q2 is better than Q1." (one-syllable = -er, never more + -er)
Grammar Practice

Choose the Tense

1. Our revenue _____increased by fifteen percent in Q2. (past fact)
2. We _____are preparing for the Agrishow trade fair. (ongoing now)
3. I _____will send the detailed report by Friday. (promise)
4. We _____are going to focus on agriculture next quarter. (planned strategy)
5. Agriculture _____is more promising than mining this year. (comparative)
6. _____Are we all on the same page about the expansion? (question form)
Grammar Review

Linking Words Spoken

"Our numbers are strong. However, we need to address the mining deal."
"Agriculture is growing. In addition, two prospects are converting this month."
"The deadline passed. Therefore, I suggest we reschedule."

Same words as emails, but SPOKEN with natural pauses.

Quick Fire

Listen and Repeat

1. Start a meeting and review results.
"Let me recap the results from last month."
2. A colleague changes the topic. Stay focused.
"Could we address that later? Let us move on to the pipeline update."
3. Ask the team for opinions.
"I would like your input on the new pricing strategy."
4. The meeting is ending. Summarize.
"Let me sum up the key takeaways from today."
5. Assign a task to a colleague.
"Wei Lin, the action item for you is to review the contract by Friday."
6. Make sure everyone agrees.
"Are we all on the same page about the timeline?"
Error Hunt

Spot the Error

"We are having eighteen contracts this quarter."
"We have eighteen contracts this quarter." (fact = present simple, not continuous)
"I will to send the report tomorrow."
"I will send the report tomorrow." (will + base form, no TO)
"Agriculture is more better than mining."
"Agriculture is better than mining." (one-syllable = -er, never more + -er)
"We are on the same page about the timeline?"
"Are we on the same page about the timeline?" (question = inversion)
