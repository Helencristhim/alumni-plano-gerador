Chapter 3

The Code

like / love / hate + verb-ing

Spot the Pattern

What Do These Have in Common?

I like reading.
I love cooking.
I don't like running.
She likes dancing.

After like / love / hate, what do we add to the verb?

Feeling+ verb-ingExample
love (♥♥)cookingI love cooking.
like (♥)readingI like reading.
don't likerunningI don't like running.
hate (✗✗)cleaningI hate cleaning.

Rule: like / love / hate + verb + -ING. For he/she, use likes / loves / hates.

Careful!

Common Mistake

I like cook.
I like cooking.
She like dancing.
She likes dancing.
Try It

Grammar Practice

I (love) ___ — ?I love cooking.
She (like) dance — ?She likes dancing.
I (not like) run — ?I don't like running.
He (love) swim — ?He loves swimming.
We (love) travel — ?We love traveling.
(you / like) read? — ?Do you like reading?
