Stage 1.3: Grammar in Context

Grammar

Pricila arrives at the hotel reception. "I have a reservation under the name Adamo," she says. "I would like a room with a view, if possible." The receptionist gives her a key card for the eighth floor. "You have to check out by eleven," she explains, "and breakfast is part of the amenities." Later, the air conditioning stops working. Pricila needs to change rooms, so she calls reception: "I would like to make a complaint. I need to change my room, please." The staff fix it quickly. Pricila stays calm and polite — and gets exactly what she needs.

1. Which sentence shows a polite desire?
A "I would like a room with a view."
B "You have to check out by eleven."
C "I need to change my room."
2. Which sentence is a hotel RULE (obligation)?
A "I would like a room with a view."
B "You have to check out by eleven."
C "I would like to make a complaint."
3. Which sentence shows a personal necessity?
A "Breakfast is part of the amenities."
B "I need to change my room."
C "Here is your key card."

Stage 1.4: Grammar Tip

Grammar

would like (to) = quero / gostaria (educado)
need to = preciso (necessidade pessoal)
have to = tenho que (regra / obrigação)

FunctionStructureExample
Desirewould like + noun / + to + verb"I would like to check in."
Necessityneed to + verb"I need to change my room."
Obligationhave to + verb"You have to check out by 11."

Common Mistake:

"I need change my room."

"I need to change my room."

Sempre use "to + verbo" depois de would like, need e have.

Chapter 3: The Code

Want, Need,
Must

would like to · need to · have to

Grammar Discovery

Spot the Pattern

I would like a room with a view. (desire)
I need to print a document. (necessity)
You have to check out by eleven. (rule)

What is the difference between these three?

UseStructureExample
Desire (polite)would like + noun / + to + verbI would like to check in.
Necessity (personal)need to + verbI need to change my room.
Obligation / rulehave to + verbYou have to check out by 11.

Key: os três usam to + verbo. would like = quero (educado), need to = preciso, have to = regra/obrigação.

Common Mistake

Watch Out!

"I would like check in." / "I need change my room."
"I would like to check in." / "I need to change my room."

Use to + verb after would like, need, and have.
