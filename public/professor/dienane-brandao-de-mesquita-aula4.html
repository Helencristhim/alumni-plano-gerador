When do we use a, an, or some?

ArticleRuleExample
abefore consonant soundsa coffee, a menu
anbefore vowel soundsan apple, an orange
someuncountable or pluralsome water, some rice
The Code

Common Mistake

I want a water.
I want some water.
Can I have a orange?
Can I have an orange?

Water is uncountable → use some. Orange starts with a vowel sound → use an.
