Chapter 3 -- The Code

Grammar Discovery

Present continuous + need to / have to

Grammar Discovery

Listen and Notice

Does each phrase describe what is happening now or what we need to do?

"The system is not working right now."

"We are running out of time."

"We need to fix the bug."

"I have to call the IT team."

Grammar Rule

Now vs. Necessary

UseStructureExample
happening nowam / is / are + verb-ingThe system is not working.
happening nowam / is / are + verb-ingWe are running out of time.
necessary actionneed to + base verbWe need to fix the bug.
necessary actionhave to + base verbI have to call the IT team.

Key idea: for a problem happening now, use is/are + verb-ing. For a necessary action, use need to or have to + base verb: need to fix (not "need fix").

Common Mistake

Watch Out

"The system not working. We need fix it."
"The system is not working. We need to fix it."

For something happening now we use is/are + verb-ing, and after need to / have to we use the base verb: need to fix, not "need fix".
