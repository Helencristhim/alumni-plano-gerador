Chapter 3 -- The Code

Present Perfect: for vs since

Present Perfect Rule

StructureUsageExamples
have/has + past participle + forDuration (how long)for five years, for two months, for a long time, for the past quarter
have/has + past participle + sincePoint in time (when it started)since 2019, since January, since the restructuring, since we hired
Chapter 3 -- The Code

Common Mistakes

"We have worked here since three years."
"We have worked here for three years."

Three years is a duration, not a point in time. Use for.

"I have been here for January."
"I have been here since January."

January is a specific point in time. Use since.
