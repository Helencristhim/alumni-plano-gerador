Grammar Review

Listen and Remember

Where does each orange structure come from?

"If we order more, you will give us a discount."

"The engines are made in Resende and every unit is inspected."

"You could try the smaller model. We should look at the volume."

"Let us start with the agenda, and we need to cover three points."

Grammar Rule

Your Four Tools

ToolFormExample
Meeting language (L11)let us / we need to + verbLet us start with the agenda.
First conditional (L12)if + present, will + verbIf we order more, you will give a discount.
Passive voice (L13)is / are + past participleThe engines are made in Resende.
Should / could (L14)should / could + base verbYou could try the smaller model.

Key idea: these four tools cover most of a real sales call. Use meeting language to open and structure, the first conditional to make a deal, the passive voice to present the product, and should and could to handle objections. Knowing when to use each one is your real progress.

Common Mistake

Watch Out

"If we will order more, you will give a discount."
"If we order more, you will give a discount."

In the first conditional, the if part uses the present, and only the result uses will: "If we order more, you will give a discount."
