Chapter 3

The Code

I'd like / Can I have / I'll have

Spot the Pattern

Polite Ordering

I would like a coffee.
Can I have some water?
I'll have the chicken.
Would you like a dessert?

How do we order in a polite way?

UseStructureExample
Order politelyI would like / I'd like + nounI'd like a salad.
Ask for somethingCan I have + noun?Can I have the bill?
Choose a dishI'll have + nounI'll have the soup.
Waiter offersWould you like + noun?Would you like a drink?

Rule: use I'd like / Can I have / I'll have + a food or drink. Add please to be polite.

Careful!

Common Mistake

I want a coffee.
I would like a coffee, please.
Give me the menu.
Can I have the menu, please?
