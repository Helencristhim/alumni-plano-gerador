Modal Verbs for Advice

ModalStrengthUse
shouldStrong advice"You should rest." (I believe this is best)
couldSofter suggestion"You could try yoga." (It is an option)
mightPossibility"It might help." (Maybe, not sure)
would recommendProfessional advice"I would recommend..." (formal, expert)
Key Rule: After should, could, and might, use the BASE FORM of the verb (no "to"). Example: "You should rest" (NOT "You should to rest").
Chapter 3 -- The Code

Common Mistake

"You should to rest more."
"You should rest more."
Why? Modal verbs (should, could, might) are ALWAYS followed by the base form of the verb. Never add "to" between the modal and the verb. Same rule for can, will, and must.
