Stage 1.4: Grammar Tip

Grammar

Aqui está a regra gramatical da aula.

There is / There are -- Existe, Tem, Há

Usamos There is para coisas no SINGULAR e There are para o PLURAL. (Use There is for singular, There are for plural.)

TypeSingularPlural
AffirmativeThere is a desk.There are two desks.
NegativeThere is not a printer.There are not any chairs.
QuestionIs there a whiteboard?Are there any folders?

Common Mistake / Erro Comum:

There is two printers. ✗

There are two printers. ✓

Regra: se é MAIS DE UM, use THERE ARE. Se é UM só, use THERE IS.

Chapter 3

The Code

There is / There are

Look at the pattern

"There is a computer on my desk."
"There are two printers in the office."
"Is there a whiteboard on the wall?"
"There are not any windows in this room."

When do we use IS and when ARE?

NumberAffirmativeQuestion
Singular (1)There is a desk.Is there a desk?
Plural (2+)There are two desks.Are there any desks?

Common Mistake

"There is two printers."
"There are two printers."

One thing = There IS. Two or more = There ARE.
