Chapter 3 -- The Code

Grammar Discovery

What do the green words have in common?

If we had a larger sample, the results would be more reliable.
If the methodology were different, we would see different outcomes.
If she were presenting this data, she would highlight the correlation.
If they replicated this study, they would find similar results.

Second Conditional

StructureUseExample
If + past simpleThe condition (hypothetical/unreal)If we had a larger sample...
would + base formThe imaginary result...the results would be more reliable.

Key rule: Use "were" for all subjects in formal English (If I were, If she were). The Second Conditional describes hypothetical or unreal situations.

Chapter 3 -- The Code

Watch Out!

"If we would have a larger sample, the results would be more reliable."

would in the if-clause = incorrect

"If we had a larger sample, the results would be more reliable."

if + past simple = correct

Past Simple vs Present Perfect (Review)

TenseWhenExample
Past SimpleSpecific time"We published the results last year."
Present PerfectUnspecified time"We have published three papers."

Title: "The Effects of Functional Ingredients on Glycemic Response: A Double-Blind Trial"

Keywords: hypothesis, glycemic control, correlation, peer-reviewed, methodology

Chapter 5 -- Practice

Spot the Error

Click each sentence to see the correction

"If we would have a larger sample, the results would be more reliable."
"If we had a larger sample, the results would be more reliable." (no "would" after "if" in Second Conditional)
"We have published the results last year."
"We published the results last year." (specific time = Past Simple, not Present Perfect)
"If the methodology would be different, we would see different outcomes."
"If the methodology were different, we would see different outcomes." (use "were" after "if" in Second Conditional)
"We replicated three studies so far."
"We have replicated three studies so far." (unspecified time + "so far" = Present Perfect)
"If she will present the data, she would highlight the correlation."
"If she were presenting / presented the data, she would highlight the correlation." (Second Conditional = past simple after "if")
