Common Mistakes

Watch Out

"You must take the minutes."
"Could you take the minutes?" (polite request)
"I want to propose the agenda."
"I would like to propose the agenda." (polite intention)
Grammar Practice

Choose the Modal

1. We _____should start with the most urgent topic. (recommendation)
2. _____Could you prepare the quarterly report by Thursday? (polite request)
3. I _____would like to propose adding a new item to the agenda. (polite intention)
4. _____Would you like to facilitate the discussion on stakeholders? (polite offer)
5. We _____could allocate fifteen minutes to the budget review. (suggestion)
6. I _____would suggest we table this until we have more data. (polite suggestion)
Grammar

Strength Scale

Should = Strong recommendation. "We should start now." (I believe this is the right action.)
Could = Softer suggestion. "We could start now." (This is one option.)
Would = Most polite/formal. "I would suggest starting now." (With all respect, this is my idea.)

In meetings: use COULD to suggest, SHOULD to recommend, WOULD to be diplomatic.

"I want to add an item to the agenda."

"We should table this topic for next week." (recommendation, not unilateral decision)
